1D-Plot   View Menu

 

 

 

This menu provides different ways of viewing the opened data set(s).

Display data value under cursor  When this option is selected a cross-hair cursor will appear in the 1D window. This cursor can be used to determine the value of data points in the window. First select the plot window to be viewed. Next press the left mouse button somewhere on the data-set of interest. A new dashed cursor will then appear (see below). By dragging the mouse the cursor will track the data and the (x,y) data values will be reported in the status region at the bottom of the 1D window.

 

If the right button is clicked while the cursor is visible then an offset is applied to the x display in the status window so that it is zeroed at this point. This allows distances along the x axis to be measured. Click the right mouse button again to remove the offset.

 

“D” as the short cut key for this mode.

The data under the cursor is reported in the status window as shown below and is updated in real time as the cursor is moved.

 

 

When in offset mode an 'O' will be displayed in the right corner of the status window

 

 

 

Select a rectangular region  This option allows the user to select a rectangular region in the current 1D plot using the mouse and cursor. This rectangle may be used to specify a region for enlargement, or for other purposes by using the getrect() command. See Zoom a region for more information. Note that the data cursor and select rectangular region option are mutually exclusive functions. “S” as the short cut key.

 

 

Move a plot  This command allows the current 1D plot to be interactively moved or dragged using the mouse, or zoomed using the mouse scroll-wheel. To move, select the point to “grab” and then drag the mouse with the left button help down. The 1D plot will follow you during this operation. To zoom, just rotate the mouse scroll wheel. In one direction it zooms in and the other is zooms out.  “M” is the short cut key.

 

Zoom a region.  This command allows you to zoom in on a region of interest. To do this, first choose the "Select a rectangular region" option in the menu and then select a rectangular region in the plot window using the mouse; click the left mouse button to define one corner and then drag the mouse, with the button held down until the rectangle is the right size. Then release the left button. If you don’t like the region you have selected you can select another region.

 

When you are satisfied with the selected region, click on the toolbar button. (Alternatively type “Z” or select the menu option) The chosen region will then be expanded to fill the plot window. This process can be repeated any number of times. When in the zoom mode a 'Z' will be displayed in the right corner of the status window:

 

 

Show last region  To move back to previously selected regions, click the history button or type “L”. When there are no previous regions to display, the message “No regions in history list” will appear.

 

Display all data  To view the complete data set(s) click the toolbar button or type “A”.

 

Show real part    When a complex data set is displayed this will toggle the real part. Type “R” as a shortcut. (Nothing will happen if the current data set is real).

 

Show imaginary part   When a complex data set is displayed this will toggle the imaginary part. Type “I” as a shortcut. (Nothing will happen if the current data set is real).

 

Toggle border display. By default the 1D plots are surrounded by borders, which include a scale and some labels. If you have a number of subplots these can take up a lot of room. To save space, the borders around the plot can be removed. Toggle this menu option to see the two different possibilities. Note that the size of the plot border can be modified by using the macro margins.mac. (In the GUI  menu).

 

 

1D plot data with border option disabled.

 

If you prefer to see a scale in this mode you can also select the Cross Axes option from the 1D toolbar.

 

 

Display fixed range : When a new plot is generated the scale is automatically adjusted to display the complete data. By toggling the fixed-range option you can turn this feature off, so that the scale range remains unchanged when new data is displayed. This is useful when trying to compare several different data sets which are displayed one after the other. Note that this option is always reset to auto-range following the multiplot command. The shortcut key is "X".

 

 

Multiplot functions.

 

Although by default the 1D plot window only displays one subwindow, it is possible to have any number of subwindows or subplots, each containing more than one overlapping data set. The multiplot menu function allows the user to select the number of cells in the 1D plot window by specifying the number of rows and columns in the plot matrix.

 

 

Note that each cell or subplot will be the same size. To activate this option, select from the list of predefined matrix sizes in the mutiplot submenu, or choose the general m by n option if you need a matrix larger than 2 by 2.

 

 

An example of a 1 by 3 multiplot.

 

Also note that the existing subplots are preserved when this function is run. This allows you to reconfigure the dimensions of the multiplot without having to reload the data. The contents of the subplots can be deleted or switched using the Edit menu functions.

 

Loading data into a multiplot window.

 

In some situations you may wish to combine several saved data sets for presentation purposes. The “Load plot” menu option in combination with a few menu options provides this functionality. Although by default a loaded plot will clear the 1D plot window (New plot) it is also possible to apply an insert, append or replace function with new plots. At the bottom of the Load Plot dialog window are a series of options for this purpose.

 

Insert plot inserts the plots to be loaded after the current plot (the one with the indented outline - obtained by clicking on it with the mouse).

 

 

The lower part of the Load Plot dialog

 

Append plot adds the new plots to the end of the current multiplot (ignoring any blank regions).

 

If there are n plots in the new file then replace plot will replace the current plot and the following n-1 plots with the new data.

 

Note that the existing multiplot will be expanded in size if necessary to cope with any new data added (this is done by adding extra rows).

 

Also note that embedded blank plots are not affected by these load options.

 

There are also several pop-up menu options that can help with organising the multiplot data - see 1D window pop-up menus for more information.

 

View one plot 

 

When multiple plots are displayed it is possible to display just the current one by selecting this option – the other subplots will be hidden. When in this mode a 'V' will be added to the end of the status bar. Shortcut key is "V".

 

 

Full view

 

This option displays all subplots if one has previously been selected with the view one plot command. Once selected the 'V' will be removed from the status bar. Shortcut key is "F"

 

 

Allow overlapping subplots .  By default, when you plot a data set it will automatically remove any previous data. However if you select this option first, then subsequent plots will simply be added to the previous data set without modifying the scales. This is useful for comparing different sets of data.

Move & scale. The next two options are pull-right menus that allow keyboard control of the currently viewed region. The Move options use the arrow keys to shift the currently viewed region left, right, up or down by 1%. Holding down an arrow key will cause the contents to move continuously.

Likewise the Scale options do the same except that the shortcut keys include the shift button. (Here the scaling factor is 10% larger or smaller).
