1D-Plot   File Menu

 

 

The File menu allows you to open and save 1D plot files (that is data with additional plot formatting information), save the plot window as an enhanced windows meta-file and print the plot.

 

Open Plot:  To load native 1D plot format files (*.pt1) use the Open command in the File menu. Simply select the file from the file dialog box which appears and if the file has the correct 1D plot format it will be displayed (see below). You can also use the 1D toolbar button to open and display a file (see right). By default, the plot window is cleared before the data is loaded (see multi-plots for other options).

 

 

Save Plot :  Here the user is prompted to select a folder and a filename using the standard file-save dialog window (see below). The file will automatically have the “pt1” extension added to its name.

Save WMF. The windows metafile, or in this case, the enhanced windows metafile, is a standard way of interchanging picture files between Windows applications. For example, use this command if you want to copy a picture of a plot into a word-processor. The first step is to adjust the size of the window to get the desired aspect ratio for the image. Then complete the Scale factor entry in the metafile dialog box that appears when this option is selected.

 

 

A scale factor of 1 means that the metafile will have the same dimensions as the plot on the screen.  A larger scale factor can be used when digitising problems occur.

 

With a scale factor of 1, the images will have the same resolution as the screen, so that when you enlarge them they can look quite digitized. The scale factor field allows you to compensate for this by scaling the size of the plot without messing up the relative sizes of the label and axes fonts. An example of the effect of modifying this parameter is shown on the right for scale factors of 1 and 5. However note that the scale factor also enlarges the size of the metafile image, so it will probably require scaling down by the same factor when it is imported into another program. (Microsoft Word has the necessary facilities).

Finally, save the metafile by selecting the save button and choosing a filename in the file-save dialog window which then appears.

 

Print plot. This command sends the currently displayed plot to a printer (if connected). Initially the adjacent dialog box will appear.

 

If you are sending the plot to a black and white printer select the “B/W” option. This will ensure that the background etc. is appropriately coloured. If you have a colour printer connected, then select the “Colour” option.

 

 

By default, the output appears on the page as a 100 by 100 mm plot surrounded by axes and labels. These dimensions can be modified by changing the appropriate fields in this dialog box. Alternatively you can choose to make best use of the paper by choosing the “Fit to page” option.

 

Depending on the printer driver there may be other options available in the standard print dialog that appears after the Print button is pressed. For example you may be able to select landscape mode, which would be appropriate for many 1D plots, or you may be able to save the plot as a postscript file which can then be loaded and edited by some other program (such as CorelDraw).
