1D-Plot   Contextual Menu

 

 

In addition to using the main menu options you can also interact with the plot directly. The following example indicates the points on the plot where a mouse click can allow you to modify the view in some way.

 

  1. All text can be selected for modification or font change.

 

  1. The plot axes can be selected for modification.

 

  1. The plot trace and symbols can be selected for modification or removal.

 

  1. A rectangular region may be selected for later enlargement.

           

  1. If the cursor display is selected (see view menu) then a left mouse click on the trace will display a cursor, which may then be dragged through the data set. The data values at each position will be displayed in the status window.

 

The trace pop-up menu shown here also allows you to copy the currently selected plot data to some global variables (by default called x and y). This data may then be manipulated via the command line interface or may be displayed in another subplot using the background pop-up menu show next.

 

This menu also allows all data in the current subplot to be removed, or provides a means by which all other subplots are deleted and the current one is the only one displayed. If multiple 1D vectors have been loaded into the program you can view them using the "Display vector" option.

"Copy 1D data to vector" copies the current trace to one or two vectors. By default the trace will be copied to the global variables x and y. You can choose to just copy the x or y-axis and you can also enter or select different 1D variable names.

 

 

The "Display vector" option allows the selection of global variables to display along the y and optionally the x axis.
