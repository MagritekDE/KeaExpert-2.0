1D-Plot   Preferences Menu

 

 

By default Prospa will try to present the data in the best way it can by automatically calculating tick spacing, data range etc. It will even choose a default title and axes labels. However the "Pref" menu provides a variety of dialog boxes for modifying the way the plot looks if you wish to customize its appearance.

 

Ticks and labels The first of these provides control over where ticks are spaced on the axes and how far apart axes labels are spaced. The dialog box displayed when this option is selected is divided into two parts; tick and labels spacing and tick length.

 

With the “Choose tick parameters automatically” an optimal spacing is determined by the program. If you deselect this option you can choose your own spacing. This will be kept regardless of the size of the region viewed.

The preferences  menu

 

The tick length option sets the length of the minor and major tick lengths in screen pixels (the major tick lengths are those associated with the labels.)

 

 

Use negative lengths to have the ticks on the inside of the plot, positive to have them facing outwards. The Apply to all regions option will change the tick lengths in all the subplots.

 

Mapping and Grids

 

This function displays the adjacent dialog and allows the mapping between data and display to be toggled between linear or logarithmic (base10) modes. Note that it will only be possible to use log mode if all the x or y data are greater than zero.

 

Also note that the tick and label spacing dialog does not affect the display settings when the log mode is selected.

 

The grid section contains check boxes for toggling the display of grids which extend between the minor ticks on either side of the plot (fine x & y grids) or between the major (label) ticks (x & y grid). You can control the colour of these grids by clicking on the coloured buttons to the right and choosing from the standard colour dialog which appears.

 

 

Axes Range. This dialog controls the visible range of the plot window. Although it will be normal to choose this range using the interactive zoom options, sometimes it is useful to manually define the view range.

 

Once this range has been specified, press the Apply button to update the display. When satisfied with the displayed region, press Exit to return to the main window.

The Auto-range checkbox performs the same functions as the fixed-range menu option.

 

Title and text font. This invokes the dialog box shown below:

 

 

Simply type in the text you would like to see above the plot. The font type, size and colour can be selected from the other fields in this dialog. By checking the “Apply to all titles in window” option, the font selections (but not the text) will be applied to all the titles in the 1D plot window.

 

Note that the font formats you choose here become current for this plot. Any other data sent to this plot will use these formats until the plot is reset using the "delete all plots" command or any of the multi-plot commands. In this case the default plot formats will be used. (See Load and Save Plot preferences below)

 

X and Y labels and font. This invokes the dialog box shown below:

 

 

Again type in the text you would like to see next to the plot. The font type, size and colour are selected in the same way as in the title dialog. By checking the “Apply to all labels in window” option the font selections (but not the text) will be applied to all the labels in the 1D plot window. The y label can be placed horizontally or vertically.

 

Note that the font formats you choose here become current for this plot. Any other data sent to this plot will use these formats until the plot is reset using the erase command. In this case the default plot formats will be used (see preferences option in main Prospa files menu).

 

Axes font. This command simply displays the following dialog box, allowing the standard font used on the numeric axes labels to be modified.

 

 

Note that the font formats you choose here become current for this plot. Any other data sent to this plot will use these formats until the plot is reset using the erase command. In this case the default plot formats will be used (see preferences option in main Prospa files menu for modifying the default).

 

 

Plot colours.

 

This dialog allows the plot axes, background, border and grid colours to be set for the current, or all 1D subplots. To modify the colour, click on the coloured squares and then choose the colour from the standard dialog which appears.  When you are satisfied with the colour selection press the "Apply to region" (or to All regions) button.

 

 

The Make default button will ensure that subsequent plots made will also use this colour scheme (however they will not be saved to file – see following option Load/Save Plot Preferences)

 

The Load default button will load the current plot window colour defaults (effectively an undo button).

 

When choosing a border colour you will notice a button called Background Color. If you choose this option then the background colour will default to the currently selected window colour rather than a fixed colour.

 

 

 

Trace settings

 

This option allows the selection of the currently selected trace colour, size, style and symbol via the following dialog box:

 

 

The box is divided into three parts; the top part modifying the trace, the middle left part an associated symbol and the bottom left an optional error-bar. Click on the top row of buttons to display the data points with a line connecting then, as a staircase function, or simply as dots. The last button does not display the trace at all. (Useful when you just want to display the symbol). The trace colour for real and imaginary data may be chosen by left clicking on the colour boxes to the right of the style buttons. The width of the trace may be set to any number from 0 (the narrowest line) up to 15. The dashed options provide 5 options numbered 0 to 4.

 

0          solid line

1          dash-dash- (-----)

2          dot-dot-dot (.....)

3          dash-dot-dash-dot- (-.-.-.-.)

4          dash-dot-dot-dash-dot-dot (-..-..-..)

 

Any other numbers entered default to 0. Press return when the cursor is in the Dash text-box to apply the new dash mode.

 

The symbol section works in much the same way. Choose the symbol shape you prefer (or none - the last button in the list), its colour and size.

 

Finally the default colour for error bars can also be chosen here. Error bars can have a fixed height which is set here or a variable height via a 2D matrix - the first row of which is the height of the top half of the error bar and the second row the height of the bottom half.

 

Fixed height error bars

Variable height error bars

 

Changing any of these parameters will immediately update the currently selected trace.

 

Using the "Make default" button you can ensure that subsequent plots will use this trace colour scheme. The "Load default" button will replace the current setting with the previous default values.

 

Make Plot Black and White

 

This changes all colours to either black or white making the plot suitable for printing or publication.

 

 

 Changes can be made to the current subplot or all of them using the two options provided:

 

 

Load plot preferences

 

You can load a previously saved set of plot preferences and apply then to subsequently draw plots using this command. The default location is the one read from when the 1D plot window is first created. Other locations can also be chosen.

 

 

Save plot preferences

 

You can save the current plot and trace colour scheme, trace settings, and label fonts etc. by using this commands. If you write to the default location then when a 1D Plot window is created it will then use these preferences. You can also choose alternative locations.

 

 

Note that each Plot preference list must be stored in a folder since two files are written.
