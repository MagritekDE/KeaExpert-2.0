CLI - Introduction

 

Although many useful operations can be achieved by using the 1D, 2D and 3D plot windows and their associated graphical user interfaces (GUI), there are strict limitations on what they can and cannot do. At some stage you will therefore need to use the command line interface or CLI to manipulate data in a way that is not available from the GUI modules.

 

Central to the CLI operation is a mathematics interpreter that allows data, in the form of strings, lists, structures, scalars, vectors and matrices to be manipulated in a fairly standard manner. This interpreter is somewhat reminiscent of the BASIC programming language although some features have also been borrowed from more advanced programming languages. The content of CLI help files applies equally to the commands entered in the CLI or  into the macro editor.
