2D-Plot  Contextual Menu

 

 

In addition to using the main menu options you can also interact with the plot directly. The following example indicates the points on the plot where a mouse click will allow you to modify the view in some way.

 

  1. All text can be selected for modification or font change.

 

  1. The plot axes can be selected for modification.

 

  1. A popup menu (see below) appears when the right mouse button is clicked over the image, allowing it to be manipulated.

 

  1. A rectangular region may be selected for later enlargement or other investigation.

           

  1. If the cursor display is selected (see view menu) then a left mouse click on the image will display a cursor, which may then be dragged around the data set. The data values at each position will be displayed in the status window.

 

The image contextual menu shown here also allows you to copy the currently selected plot data to some global matrix (by default called m). 

 

 

 

 

The contextual image menu

 

This data may then be manipulated via the command line interface or may be displayed in another subplot using the Display matrix option in this menu. Note that the region axes range can also be saved.

 

 

The display matrix dialog also allows the x and y axes range to be set. These are vectors with 2 elements signifying the minimum and maximum range for that axis.

 

This menu also allows all data in the current subplot to be removed and  provides a means by which all other subplots are deleted and the current one is the only one displayed.
