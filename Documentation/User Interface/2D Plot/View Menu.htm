2D-Plot  View Menu

 

 

This menu provides different ways of viewing the opened data set(s).

 

 Show data value under cursor   When this option is selected a cross hair cursor will appear.

 

 

 

The view menu

 

 

This cursor can be used to determine the values of data points in the window. First select the plot window to be viewed. Next click somewhere on the data set of interest. The new cursor will then appear. By moving the mouse, while the left mouse button is pressed, the cursor will track the data and the cursor coordinates and data values will be reported in the status region at the bottom left of the 2D window.

 

 

If the right button is clicked while the cursor is visible then an offset is applied to the x,y coordinates displayed in the status window so that it is zeroed at this point. This allows distances along the x axis to be measured. In this mode the distance from the reference point is also displayed in the status-bar. (0.282 in this example).

 

When in offset mode an 'O' will be displayed in the right corner of the status window

 

 

Click the right mouse button again to remove the offset.

 

“D” as the short cut key for this mode.

 

 

Select a rectangular region  This option allows the user to select a rectangular region in the current 1D plot using the mouse and cursor. This rectangle may be used to specify a region for enlargement, or for other purposes by using the getrect() command. See Zoom a region for more information. Note that the data cursor and select rectangular region option are mutually exclusive functions. “S” as the short cut key.

 

Selecting a rectangular region

After zooming

 

Zoom a region  This command allows you to zoom in on a region of interest. To do this, first choose the "Select a rectangular region" option in the menu and then select a rectangular region in the plot window using the mouse; click the left mouse button to define one corner and then drag the mouse, with the button held down until the rectangle is the right size. Then release the left button. If you don’t like the region you have selected you can select another region.

 

When you are satisfied with the selected region, click on the toolbar button. (Alternatively type “Z” or select the menu option) The chosen region will then be expanded to fill the plot window. This process can be repeated any number of times. When in the zoom mode a 'Z' will be displayed in the right corner of the status window

 

 

 

Show last region  To move back to previously selected regions, click the history button or type “L”. When there are no previous regions to display, the message “No regions in history list” will appear.

 

Display all data  To view the complete data set(s) click the toolbar button or type “A”.

 

Move a plot  This command allows the current 1D plot to be interactively moved or dragged using the mouse, or zoomed using the mouse scroll-wheel. To move, select the point to “grab” and then drag the mouse with the left button help down. The 1D plot will follow you during this operation. To zoom, just rotate the mouse scroll wheel. In one direction it zooms in and the other is zooms out.  “M” is the short cut key.

 

Scroll-wheel zooming

 

While in any of the 3 cursor modes it is possible to zoom the image using the scroll wheel on the mouse. Rotating it towards you zooms in, while rotating it away from you zooms out.

 

Show row data

 

In this mode row data under the cursor is copied into the 1D window for display. A horizontal cursor appears which can be dragged across the data set. As the cursor is moved the corresponding 1D data set under the cursor is displayed.

 

 

A row selected in the 2D plot and the corresponding data set displayed in the current 1D plot

 

 

Show column data

 

 In this mode column data under the cursor is copied into the 1D window for display. A vertical cursor appears which can be dragged across the data set. As the cursor is moved the corresponding 1D data set under the cursor is displayed.

 

 

 

A column selected in the 2D plot and the corresponding data set displayed in the current 1D plot

 

Note that if the data in the 2D image is complex, then complex row or column data will be displayed. Also note that row and column data is automatically stored in the variable v1 so that it can be manipulated by the command line interface functions if desired.

 

Comparing rows and columns

 

When displaying row or column data from a 2D plot in the 1D window the data is by default scaled so that it fills the 1D window vertically (This is called Auto Ranging). However if you wish to compare absolute amplitudes as you view different cross-sections, then first display the row or column with the maximum amplitude and then check the Fixed Range option in the 1D window menu. After this has been done new rows or columns will be displayed with a fixed vertical scale.

 

If you wish to compare rows or columns from different 2D data sets you have two options. You can either display them in different 1D windows (for example by choosing the 2 rows 1 column option from the 1D window view menu.

 

 

Comparing rows

 

Alternatively you can use a single window and select the "Allow overlapping plots" 1D toolbar option. With this option selected new rows and columns will be drawn over the top of the previous one but will not become permanent until the mouse button is released.

Multiplot functions

 

Although by default the 2D plot window only displays 1 subplot, it is possible to have any number of subplots. The multiplot menu function allows the user to select the number of cells in the 2D plot window by specifying the number of rows and columns in the subplot matrix.

 

Note that each cell or subplot will be the same size. To activate this option select from the list of predefined matrix sizes in the mutiplot submenu, or choose the general m by n option if you need a matrix larger than 2 by 2.

 

 

An example of a 1 by 3 multiplot

 

Also note that any existing subplots are preserved when this function is run. This allows you to reconfigure the dimensions of the multiplot without having to reload the data. To delete the contents of the one or all subplots use the contextual menu options.

 

Loading data into a multiplot window

 

In some situations you may wish to combine several saved data sets for presentation. The Load plot command in combination with a few menu options provides this functionality. Although by default a loaded plot will clear the 2D plot window (New plot) it is also possible to insert, append or replace when loading plots. At the bottom of the Load Plot dialog window are a series of options for this purpose.

 

Insert plot inserts the plots to be loaded after the current plot (the one with the indented outline - obtained by clicking on it with the mouse).

 

 

The lower part of the Load 2D Plot dialog

 

Append plot

 

Append plot simply adds the new plot(s) to the end of the current multiplot (ignoring any blank regions).

 

Replace plot

 

If there are n plots in the new file then replace plot will replace the current plot and the following n-1 plots with the new data.

 

Note that the existing multiplot will be expanded in size if necessary to cope with any new data added (this is done by adding extra rows).

 

Also note that embedded blank plots are not affected by these load options. If you wish to remove these then use the Remove blank plots option.

 

There are also several pop-up menu options that can help with organising the multiplot data - see 2D window pop-up menus for more information.

 

View one plot 

 

When multiple plots are displayed it is possible to display just the current one by selecting this option – the other subplots will be hidden. When in this mode a 'V' will be added to the end of the status bar. Shortcut key is "V".

 

 

Full view

 

This option displays all subplots if one has previously been selected with the view one plot command. Once selected the 'V' will be removed from the status bar. Shortcut key is "F"

 

 

Move & scale

 

The next two options are pull-right menus that allow keyboard control of the currently viewed region. The Move options use the arrow keys to shift the currently viewed region left, right, up or down by 1%. Holding down an arrow key will cause the contents to move continuously.

 

 

 

Move submenu

 

Likewise the Scale options do the same except that the shortcut keys include the shift or control button. (Here the scaling factor is 10% larger or smaller).

 

Scale submenu

 

 

Colour-scale

 

Toggles the display of a colour-scale next to the current plot. Note that the colour-scale is only displayed if borders are selected.

 

 

Toggle border display

 

By default the 2D plots are surrounded by borders, which include a scale and some labels. If you have a number of subplots these can take up a lot of room. To save space, the borders around the plot can be removed. Toggle this menu option to see the two different possibilities. Note that the size of the plot border can be modified by using the macro margins.mac (found in the utilities menu).

 

 

2D display mode

 

This dialog box (shown below) provides an interface that lets the user control the appearance of the 2D plot.

 

 

 

The 2D plot window can display data in 3 different ways As an intensity plot with different colours representing different data intensities, as a contour plot, with certain levels indicated with a solid coloured line, or as an interpolated plot. This is a contour plot with solid colour filling the space between the contours. If a large number of levels are chosen, this mode takes on the appearance of higher resolution version of the original data set. Examples of the 3 modes are given below

 

 

An enlarged region of a 2D plot showing the same region plotted in intensity, contour and interpolated mode. In the contour mode 10 levels are drawn while in the interpolated mode 50 levels are used to give a smooth colour variation.

 

Normally the colours for each data point are chosen based on the full range of data present in the visible image. However it is possible to override this using the controls on the right side of the dialog. Deselect the "Auto range" check box to allow control of the colour mapping. You can then specify the maximum and minimum mapping levels manually or you can adjust them using the sliders provided. Notice that if the colour scale is also displayed, then the minimum level corresponds to the colour at the bottom of the scale (usually black) while the maximum level corresponds to the colour at the top of the scale (e.g. white). The following figure shows an example of using these controls

 

 

The same region viewed with different maximum and minimum intensity settings

 

An exception to this occurs when the colour scale is of the positive and negative type. In this case the max and min levels should both be positive and apply equally to positive and negative numbers.

 

 

 Surface plot

 

 Selecting this option will draw a surface plot in the 3D plot window representing the surface of the currently display image

 

 

A surface plot (right) of part of an image (left)

 

Holding down the shift-key while the current region is dragged or zoomed will automatically update the 3D plot.

 

More information about making surface plots may be found in the 3D manual. Note that surface plots of large regions (> 256 by 256 pixels) can take a while to calculate.
