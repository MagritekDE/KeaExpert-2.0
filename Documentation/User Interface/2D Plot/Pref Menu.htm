2D-Plot  Preferences Menu

 

By default Prospa will try to present the data in the best way it can by automatically calculating tick spacing, data range etc. It will even choose a default title and axes labels. However the "Pref" menu provides a variety of dialog boxes for modifying the way the plot looks if you wish to customize its appearance.

 

Tick spacing The first of these provides control over where ticks are spaced on the axes and how far apart axes labels are spaced. The dialog box displayed when this option is selected is divided into two parts; tick and labels spacing and tick length.

 

With the “Choose tick parameters automatically” an optimal spacing is determined by the program. If you deselect this option you can choose your own spacing. This will be kept regardless of the size of the region viewed.

 

The tick length option sets the length of the minor and major tick lengths in screen pixels (the major tick lengths are those associated with the labels.)

 

 

Use negative lengths to have the ticks on the inside of the plot, positive to have them facing outwards. The Apply to all regions option will change the tick lengths in all the subplots.

 

Command-line alternative (for current region):

 

axes("autoscale","true",

     "xtickspacing",0.2,

     "ytickspacing",0.2,

     "xticksperlabel",5,

     "yticksperlabel",5,

     "smticksize",5,

     "lgticksize",10)

 

 

 

2D Grids

 

This function displays the following dialog and allows the display of grids. The grid section contains check boxes for toggling the display of grids which extend between the minor ticks on either side of the plot (fine x & y grids) or between the major (label) ticks (x & y grid). You can control the colour of these grids by clicking on the coloured buttons to the right and choosing from the standard colour dialog which appears.

 

 

 

The log/linear axes mode is not implemented yet.

 

Command line alternative:

 

 grid("xgrid","true",

     "ygrid","true",

     "color",[136,136,255],

     "finexgrid","true",

     "fineygrid","true",

     "finecolor",[0,0,210])

 

Axes Range  This dialog controls the visible range of the plot window. Although it will be normal to choose this range using the interactive zoom options, sometimes it is useful to manually define the view range.

Command line alternative:

 

zoom2d(-1,1,-0.5,0.5)

 

Title and text font  This invokes the dialog box shown below:

 

 

Simply type in the text you would like to see above the plot. The font type, size and colour can be selected from the other fields in this dialog. By checking the “Apply to all titles in window” option, the font selections (but not the text) will be applied to all the titles in the 2D plot window.

 

Note that the font formats you choose here become current for this plot. Any other data sent to this plot will use these formats until the plot is reset using the "delete all plots" command or any of the multi-plot commands. In this case the default plot formats will be used. (See Load and Save Plot preferences below)

 

X and Y labels and font  This invokes the dialog box shown below:

 

 

Again type in the text you would like to see next to the plot. The font type, size and colour are selected in the same way as in the title dialog. By checking the “Apply to all labels in window” option the font selections (but not the text) will be applied to all the labels in the 2D plot window. The y label can be placed horizontally or vertically.

 

Note that the font formats you choose here become current for this plot. Any other data sent to this plot will use these formats until the plot is reset using the erase command. In this case the default plot formats will be used (see preferences option in main Prospa files menu).

 

Axes font  This command simply displays the following dialog box, allowing the standard font used on the numeric axes labels to be modified.

 

 

Note that the font formats you choose here become current for this plot. Any other data sent to this plot will use these formats until the plot is reset using the erase command. In this case the default plot formats will be used (see preferences option in main Prospa files menu for modifying the default).

 

 

Plot colours

 

This dialog allows the plot axes, background, border and grid colours to be set for the current, or all 2D subplots. To modify the colour, click on the coloured squares and then choose the colour from the standard dialog which appears.  When you are satisfied with the colour selection press the "Apply to region" (or to All regions) button.

 

 

 

The Make default button will ensure that subsequent plots made will also use this colour scheme (however they will not be saved to file – see following option Load/Save Plot Preferences)

 

The Load default button will load the current plot window colour defaults (effectively an undo button).

 

When choosing a border colour you will notice a button called Background Color. If you choose this option then the background colour will default to the currently selected window colour rather than a fixed colour.

 

 

 

 

 Select image colour-map

Prospa includes several predefined colour schemes for the intensity plots and contour plots. Selecting from one of these options (copper, grey, hot, rainbow, hotandcold and spectrum) will cause the plot to be redrawn in the new colour scheme. Note that this applies to the current plot alone.

 

 

The following example shows the "hot" and "rainbow" colour schemes. The corresponding colour scales have also been displayed using the colour scale button . The colour-scales labelled +/- are for use with positive and negative data. In these cases the top half of the colour-scale is used for positive data values and the bottom half for negative value. Zero will be mapped to the central colour.

 

 

 

 

Adding new colour scales

 

Colour scales are just global RGB matrices (red-green-blue, with 0 minimum and 1 maximum), with 3 columns and N rows. N determines the number of colours in the colour scale. To add a new colour-scale first define the new matrix e.g.

 

newcs = [0,0,0;

        .33,0,0;

        .66,0,0;

        1,0,0;

        0,0,0]

 

and then add it to the colorscales list:

 

colorscales = colorscales + "newcs"

 

 

 

 

A 2D image with a new colour-scale

This new entry will now appear in the colours menu.

 

Note that the last entry in the colour-scale (bold above), determines whether the colour-scale is normal or +/-. (A normal colour-scale maps the minimum value in the data set to the bottom colour and the maximum value to the top colour). A zero (0) in the red position (as in this case) implies a normal scale while a one (1) implies a +/- scale.

 

To make your new colorscale permanent you will need to add it the startup macro colormaps.mac, this can be found in the prospa preferences folder which can be located using the following command:

 

cd "$prefdir$\Startup\"

 

 

Make Plot Black and White

 

This changes the border and background (for contours) along with all labels to black and white. However image data and grids will retain their original colours.

 

 

 Changes can be made to the current subplot or all of them using the two options provided:

 

 

 

Load plot preferences

 

You can load a previously saved set of plot preferences and apply then to subsequently draw plots using this command. The default location is the one read from when the 2D plot window is first created. Other locations can also be chosen.

 

 

 

Save plot preferences

 

You can save the current plot and trace colour scheme, trace settings, and label fonts etc. by using this commands. If you write to the default location then when a 2D Plot window is created it will then use these preferences. You can also choose alternative locations.

 

 

Note that each Plot preference list must be stored in a folder since two files are written.
