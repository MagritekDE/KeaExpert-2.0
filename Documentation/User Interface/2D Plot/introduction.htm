2D-Plot  Introduction

 

  

The 2D Plot is that part of the Prospa application that provides facilities for loading, saving, manipulating and viewing two-dimensional (2D) data sets.

 

Multiple data sets may be simultaneously displayed and interactively zoomed, shifted and measured.

 

Axes parameters may be automatically or manually selected, as can the data set colour scale.

 

Through the use of user-defined macros, data sets may also be modified and combined.
