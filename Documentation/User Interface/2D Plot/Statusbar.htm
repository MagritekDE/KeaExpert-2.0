2D-Plot  Status-bar

 

 

This appears at the bottom of the 2D plot window. It contains four fields, the left one displays data values - obtained using the data cursor option, the next field provides instructions to the user about the current use of the left and right mouse button (not completely shown below), the next field shows the type and size of the current data set while the rightmost field displays a series of status letters which indicate the current special display mode:

 

Z ... plot is zoomed

V ... one subplot is displayed the others are hidden

O ... and offset is applied to the (x,y) cursor.
