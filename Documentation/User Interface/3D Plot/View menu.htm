3D-Plot   View menu

 

 

The view menu provides several options:

 

Specular lighting

 

 By default the lighting used to illuminate the 3D plot is reflected specularly from the surface, this gives rise to highlights and makes the plot appear more realistic. However you can turn this option off if you so desire.

 

 

Smooth shading

 

Surface plots are made from a large number of polygons. These may be displayed using constant normal vectors, or interpolation may be used to provide smoother shading. The result of this for the 2D surface plot option is shown below. This example also illustrates the effect of turning specular lighting on and off:

The preferences dialog

 

This option runs the macro:

 

<appdir>\macros\coreMacros\ preferences3d.mac

 

which displays a window that allows the modification of certain 3D surface plot parameters.

 

 

 

·        Depth cueing means that objects further away from the viewer will be merged with the background giving an impression of increasing distance from the observer.

 

 

With depth cueing

 

 

Without depth cueing

 

The numbers used for adjusting the depth cueing parameters are the point at which depth cueing begins (full brightness) to the point where it ends (background colour). Zero (0) corresponds to the location of the rotation origin (see below) which in the above example is in the middle of the image. Negative numbers are away from the origin (into the page) while positive number are out of the page. Note that the scale on the image may not be much help in determining these numbers so some trial and error may be necessary to work out the appropriate values.

 

·        The background colour button allows the selection of the background colour for the 3D plot.

 

·        Adjusting the rotation origin

 

The rotation origin specifies the point about which the image rotates. Note that the numbers refer to the data set indices of this point and assume 3D scale factors of 1. This information can be obtained from the 2D slice display. Note that the 3D data set axes may not be a good source of this information if it contains offsets or is scaled.

 

Normally you won't need to modify these numbers (which are generated automatically), however it can be useful if you want to investigate one point in the image closely. Because of the complexities of the rendering process it is not possible to click on the plot and use the measured coordinate to determine a new rotation centre. Instead adjust the location of the rotation origin interactively by moving the cursor position while holding down the control (x-y adjustment) or control+shift (z adjustment) keys. As the position of the origin is adjusted, feedback of its new location is given by means of a red x-y-z axis set. Subsequent to this adjustment rotations will occur about the chosen point.

·        GUI sensitivity

 

These numbers modify the sensitivity of the controls used to adjust the viewing angle, scale, shift position and distance. Larger numbers mean bigger steps when changing the plot view. Note that these numbers are not recorded between sessions.

 

 

Rotate plot

 

To rotate the plot use the arrow keys as specified in the menu. (ccw = counter clockwise, cw = clockwise) or use the mouse while over the image (left-right, up-down and shift-rotate provide the different viewing angle adjustments.)

 

 

 

Move plot

 

To move the rotation origin use the arrow keys as specified in the menu or press the control or control-shift button while moving the mouse.

 

 

Scale plot

 

Scaling the plot involves increasing or decreasing the size of the plot along each of the 3 axes. Note that this also scales the text size.

 

 

Mouse controls

 

The mouse can also be used to control the plot. Rotating the scroll-wheel will make the plot appear closer or further away while moving it will rotate the plot about the azimuthal or elevation axes. (Use shift button to allow rotation about the twist axis)

 

The Hide window

 

The hide window menu option hides the window. This can be restored from the main Window menu.
