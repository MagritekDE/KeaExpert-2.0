Creating 3D plots

 

 

Although the 2D and 3D surface plot macros provide a simple way to plot 3D data, more sophisticated plots can be generated using purpose written macros. For example the following plot shows the fields generated by a simple Helmholtz pair coil (demo\helmholtz_coil.mac).  The cyan (light blue) circles represent the wires, while the red and green surfaces represent two different Bz field values. Note that a variety of different axes can be drawn.

 

 

3D plots elements are cumulative, so in this case we draw the wire, the axes and the two surfaces using separate commands. It is even possible to display data from two or more matrices using such a macro.
