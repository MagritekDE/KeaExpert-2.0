3D-Plot Contextual menu

 

 

A pop-up contextual menu will appear when you click the right mouse button over the 3D plot. This menu (shown on the right) allows the display of existing 2D and 3D matrices as surface plots. If the selection is a 2D matrix then a surface plot will be drawn immediately, for the 3D surface plot a dialog will appear allowing control of a number of display parameters.

 Display a 3D isosurface

 

 

 

 

 

When first displayed, this dialog will present the range of data values in the 3D matrix, making a guess at the isosurface to display. You can modify the isosurface level to plot the surface of interest and adjust the range of values show on the x, y and z axes. (The default is just the index range for each axis). The colour of the isosurface can be selected by clicking on the surface level button while the  transparency of the surface can also be specified; 1 is opaque and 0 is completely transparent

 

Example of two isosurfaces plotted with the outer surface having an alpha = 0.5.

 

Multiple isosurfaces can be displayed by running this macro more than once. However when overlaying plots in this way remember to uncheck the "Clear plot" control after the first surface is drawn and set the axes mode to "No axes", otherwise they will be drawn twice.

Surface plot showing axes

 

In addition to displaying the isosurface, this macro can also add scales to the plot with an optional box as shown above. In addition to showing the matrix indices you can also choose your own axes range by modifying the default values in the axes range group box. The menus to the right of these entries allow rapid addition of scales from 1D global vectors which include maximum and minimum values (you will need to define these first).

 

Finally, if the data surface you are interested in is very small relative to the complete matrix you can choose the fitted axes option which will draw an axes only 25% bigger than the display surface dimensions. Note that in this case you will need to add your own range since the dialog box is not aware of the size of the 3D surface until after it is plotted.
