Adding clipping planes

 

 

The clipping plane macro (found under the 3D menu), allows you to remove surfaces from a plot to reveal hidden detail. The macro interface window is shown below

 

 

and provides 2 clipping planes, each of which can be applied along one of 3 axes. The range of plane positions is determined by the matrix specified (useful if only one data set is displayed), or alternatively you can manually enter a range of values for each axes. Either side of the plane can be made visible using the side radio buttons.
