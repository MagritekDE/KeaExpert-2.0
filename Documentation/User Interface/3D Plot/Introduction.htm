3D-Plot   Introduction

 

 

 

The 3D plot window can be viewed by selecting the option "threed" from the main menu "Layout" or by selecting the "3D Window" option from the main Window.

 

 

At present the 3D window provides only a few options since most viewing of 3D data sets will be done using the 2D plot and the macro "plot3dslices" which allows the viewing of planes from a 3D data set.

 

However the 3D plot does provide 2 special display modes. The first provides visualization of the surface of a 2D plot, (that is it represents the amplitude of the 2D data as a height), while the second displays isosurfaces from 3D data sets (i.e. surfaces with a constant amplitude).

 

 

 

The coordinate system in the 3D window is such that the observer views the data from different directions while the data remains stationary. The coordinate system defaults to having the x axis horizontal the y axis vertical and the z axis coming out of the screen.

 

 

There are several ways of modifying the viewing position you can:

 

i)                    Move around the vertical axis (the azimuthal angle). This is done by dragging the mouse horizontally.

ii)                   Move above or below the horizontal plane (the angle of elevation). This is done by dragging the mouse vertically.

iii)                 Change the observer’s viewing angle (the twist angle) - 180 degrees means you are standing on your head!). This is done my dragging the mouse in a circle while holding down the shift key.

iv)                 Changing your viewing distance from the object. This is done by turning the mouse scroll-wheel.

 

These 4 parameters are updated and reported in the 3D status bar at the bottom of the plot.

 

Note that azimuthal and twist angle can take any value from  0 to 359.9 degrees while the angle of elevation is limited to ± 90 degrees. The default view angles are 0 degrees and the default viewing distance is 200 units. You can change the sensitivity of these adjustments by modifying the values in the preferences dialog.
