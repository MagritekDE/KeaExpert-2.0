Text-Editor   View Menu

 

 

 

These commands allow rapid access to different parts of macro or modify the way the editor appears to the user.

 

 Go to line number jumps the view to the specified line (starting from line 1).

Go to procedure can be used to jump rapidly from the procedure call to the procedure itself. Just placed the cursor somewhere on the procedure call and then double click the mouse while the control key is held down.

 

Before control-double clicking

After control double-clicking

 

Note that the title bar reflects the current procedure the cursor insertion point is in.

 

Go back jumps the view to the last point at which the “Go to procedure” option was chosen, while “Go forward” reverses this step.

 

Increment font size  increases the current text editor font size by 1 point.

 

Decrement font size  decreases the current text editor font size by 1 point.

The muti-edit option allows the display of a number of edit windows by subdividing the main window. Just select the desired layout from the submenu

 

 

Each subwindow acts independently of the other and the menus relate to the editor with the current focus (i.e. the one you clicked on last). The dividing bar can be moved with the mouse to change the relative sizes of the two regions.

 

Command Help. Detailed information about a particular command can be found by placing the insertion point somewhere in the command name and selecting this option (or type F1).
