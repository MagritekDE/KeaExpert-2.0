Text-Editor   Edit Menu

 

 

 

The first four commands in this menu are the usual Copy and Paste  commands which are found on most text editors. These allow the copying and pasting of selected text from one part of the window to the other or from the text editor window to another application window which accepts text. Cut removes the selected area of text (but still allows it to be repasted) while undo undoes the last action.

 

Copy selection to a new window opens a new text editor window and copies the selected text in the old window to the new one.

 

Select all selects all text in the current text editor.

 

 

 

 

 

indent/unindent  adds/removes 3 spaces infront of each selected line. Note that Prospa does not use real tabs. In the following example indenting has been applied twice to improve code readability.

 

# Load the plot preferences

bak = getcwd()

cd("$prefdir$\\Core Macros")

try

plotPreferences()

tracePreferences()

catch

endtry

cd(bak)

Find up/down searches for the next/last instance of the word under the cursor, while Find and replace allows instances of a word to be replaced with another.

 

 

 

Tabs to spaces allows text generated or edited using another editor to become “tab free” (the Prospa standard).
