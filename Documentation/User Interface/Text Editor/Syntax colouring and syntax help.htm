Syntax Colouring and Help

 

As you will have noticed, Prospa automatically performs syntax colouring on macro text.

 

  • Comments are red
  • Control statements are blue
  • Internal commands are green
  • DLL commands are violet
  • macros and procedures are cyan
  • Other statements are black

 

In addition, when you type a command or if you click on one, then the syntax for this command will be listed in the status-bar.  More detailed help for a command can be found by pressing F1 while the insertion point is in the command name, or by browsing through the main help file available from the main Prospa window Help menu.
