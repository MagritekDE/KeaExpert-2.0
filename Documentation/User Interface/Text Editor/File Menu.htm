Text-Editor   File Menu

 

 

 

Prospa macros or scripts can be loaded into the editor using the open macro commands or by dragging and dropping the macro file into the editor.

 

Open Macro

 

The Open macro command will display a standard open dialog in which you can choose from a variety of text file types *.mac (the standard macro extension), *.txt - generic text files, *.par – parameter files, *.lst – list files and *.pex – Prospa executable macros. (These the same as normal macros except that they will be executed when opened.)

Save Macro and Save Macro As

 

The save macro commands will save the macro using the current name while the Save macro as command will display a standard Save dialog so you can choose a new folder or filename.

 

The save as macro dialog

 

 

 

Run Text and Save and Run Text

 

Prospa macros can be stored in a file or just displayed in the text editor. The Run Text menu option will run the macro from the text displayed in the current text editor. The Save and Run Text will save the macro text to a file first. You will prompted with the Save a macro dialog (see above) first.

 

Note that there are some limitations to the Run Text option.

 

Print Macro

 

This sends the contents of the current text editor window to the selected printer.

 

Close Macro

 

This erases the current macro but will first ask if you wish to save the existing macro if it has been edited (this is signified by an asterisk at the end of the filename.

 

Close All Macros

 

As explained in the View menu the text editor window can be split into any number of parts. Each of these parts can display a different text file. Running this command will close all visible macros replacing the multiple view with a single one.

 

 

The text editor supplied with Prospa supplies a number of specialized editing features. These may be accessed using the menu or short-cut features
