Text-Editor   Introduction

 

Prospa includes a facility for combining commands together to form more powerful scripts or macros. This provides a full interpreted programming facility.

 

To write a command script you can use any word-processor that can save in simple text format. However Prospa also includes a built-in editor for this purpose which includes context colouring, syntax prompting and a variety of other features to make editing easier.

 

Macros should be saved with the extension ".mac". An example macro script is shown below:

Note that any characters after the  "#" symbol are ignored and treated as comments.

 

To use this macro you would simply type the following at the command line interface prompt

 

> (x,y) = cosine(1024,20,100,5)

> plot(x,y)

 

Note that it is not necessary to specify the macro extension if ".mac" was used.

 

 

The resulting plot is shown above. If the macro script is stored in a folder which is in the macro search path (as defined in the main File->Preferences menu) then the program will always be able to find it. Alternatively you can make the script directly accessible from the main menu. This method is explained in the main menu preferences section.
