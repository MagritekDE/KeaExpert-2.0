How to find and delete your Prospa preferences folder

 

 

 

Note:

 

·         If your Prospa installation is more recent that July 2012 there will be a menu item in the main files menu which takes you directly to the Prospa preferences folder.

 

·         If your Prospa installation is more recent than October 2012 there will be a batch file in the install folder called movePref.bat. Running this manually or during the install process will move the previous preferences folder to a backup folder called OldProspaPref. This will be stored in the same location as the original Prospa preferences folder.

 

The following discussion is relevant in those cases where Prospa cannot be started because of a preferences file fault and if the batch file movePref.bat is not present or doesn’t work.

 

The preferences folder can be found by running Prospa and in the command line interface typing

> pr prefdir

However if you can't run Prospa then first go to the Windows Start menu and select the Run option. In the run window type "cmd". Then in the command window type

> set

Find the output line starting

USERPROFILE=

Take the directory path listed to the right of this and add "\Application Data\Prospa" to get the full Prospa preferences path. You can safely delete this folder before running Prospa again. Now sometimes (in Vista or Windows 7), this will be an alias for some other folder. e.g on my Vista system the preferences folder is

C:\Users\CEccles.Craig-PC\Application Data\Prospa

Windows also knows this as

C:\Users\CEccles.Craig-PC\AppData\Roaming\Prospa

These folders can sometimes be hidden so make sure you select the "Show hidden files and folders options" in the Explorer->Tools->Folder Options dialog



While you are sorting out your preferences it is a good idea to make a shortcut to the parent folder so you don't have to search for it all the time.
