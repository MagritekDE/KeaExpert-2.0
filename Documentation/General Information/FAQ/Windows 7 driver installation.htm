Windows 7/Vista Terranova, Aurora or Kea USB Driver Installation

 

 

To install the Windows 7/Vista USB driver:

 

1. Plug in the spectrometer USB cable to your Windows 7 computer and switch the spectrometer on. Wait until the "plug and play" USB driver installer says that it can't find the driver.

 

2. Right click on the "Computer" icon and select Properties. Select Device Manager:

 

 

 

3. Search for the Unknown device icon and right click on it:

 

 

4. Select "Update Driver Software"

 

5. Select Browse my computer for updated driver software.

 

6. Browse to the USB driver folder in your new Prospa install

 

 

7. Press Next and then select "Install this driver software anyway"

 

 

8. Success!
