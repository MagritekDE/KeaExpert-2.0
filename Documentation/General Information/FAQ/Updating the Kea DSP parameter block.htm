Updating the Kea DSP parameter block

 

 

From 2011 new Keas save data about the Kea system in the DSP flash memory. In this way you don't have to remember things like the serial number, type of Kea (low frequency, high frequency ...), firmware versions and so on.

 

Older Keas do not have this information and to run the latest version of the Kea software you will need to update the parameter block on your Kea's DSP otherwise you will get the following error message when trying to use the new software:

 

 

 

Here are the steps to update the parameter block:

 

  1. Install the new version of Kea software onto your PC.
  2. Open the Prospa preferences dialog File->Prospa preferences

 

 

  1. In the Menus tab (above) press the "Add name" button and add an item "Kea Update"

 

 

  1. Press the Add folder button and select the Kea-Update folder within the Prospa Macros folder (press the Macros button on the left to get quick access to this folder)

 

 

  1. Press OK and then press the Save button on right of the preferences dialog to update the menu preferences. You will now have a new menu called Kea-Update:

 

 

Select the DSPParameterUpdater option and press the "Read DSP parameters" button. You should see something like the following if your DSP parameter block has never been written to.

 

 

Select the "Load from file" option on the right and select your Kea type from the list.

Once loaded update the serial number to match the one on the rear of the spectrometer (Kxxx) ignoring the leading K.  (Note LF = 0-1 MHz,  HF = 1-100 MHz, VHF = 1-400 MHz).

 

The final version should look something like this for a dual channel 1-100 MHz (HF) Kea 2).

 

 

(your serial number will be different). Press the Write DSP parameters to save the data to the DSP parameter flash. Note that doing this cannot damage your DSP.

 

This information will be used whenever you run a Kea NMR experiment and ensures that the correct software is always used.
