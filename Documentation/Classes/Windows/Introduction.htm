Prospa window classes

 

Prospa windows are built up of a number of controls. These objects in turn are built up of parts. These windows, controls and parts form a hierarchy as shown below.

 

                                  window

                        │

                  ┌─────┴──────┐

               controls     properties

              ┌───┴──────┐

              │          │

            parts    properties

              │

           properties

 

 

Each non-property term in this diagram is an object which is described by a template called its class. In the following list all the classes currently defined in Prospa which are either windows or controls are listed. Clicking on one of these classes will display another page in which all the class properties as listed. These properties allow the setting and getting of window, object, and object part parameters such as width, color etc.
