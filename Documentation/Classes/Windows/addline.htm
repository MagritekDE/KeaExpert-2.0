addline

 

Add a line to the plot region

 

Syntax

 

plotRegionName->addline(x0, y0, x1, y1, [color, width, style, [units]])

 

x0, y0 .... start coordinate for line (see units) (FLOAT/INT)

x1, y1 .... end coordinate for line (see units) (FLOAT/INT)

color ..... line color (rgb vector 0-255 per color) default is [255,0,0] = red (VEC)

width ..... line width (integer 1, 2, ...) (INT)

style ..... line stype ("dashes/dots/solid/dashdot/dashdotdot") default is "solid" (STR)

units ..... units: 0 = plot units, 1 = pixels, 2 = plot units (x) and fractional height (y).  Default is 0. (INT)

 

Description

 

A line will be drawn from (x0,y0) to (x1,y1) using the coordinate system specified. The line will be drawn with RGB color color of width width and style style.

 

Special case values

 

If you want to draw lines which always reach the edge of the plot you can use the inf coordinate (infinity) this means that if you scale the data the line will remain connected to the edge of the plot.

 

Example

 

curplot("1d")->plot(noise(100))

curplot()->addline(0,1,100,1)

curplot()->addline(20,inf,20,-inf,[0,128,0])

 

This draws a noisy data set with SD = 1 and then draws a horizontal line on top with this value. It also draws a vertical line in dark green which will always span the full plot, even after zooming out – as shown below.

 

 

An alternative to using the inf parameter is to change units to option 2. However this also requires specifying the color, line width and style.

 

curplot("1d")->plot(noise(100))

curplot()->addline(0,1,100,1)

curplot()->addline(20,0,20,1,[0,128,0],1,"solid",2)

 

Here the line goes from 0% (0) of the plot height to 100% (1) of the plot height.
