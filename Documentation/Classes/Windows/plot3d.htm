The plot3d control

 

The plot3d control is one of a number of objects which can make up a window. A plot3d is generated using the plot3d command.

  

Syntax

 

plt = plot3d(objNr, x, y, w, h)       (eeeee:ddddd)

 

nr ............ control number ID (INT)

x,y ........... top left coordinates of control in pixels (INT)

w,h ........... width and height of control in pixels     (INT)

plt ........... access to plot internal functions and parameters (optional)

 

Description

 

This command adds a three-dimensional plot control to the current GUI window being edited. This can be used for displaying 3D line data or image data as a surface or as a volume. The plot can be referred to using its unique ID, nr. Its location and size in the parent window (in pixels) is determined by the (x,y) (w,h) integer parameters.

 

Additional plot parameters can be set or retrieved using the plot object handle, which can be obtained using either of the following methods:

 

obj = plot3d(ctrlNr,x,y,w,h)

obj = getobj(winNr, ctrlNr)

  

Once the plot object handle has been obtained, you can access the plot parameters using the syntax

 

result = obj->parameter()

obj->parameter(value)

 

where parameter is the name of the parameter you wish to access. The first form gets the current parameter value while the second sets the parameter value.

 

Note that the 3D plot must be initialised before it may be used. This can be done by invoking the initialise function on the plot object with an empty string parameter, i.e.,

 

obj = plot3d(ctrlNr, x,y,w,h)

obj->initialise("")

 

Class function list

 

Following is a list of the available plot3d parameters. Note that the results of these commands are identical to those obtained using the setpar and getpar functions.

 

All parameters must be prefixed with the object reference operator "obj->" where obj is whatever name you have chosen for the control object. As in the above two examples the "set" version requires a value be added as an argument while the "get" version will return the value.

 

initialise(STR init) ............ initialise this 3D plot window.

titleupdate(STR tf) ............. whether the plot window title should be updated when a control is selected. ("true"/"false")

 

In addition, all the generic parameters listed in the control class page can applied to the button object.

 

Details

 

initialise

 

Initialise this 3D plot window. This function must be called with an empty string parameter prior to plotting in the window.

 

titleupdate

 

Whether the plot window title should be updated when a control is selected. ("true"/"false").
