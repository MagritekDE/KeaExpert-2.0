The trace object

 

The trace object is a single (x,y) data set which is part of a plotregion object. In addition to retrieving or setting this data, its display properties may be manipulated using the trace object functions which are listed below.   

 

Class hierarchy for the trace object

Class function list for trace objects

 

Note : All functions should be prefixed with the trace object name e.g. tc->tracecolor([255,0,0]). Any function which does not explicitly state that it gets data will return the current values when passed an empty argument. e.g. INT sz = symbolsize().

  

OBJ rgn = parent() ................   get the trace plot-region parent

(VEC x, VEC y) = getdata() ........   get the trace data

INT id() ..........................   get the id number of this trace

axis(STR "left"/"right") ..........   get or set the current y axis of this trace

color(VEC [r,g,b]) ................   trace color  (real + imag)

realcolor(VEC [r,g,b]) ............   trace color  (real)

imagcolor(VEC [r,g,b]) ............   trace color  (imag)

inlegend(STR "true"/"false") ......   get or set whether the trace appears in the plot legend

tracetype(INT t) ..................   trace line style (real) (0-4)

                                    lines/stairs/dots/none

tracewidth(INT w) .................   trace width

realstyle(INT dash) ................   real trace line dash mode (0-4)

imagstyle(INT dash) ................   imag trace line dash mode (0-4)

 

antialiasing(STR tf) ..............   antialiasing enabled (true/false)

symbolshape(STR shape) ............   symbol shape:

                                    opensquare/square/opencircle/circle/

                                    opentriangle/triangle/openinvtriangle/invtriangle/

                                    opendiamond/diamond/cross/plus/none

symbolcolor(VEC col) ..............   symbol color

symbolsize(INT sz) ................   symbol size (1-19)

ebarshow(STR "true"/"false") ......   show or hide error bars

ebarfixed(STR "true"/"false") .....   use fixed or variable length error bars

ebarcolor(VEC [r,g,b]) ............   error bar color

ebarsize(FLOAT sz) ................   length of fixed error-bars

ebararray(VEC eb) .................   array of error-bar lengths

 

Also note that these functions only apply to the specified trace object. To make them apply to all subsequently drawn traces, use the tracepref command.
