The statictext control

 

The static text control is one of a number of objects which can make up a window. Static text is generated using the statictext command.

 

Syntax

 

obj = statictext(nr,x,y,justification,label) (eeeee:dddss)

 

nr ............ control number (INT)

x,y ........... top left coordinates of text in pixels (INT)

justification . right/left/centre

label ......... text to appear in window.

 

Description

 

This command adds a static text label to the current GUI window being edited. The statictext can be referred to using its unique ID, nr. Its contents are specified by label. Its location in the parent window is determined by the (x,y) integer parameters and left, centre, or right justified to that coordinate. By default, justification is left.

 

Additional statictext parameters can be set or retrieved using the statictext object handle which can be obtained using either of the following methods:

 

obj = statictext (ctrlNr,x,y,justification,label)

obj = getobj(winNr, ctrlNr)

  

Once the statictext object handle has been obtained, you can access the statictext parameters using the syntax

 

result = obj->parameter()

obj->parameter(value)

 

where parameter is the name of the parameter you wish to access. The first form gets the current parameter value while the second sets the parameter value.

 

Class function list

 

Following is a list of the available statictext parameters. Note that the results of these commands are identical to those obtained using the setpar and getpar functions.

 

All parameters must be prefixed with the object reference operator "obj->" where obj is whatever name you have chosen for the control object. As in the above two examples the "set" version requires a value be added as an argument while the "get" version will return the value.

 

text(STR label) ............. the contents of the statictext.

justification(STR text) ..... the justification of the contents of the statictext with regard to its x,y position.

 

In addition, all the generic parameters listed in the control class page can applied to the button object.

 

Details

 

text

 

The contents of the statictext label.

 

justification

 

The positioning of the contents of the statictext label around its x,y position.

 

left …...  The top left of the first character in the statictext is positioned at the x,y position.

right......  The top right of the last character in the statictext is positioned at the x,y position.

centre....   The middle of the statictext is positioned at the x,y position.

  

Example

 

n = window("test",100,100,200,100)

t = statictext(1,80,20,"right","Justification")

b = button(2, 70, 55, 50,20,"Change", :swapJustification(t))

showobjects(n)

 

procedure (swapJustification, t)

  if (t->justification() == "left")

    t->justification("right")

  else

    t->justification("left")

  endif

endproc

 

 

This example demonstrates the use of the left and right justification options.
