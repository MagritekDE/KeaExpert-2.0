The plot-region object

  

The plot region object represents an individual area (region) in the plot1d control. In addition to retrieving or setting this data, its display properties may be manipulated using the plotregion object functions which are listed below.   

 

Class hierarchy for the plot-region object

Note: All functions should be prefixed with the region object name e.g. rg->ylabel("my label"). Any function which does not explicitly state that it gets data can also return the current values by passing a empty argument. e.g. VEC col = bordercolor().

 

Generic class functions for 1D and 2D plot-region objects

 

addline(FLOAT x0, y0, x1, y1, ... . add a line to the current plot

        VEC color, INT width,       between (x0,y0) and (x1,y1)

        STR style)                  of defined color, width and style

autorange("true"/"false") .......... set the x/y scale range  automatically or not.

OBJ ax = axes() .................... get the axes object

border("show/"hide"/"toggle") ...... change border visibility

bordercolor(VEC [r,g,b]) ........... border color e.g. [128,128,128]

bkgcolor(VEC [r,g,b]) .............. background color e.g. [0,0,128]

clear() ............................ clear the plot.

dim(INT d) ......................... get data dimension (1 or 2)

draw("true"/"false") ............... update plot after each command

STRING filename()................... get file name for plot region

INT fileversion() .................. return the current file version for loaded file.

STRING filepath()................... get file path for plot region

OBJ gd = grid() .................... get plot grid object

INT height = height() .............. get the plot region height in pixels

load(STR filename) ................. replace 1D or 2D plot with plot file (note: all plot references will be deleted).

margins(INT left,top,right,base) ... get or set margins for this plot.

OBJ plt = parent() ................. get plot parent object

(INT x,y) = position() ............. get plot region position in plot object matrix

rmlines() .......................... remove all lines added to the plot with addline

save(STR filename) ................. save the plot region to a file (1D or 2D). 

OBJ lbl = title() .................. get the title object

title(STR txt) ..................... set the title text

INT width = width() ................ get the plot region width in pixels

INT x_offset = x() ................. get the x offset of the region in pixels

OBJ lbl = xlabel().................. get the x-label object

xlabel(STR txt) .................... set the x-label text

OBJ lbl = ylabel().................. get the y-label object

OBJ lbl = ylabelleft().............. get the left y-label object

OBJ lbl = ylabelright()............. get the right y-label object

INT y_offset = y() ................. get the y offset of the region in pixels

ylabel(STR txt) .................... set the current y-label text

ylabelleft(STR txt) ................ set the left y-label text

ylabelright(STR txt) ............... set the right y-label text

ylabelvert(STR "true"/"false") ..... whether plot y-label will be vertical or not

 

Class function list for the 1D plot-region object

 

antialiasing1d("true"/"false") ..... enable or disable antialiasing

currentaxis(STR "left"/"right") .... set or get the current axes for plotting.

fullregion() ....................... display the complete dataset

(VEC x,y) = getdata() .............. get current region trace data

hold("on"/"off") ................... allow trace overlapping (or not)

lockgrid(STR "true"/"false") ....... whether left and right grid will be locked together.

INT id = plot(VEC x [,VEC y]) ...... set plot data returning a trace index

OBJ tc = trace(INT id) ............. get a trace using zero-based index

tracepref(parameter, value) ........ set or get trace preferences (see tracepref cmd).

VEC lst = tracelist() .............. get the list of trace indices

tracexborder(FLOAT factor) ......... fraction of x axis used as border (0-1 default 0.0)

traceyborder(FLOAT factor) ......... fraction of y axis used as border (0-1 default 0.1)

rmtrace(INT id) .................... remove trace with specified ID.

showlegend(STR "true"/"false") ..... whether plot legend will be displayed.

syncaxes(STR "true"/"false") ....... whether left and right axes will be synchronised.

zoom(x1,x2[,y1,y2]) ................ select a region to display

 

Class function list for the 2D plot-region object

 

cmap(VEC color_map) ................ get or set the color-map for the 2D plot.

contour(VEC levels, STR mode) ...... get or set the display mode and contour levels (see cmd contour)

(MAT m, VEC x, y) = getdata() ...... get current region image data

image(MAT m, VEC x, y) ............. set current region image data

imagerange(FLOAT min, max) ......... set the range of image values to map to the color scale.

showcmap(STR "true"/"false") ....... get or set the visibility of the color scale.

zoom(x1,x2,y1,y2) .................. select a region to display
