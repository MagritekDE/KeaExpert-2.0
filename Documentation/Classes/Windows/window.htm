The window class

 

A window is a container for other graphical objects. It can be moved about the screen, resized, closed, zoomed and stacked above or below other windows. All windows in Prospa are created using the window command that returns a number unique to that window. The command getobj can be used to extract the window object. This is a handle which can be used to extract and modify information about the window such as its position.

 

n = window(x,y,w,h,name)

obj = getobj(n)

 

or

 

(n,obj) = window(x,y,w,h,name)

 

Once the window object has been obtained you can access the window parameters, including any embedded controls, using the syntax

 

result = obj->parameter()

obj->parameter(value)

 

where parameter is the name of the parameter you wish to access. The first form gets the current parameter value while the second sets the parameter value.

 

Here is a list of the window parameters with links (where necessary) to their corresponding help pages. Note that the results of these commands are identical to those obtained using the setwindowpar and getwindowpar functions.

 

All parameters will need to be suffixed with the object reference operator "obj->" where obj is whatever name you have chosen for the window object. As in the above two examples the "set" version requires a value be added as an argument while the "get" version will return the value. Those parameters which start with the word "get" in their description cannot be used in the "set" form.

 

VEC col = bgcolor() ........ get the window background color (RGBA)

INT sz = bordersize() ...... the border size in pixels.

VEC lst = ctrllist() ....... get a list of control numbers for this window.

VEC dim = dimensions() ..... the window dimensions returned as an array.

LST dim = dimensionstxt() .. the window dimensions returned as an list.

INT gs = gridspacing() ..... pixel spacing of the window's grid

INT h = height() ........... the height of the window in pixels.

STR name = macroname() ..... get the name of the macro used to define the window.

STR path = macropath() ..... get the directory of macro used to define the window.

STR mw = mainwindow() ...... whether the window is the main window ("true"/"false")

VEC bar = menubar() ........ the list of menus (by number) to display in the menu bar.

STR n = name() ............. the window name.

INT n = winnr() ............ get the window number.

STR state = permanent() .... whether the window can be deleted ("true"/"false").

VEC pos = position() ....... (x,y) coordinates of the window.

INT w = width() ............ the width of the window in pixels.

STR rs = resizable() ....... whether the window can be resized ("true"/"false").

STR sg = showgrid() ........ whether the window grid is visible ("true"/"false").

VEC sl = sizelimits() ...... the max and min width and height of the window ([minWidth, maxWidth, minHeight, maxHeight])

STR stg = snaptogrid() ..... whether grid-snapping is enabled ("true"/"false").

STR txt = title() .......... the window title.

INT bh = titlebarheight() .. get the height of the window title bar in pixels.

STR tu = titleupdate() ..... whether the window title updates on control selection.

STR vis = visible() ........ whether the window is visible ("true"/"false").

INT left = x() ............. the screen coordinate of window’s left edge.

INT top = y() .............. the screen coordinate of window’s top edge.
