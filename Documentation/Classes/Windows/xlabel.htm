The xlabel object

 

The xlabel object represents the label along the x axis of a plot region. It is part of a plotregion object. In addition to retrieving or setting this data, its display properties may be manipulated using the xlabel object functions which are listed below.

 Class function list for xlabel objects

 

Note : All xlabel parameter functions should be prefixed with the xlabel object name

e.g. lb->color([255,0,0])

 

OBJ rgn = parent() ........ get the xlabel plot-region parent

VEC col = color() ......... the xlabel color e.g. [255,0,0]

INT sz = size() ........... the xlabel font size

STR name = fontname() ..... the xlabel font name e.g.

                            "Times New Roman"

                            "Courier New"

STR name = style() ........ the font style. A string consisting of selections from:

"bold"

"italic"

"underline"

"regular"

separated with the '+' character.
