The statusbox control

 

The statusbox is one of a number of objects which can make up a window. A statusbox is generated using the statusbox command.

 

Syntax

 

obj = statusbox(nr) (e:d)

 

nr .... object number (INT)

obj ... access to statusbox internal functions and parameters (optional)

 

Description

 

The status box is a strip at the bottom of a window into which information may be written. It is often used to report the status of a macro while it is running. This command adds a statusbox to the current GUI window being edited. The statusbox can be referred to using its unique ID, nr.

 

Additional statusbox parameters can be set or retrieved using the statusbox object handle which can be obtained using either of the following methods:

 

obj = statusbox (ctrlNr)

obj = getobj(winNr, ctrlNr)

  

Once the statusbox object handle has been obtained, you can access the statusbox parameters using the syntax

 

result = obj->parameter()

obj->parameter(value)

 

where parameter is the name of the parameter you wish to access. The first form gets the current parameter value while the second sets the parameter value.

 

Class function list

 

Following is a list of the available statusbox parameters. Note that the results of these commands are identical to those obtained using the setpar and getpar functions.

 

All parameters must be prefixed with the object reference operator "obj->" where obj is whatever name you have chosen for the control object. As in the above two examples the "set" version requires a value be added as an argument while the "get" version will return the value.

 

text(STR contents) ....... sets the contents of the statusbox

syntaxwindow() ...........

 

In addition, all the generic parameters listed in the control class page can applied to the statusbox object.

 

Details

 

text 

 

contents is the new text string to be displayed in the statusbox.

 

syntaxwindow 

 

Buttons may have several special functions specified by the mode parameter.

 

Example

       

n = window("test",100,100,200,100)

e = editor(1,0,0,200,70)

s = statusbox(2)

showobjects(n)

showwindow(n)

s->text("window displayed")

s->syntaxwindow(n)
