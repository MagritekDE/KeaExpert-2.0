The grid object

  

The grid object represents a grid drawn on the background of a plot region. It is part of a plotregion object. In addition to retrieving or setting this data, its display properties may be manipulated using the grid object functions which are listed below.

 Class function list for grid objects

 

Note : All grid parameter functions should be prefixed with the grid object name

e.g. gr->ygrid("on")

 

OBJ rgn = parent() ..................      get the grid plot-region parent

VEC col = color(VEC rgb) ............      main grid color

VEC col = finecolor(VEC rgb) ........      fine grid color

STR state = xgrid(STR state) ........      x-grid state ("on"/"off")

STR state = ygrid(STR state) ........      y-grid state ("on"/"off")

STR state = finexgrid(STR state) ....      fine x-grid state ("on"/"off")

STR state = fineygrid(STR state) ....      fine y-grid state ("on"/"off")
