The control class

  

A control is one of a number of objects which make up a window. Each control has its special functions and features but all share certain parameters. Each control is created using a different command which returns the control object. Alternatively if the control number is know the command getobj can be used to extract the control object.

 

obj = control(ctrlNr,x,y,w,h,...)

obj = getobj(winNr, ctrlNr)

 

Once the control object has been obtained, you can access the control parameters, including any embedded objects using the syntax

 

result = obj->parameter()

obj->parameter(value)

 

where parameter is the name of the parameter you wish to access. The first form gets the current parameter value while the second sets the parameter value.

 

Following is a list of the generic control parameters with links (where necessary) to their corresponding help pages. Note that the results of these commands are identical to those obtained using the setpar and getpar functions.

 

All parameters will need to be suffixed with the object reference operator "obj->" where obj is whatever name you have chosen for the control object. As in the above two examples the "set" version requires a value be added as an argument while the "get" version will return the value. Those parameters which start with the word "get" in their description cannot be used in the "set" form.

 

STR ac = active() .........  whether the control is active or not ("true"/"false").

INT n = ctrlnr() ..........  The control number. This number should be unique to each control.

STR name = ctrlid() .......  ID with which to extract the control's object.

STR txt = ctrltype() ......  get the type of control.

STR en = enable() .........  whether the control is enabled ("true"/"false").

STR ac = eventactive() ....  whether the control responds to events or not ("true"/"false").

STR txt = event() .........  the event which cause the control to execute its call-back procedure.

STR type = datatype() .....  the type of data expected by the control.

VEC dim = dimensions() ....  get the location and size of the control as a vector [x,y,w,h].

LST dim = dimensionstxt() .  get the location and size of the control as a list of expressions [x,y,w,h].

draw(STR d) ...............  set whether the control should be redrawn or not ("true"/"false").

STR status = focus() ......  whether this control has the keyboard focus ("true"/"false").

INT h = height() ..........  the height of the control in pixels.

STR h = h_exp() ...........  the control's height as an expression.

STR txt = label() .........  the control's label.

STR n = name() ............  old form of valueid().

VEC mb = menubar() ........  the menu bar (as an array of control numbers) associated with this control.

CLS par = parent() ........  gets the window parent.

STR txt = procedure() .....  the callback procedure for the control.

STR ro = readonly() .......  whether the controls contents are readonly or not.

VEC r = range() ...........  the minimum and maximum allowable values for the control.

VEC r = region() ..........  specify the controls surrounding this one for the purposes of resizing.

INT n = statusbox() .......  the statusbox associated with this control.

VEC [tb,pg] = tabparent() .  gets the parent tab control number and tab page number (both zero indexed).

INT n = tabnr() ...........  the tab number for the control (keyboard tab).

STR txt = tag() ...........  a user definable tag string.

INT n = toolbar() .........  the toolbar associated with this control.

STR name = valueid() ......  ID with which to extract the control's value.

STR vis = visible() .......  whether the control is visible ("true"/"false").

INT wn = winnr() ..........  gets the parent's window number.

INT w = width() ...........  the width of the control in pixels.

STR w = w_exp() ...........  the control's width as an expression.

INT left = x() ............  the window coordinate of control left edge.

STR left = x_exp() ........  the control's x coordinate as an expression.

INT top = y() .............  the window coordinate of control top edge.

STR top = y_exp() .........  the control's y coordinate as an expression.
