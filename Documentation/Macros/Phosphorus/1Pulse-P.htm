Experiment : 1Pulse-P

 

Description

Pulse and collect experiment for Phosphorus 31. Generates an FID from which a spectrum can be calculated.

 

Pulse program

 

A single RF pulse followed by a short ringdown delay following by complex data acquisition:

 

pulse-acqDelay-acquire

 

Parameters

 

Pulse sequence specific parameters

·         B1 frequency – frequency of RF pulse.

·         Repetition time – time between scans (in ms)

·         Pulse amplitude – size of RF pulse in dB (0 dB = 1 mW)

·         Pulse length – pulse duration in microseconds

·         Pulse acquisition delay – delay between end of RF pulse and start of acquisition (in microseconds). Should be long enough to avoid ring-down from the probe and receiver recovery time.

 

Acquisition parameters

·         Receiver gain – internal receiver gain setting in dB. Should be adjusted to avoid overload and give best SNR. Note that signal level will not change significantly when this parameter is changed as the displayed voltage is the probe voltage.

·         Receiver channel – specifies which nucleus should be detected.

·         Receiver phase – a phase shift in degrees applied to the data before collection by the computer. Can be adjusted to give required frequency domain phasing.

·         Number of points – the number of complex data points to be collected.

·         Dwell time – sampling time. i.e. time between each data point. Units are microseconds.

·         Number of scans – the number of times the experiment will be repeated.

·         Flat filter – select this if you want all frequencies to be equally weighted. Deselect if you expect all the data to be close to the central frequency.

·         Accumulate data – whether scans should be added coadded to improve SNR or reduce artefacts.

·         Phase cycle – whether the phase cycle specified in the pulse sequence should be applied.

·         Bandwidth – the collected spectral frequency range centred on the B1 frequency. (Read only)

·         Acquisition time – the total duration of the FID in milliseconds. Equal to the number of points multiplied by the dwell time. (Read only)

 

Processing parameters

·         Zero fill factor – the FID can be filled with zeroes to improve resolution in the frequency domain. The parameter determines the total length of the FID. 1 means it will not be filled with zeros, 2 it will have equal number of zeros as fid and so on.

·         Apodization filter – a window function applied to the non-zero filled FID to improve SNR or bring out specific spectral features.

·         Filter type- the particular apodization filter to be applied.

·         Frequency domain phasing – whether the frequency domain with be autophased or have its magnitude taken. The 'none' option does not apply a phase correction.

 

Display parameters

·         Use ppm scale – if selected the frequency domain data will be reversed and scaled to show a parts per million. The central frequency may be adjusted using the Set reference tool.

·         Minimum ppm value – if the ppm scale is selected it will range from this value to Maximum ppm value.

·         Maximum ppm value – see Minimum ppm value.

·         Display range (kHz) – if the Use ppm scale option is not selected then the display range is set by this parameter. It will center the scale about the b1 frequency. Setting this parameter to zero will display the full spectral range (i.e. the bandwidth)

 

File parameters

·         Save data – if selected the data will be saved at the end of the experiment.

 

Post processing

Several options are provided for post processing the collected FID and spectrum. Refer to the help in the relevant dialog for more details

·         R/C – toggles the display of real or complex data.

·         Save – saves the currently selected plot and moves the temporary proc.par file to proc.par.

·         Export – allows plot data to be exported as either an ASCII (text) file or as a .1d Prospa file.

·         Phase – allows the adjustment of zeroth and first order phase correction parameters for the spectrum.

·         Apodize – allows a variety of apodization filters to be applied to the spectral data.

·         SNR – measures the signal to noise ratio of the FID or the spectrum. Based on measuring the maximum amplitude and dividing by the standard deviation of the data at the end of the FID or the start of the spectrum.

·         ppm/Hz – toggles the display of parts per million or Hz in the spectrum.

·         Set reference – allows the zero point in the spectrum to be set if in ppm mode.

·         Peak picking – allows a list of peaks to be selected automatically or manually in the spectrum.

·         Integration – allows a list of manually integrated  peaks to be generated
