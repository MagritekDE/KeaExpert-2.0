qNMR Developer

The qNMR developer tool allows the user to develop parameters needed to measure the purity of a sample calibrated with an internal standard. The interface runs a 1Pulse experiment with hard coded acquisition parameter and uses the resulting spectrum to calculate purity. Once appropriate parameters are determined for a compound and reference procedure they can be published to a fixed procedure that can be read by the simple interface. The simplified interface only allows the user to enter weights for the reference and sample and the purity of the reference standard.

Introduction to qNMR

 Quantitative NMR is a method of using NMR signal amplitudes to derive quantitative measurements. This is possible because the signal detected by the NMR spectrometer (INMR) is proportional to the number of active nuclei in the sample (Nnuc). For practical purposes we will use the proton (hydrogen nucleus) here, but technically this can be applied to any NMR active nucleus. Because molecules have a consistent and predictable number of protons this signal (I) is proportional to the number of molecules (Nmol) in the sample.

Once you know the number of nuclei in your sample you can use that information to calculate concentrations (e.g. molarity, purity, vol/vol ratio, etc.).

Purity measurement

The goal of a purity measurement is to quantify how much of a material is the desired component compound and how much is composed of impurities. You therefore measure the signal detected of the target component and compare that with the theoretical signal that you would get if the compound was pure. The deficit in signal is due to impurities. The NMR signal is determined by integrating one or more NMR peaks in the spectrum associated with the reference compound or sample. Using the software

Primary control buttons

·         The Run button runs the purity calculation experiment.

·         Abort stops the data collection

·         Close saves the parameters entered into the winder and closes the window

·         Help opens this help document

Publish

This takes the data in this window and saves the parameters to a file in the Publish Directory. This file can be used to load all the values needed to calculate purity for a specific sample using the same reference material and chemical shift reference used in the developer interface.

·         Name will be the name of file saved with the suffix .par appended to it. It will also appear in the pull down list of procedures in the simplified interface

·         The Publish button writes all the parameters to the file. CAUTION: This will overwrite the file without asking.

Developer box

The developer box is used for developing a purity measurement that will be used in the simplified qNMR interface. This allows you to collect one data set and optimize the analysis on that data set without needing to collect a new data set each time.

·         Selecting Load data from file uses data stored in the Data Folder  when calculating the purity

·         The Publish Directory is where the qNMR parameters are stored.

Reference Deconvolution

Checking the Apply box applies reference deconvolution to the acquired spectrum. This does not apply reference deconvolution to saved data. You should save a reference deconvolved data set if you want to use that to develop the qNMR experiment.  Choose the directory where the reference spectrum is stored by clicking on “…” or pasting the directory into the field next to it.

Chemical shift reference

This box allows you to reference chemical shift to a zero ppm reference or the water signal.

·         Checking TMS/TSP will look for the largest a peak between -0.25 and positive 0.25 ppm and set that peak to 0 ppm. This will now work properly if there are lager peaks in that region.

·         Checking Water will set the tallest peak between 4.7 and 4.95 to 4.78ppm. Again, this will now work properly if there are lager peaks in that region.

Reference

This box sets the qNMR parameters for the internal reference.

·         Mass is the mass of the reference material.

·         Mol. Weight is the molecular weight of the reference material.

·         Nuclides is the number of protons in the spectral peak used as the reference.

·         Integration Range is the range that will be integrated to determine the peak integral. The lower ppm value needs to be entered on the left and the higher on the right.

Sample

This box sets the qNMR parameters for the test sample for with the purity is being determined.

·         Mass is the mass of the test sample.

·         Mol. Weight is the molecular weight of the test sample.

·         Nuclides is the number of protons in the spectral peak of the sample used to determine purity.

·         Integration Range is the range that will be integrated to determine the peak integral. The lower ppm value needs to be entered on the left and the higher on the right. This range and the number of nuclides need to be related to the same chemical group.

·         Purity is the purity calculation output calculated from the integral range and number of nuclides data in the same line.

Plot output

The plot window will display the peak ranges that are integrated in separate windows for each peak used in the purity calculation. The final plot will display the entire spectrum.

 

Notes

·         Air buoyancy is not compensated for in the calculation. If you are correcting for this, then enter the corrected weight.
