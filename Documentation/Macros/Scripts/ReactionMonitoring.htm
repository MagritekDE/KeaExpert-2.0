Reaction Monitoring Script

 

This script will run the 1Pulse-H experiment at well-defined intervals, saving the results in separate files which are compatible with MestraNova.  Each experiment may perform several averages (scans) to improve SNR.

The interface has several fields that must be filled:

1.       The number of fids/spectra to collect.

2.       The time in minutes between the start of each set of scans (repetition time per spectrum).

3.       The number of averages per acquired fid/spectrum.

4.       The time in seconds between the start of each average (repetition time per average).

In addition you need to define a project folder into which the data will be stored and a sample name. These can be different from the Project folder defined in SpinsolveExpert.

If you want to maximize the amount of signal averaging just enter a large number for Averages per spectrum and the program will suggest the maximum number possible when you press the Start button.

 

Before starting the reaction monitoring script make sure that you have run a 1Pulse-H experiment in the normal SpinsolveExpert interface with the parameters you want to use during the reaction monitoring scans.

Press the Start button to begin the experiment. The left progress bar indicates the time to wait before the next set of averages and the progress bar on the far right tracks the overall progress. The Pause button will temporarily halt the experiment but the timing may be messed up if you wait too long to continue. The Abort button will stop the experiment after the current scan.

Note that as the script is currently written, data is not saved inside the SpinsolveExpert environment (apart from acqu.par files), but only at the location indicated in the script interface.  It is expected that the data will then be exported to other software, in particular MestraNova.

To view this data in MestraNova drag the Ibuprofen folder onto MestraNova and then in MestraNova select all the files. Select the option 'Stack Spectra' in the 'Stack' menu to view all spectra. Note that you will have to reapply any 1st order phase correction required as this is not carried over from SpinsolveExpert.

Finally, be aware that you should not attempt to load this data back into SpinsolveExpert as the acqu.par files have been modified to work with MNova.
