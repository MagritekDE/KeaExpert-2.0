ArrayedExperiment

This script allows the selection of a 1D/2D experiment, which will be repeated with a parameter being stepped between experiments based on the contents of a parameter array. The results will be stored in subfolders with sequential numbers and reloaded as a stacked plot (1D) or a series of images (2D). The experiment can be repeated without changing a parameter by choosing the parameter 'Repeat'. In this case, the parameter array just defines the number of repeats.

The arrayed experiment can be carried out to calibrate and optimise experimental parameters, such as pulse lengths, coupling constants, time delays, etc.

Experiment parameters

Experimental parameters are changed according to the chosen protocol.

The top part of the panel, named Arrayed-Experiment, displays the basic parameters of the arrayed script:

         Protocol

o   Contains all protocols from the experimental menu

         Parameter to change

o   Contains all parameters from the experiment interface

o   When option Repeat is chosen, the experiment is repeated without changing any parameters. In that case, the parameter array defines the number of repeats

         Parameter array

o   Defines the range of changes of the chosen parameter in the form of [start:step:end]

o   As an example, in Proton protocol, array [1:2:20] will step between 1 us and 20 us of the pulse length with a step 2 us

         Data to display

o   Defines the display options to present the data. This can be changed after the experiment is done

  spectrum.pt1 - shows the frequency domain data

  fid.pt1 - shows the raw time domain data

         Display as

o   Defines the final presentation of the arrayed experiment data

  Stacked Plot

  Multi-Plots

The rest of the experimental parameters correspond to the particular experiment type.

 

Postprocessing

1.      Click on stacked plot and press on the right panel

2.      Follow to Display mode -> Toogle integrate mode (Shift+I)

3.      Integrating and fitting tools are displayed on the left panel

4.      For more information, check the help menu
