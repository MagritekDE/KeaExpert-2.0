Limitations

 

Note that this macro is designed for fitting single exponential decays. Multi-exponential decays should use the inverse Laplace options in the NNLS menu. Also, because we provide control over each point in the data set this macro is limited to 200 points. If you have more points than this you should consider log-binning the data first using the following additional commands:

 

x = [2:2:3000]/1000

y = 2*exp(-x/1)+ noise(size(x))*0.01

 

(xb,yb) = logbin(x,y,50)

plot(xb,yb)

 

 Note that the x axis must be positive in this case so the first point has been given a small positive value.
