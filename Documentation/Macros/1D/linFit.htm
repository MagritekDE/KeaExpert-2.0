linFit

 

If you want to perform a simple two parameter linear fit to a data set then this macro can perform the necessary analysis. To illustrate its use, run the following commands to generate the linear data set y = 2x + 5 + noise:

 

x = [-10:1:10]

y = 2*x + 5 + 2*noise(size(x))

plot(x,y)

 

Then run this macro. The result is shown below:

 

 

 

The data points are plotted as red squares, with the best fit shown in yellow. The slope and intercept of this line, along with their standard errors are given in the plot title and reported to the CLI.
