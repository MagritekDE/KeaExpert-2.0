applyFunction

 

This macro applies a one line operation the current 1D plot vector. The dialog contains a one line text box into which you can type the operation to apply to the plot vector. At your disposal you have the following variables; the x axis vector, x the y axis data y, and the width w of the vector.

 

 

In the text box, the current data set is represented by the variable y. Essentially the macro applies the following macro

 

   y = expression in text box

 

So the default action is to do nothing.

 

Some examples

 

Suppose you want to multiply the current data set by a ramp going from 0 to 1. In this case you could enter the following line and press the “Apply” button.

 

y .* linvec(0,1,w)

 

To reverse the direction of the column data you would type

 

reflect(y)

  

Press the “Update” button to ensure the current plot parameters data are correct (this should be done automatically most of the time).

 

Press the “Undo” button to reverse the last operation applied.
