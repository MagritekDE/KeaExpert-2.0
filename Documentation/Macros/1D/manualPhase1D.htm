manualPhase1D

 

 

 

This macro allows the user to apply a real-time zero and first order phase shift to the current 1D data set. Typically this is used to get the peaks in a spectrum into absorption mode. If the data set is named y0 then this macro plots the function:

 

 

where f is the frequency, p0 is the zeroth order phase shift in degrees and p1 is the first order phase shift in degrees. f0 is called the pivot point and is the frequency at which there is no first order correction.

 

 

A spectrum with zero and first order phase shifts

 

To use the macro, first try applying a simple zeroth order phase correction by moving the top slider until the spectrum is in absorption mode. If this does not appear to be possible (i.e. some parts are phased correctly while others are not - as in the example below) then press the reset button and try the second method.

 

 

Spectrum after pivot selection and zeroth order phasing (in the program the red line is not visible)

 

First press the pivot button and select a clearly resolved peak to be the pivot point. Next, adjust the p0 slider to phase this peak. With this peak phased now adjust the p1 slider to bring the rest of the spectrum into absorption mode.

Phasing real data sets

 

If you have a real data set with a size equal to a power of two you can also apply this real time phase procedure, which in this case uses a Hilbert transform to determine the imaginary part.
