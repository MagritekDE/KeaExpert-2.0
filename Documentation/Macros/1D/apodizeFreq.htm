apodizeFreq

 

This macro can be used to smooth or sharpen frequency domain 1D data sets. It does this by applying an inverse Fourier transform to the data set, multiplying by a filter (i.e. apodization) function and then applying a forward Fourier transform. Because of the two Fourier transform steps the data set must be a power of 2 in length.

 

 

The effect of applying exponential broadening to a spectrum

 

The macro has several built in filters. The macro equations are listed below:

 

exponential       exp(-abs(x-p1)*p2)

sinebell          cos(0.5*pi*(x-p1)/(N-p1))

sinebellsquared   cos(0.5*pi*(x-p1)/(N-p1))^2

gauss             exp(-(x-p1)^2*p2^2)

gaussdeconv       exp((x-p1)*p3-(x-p1)^2*p2^2)

 

In these equations p1 = 0, p2 is taken from the first slider (labelled Broadening) and p3 from the second slider (labelled Narrowing).

 

 

The 1D apodization interface

 

When using the Gaussian deconvolution filter, non-zero noise at the end of the data set means that you will need to add sufficient broadening to prevent the data set from going to infinity.

 

 

 

Gaussian deconvolution applied to reduce the line-width and so enhance resolution.

 

At all times the apodization macro is working on the data set which was present when you first ran the macro. However if you want to work on a new data set without restarting the macro you will need to press the Update button before applying the broadening.
