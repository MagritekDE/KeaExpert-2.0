baselineCorrection

 

Free induction decays often have a dc offset which needs to be removed before Fourier transformation. This macro allows the user to select a region of the 1D data set which contains a region of baseline (typically at the end of the fid). This region is then averaged and subtracted from the original data set.

 

 

To use the macro press the “Correct” button and then select two horizontal  points in the 1D data set which bound the baseline region you wish to use. The correction will then be applied.

 

Press the undo button to remove this correction.
