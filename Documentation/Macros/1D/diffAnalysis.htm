diffAnalysis

 

The PGSE analysis macro determines the characteristic self-diffusion coefficient D by fitting the plot data to the following function:

 

 

There are 4 ways to use this macro depending on the data available:

 

  1. Simple fit – in this case the plot data consists of echo amplitude E on the y axis and  on the x axis (scaled by 10­-9). No additional parameters are required to analyse this data.
  2. Variable gradient – in this case the abscissa are gradient values (T/m) and the PGSE pulse spacing and pulse duration needs to be entered (in seconds).
  3. Variable large delta (D) – here the abscissa are the times between PGSE gradient pulses (in ms) and the pulse durations (d) and gradient amplitude G need to be entered (in seconds and T/m)
  4. Variable small delta (d) – now the abscissa are the pulse duration times (in ms) and the pulse separation (D) and gradient amplitude G need to be entered (in seconds and T/m).

In the above examples, the data being displayed (red squares) is given by the following macros:

 

Top plot – variable gradient data

 

G = [0.01:0.05:0.5]

delta = 2e-3

Delta = 20e-3

D = 2e-9

E = 10*exp(-gamma^2*delta^2*G^2*D*(Delta-delta/3))

E = E + 0.1*noise(size(G))

plot(G,E)

 

Bottom plot – simple fit

 

q = [0.1:0.1:1]

D = 2e-9

E = 10*exp(-q*D)

E = E + 0.1*noise(size(q))

plot(q,E)
