copyRegion

 

Sometimes it is useful to save the current zoom limits so that the current or another 1D data set can be viewed in the same way. This macro has a simple two button interface to achieve this. Just click on the “Copy region” button to save the size and position of the current region. Display or select the new data and then press “Paste region” to display the same region within the new data set.

 

Note that the window must not be closed between the copy and paste operations since the zoom region variables are saved within the macro window.
