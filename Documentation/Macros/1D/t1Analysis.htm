t1Analysis

 

The T1 analysis macro determines the characteristic spin-relaxation recovery constant T1 and final amplitude E0 of data by fitting it one of the following functions:

 

 

Variable delay experiment

 

Inversion recovery

 

In addition to performing the fit, the macro provides the possibility of removing some of the data points from the fit.

 

 

In the above examples, the data being displayed (red squares) is given by the following macros:

 

Top plot – inversion recovery data

 

x = [0:200:3000]/1000

y = (1-2*exp(-x/1))+ noise(size(x))*0.01

plot(x,y)

 

Bottom plot – variable delay data

 

x = [0:200:3000]/1000

y = (1-1*exp(-x/1))+ noise(size(x))*0.01

plot(x,y)

 

The orange lines correspond to the best fit and the title in each plot reports the fit parameters and their standard deviation.

 

To remove unwanted data points from the analysis just deselect the corresponding checkbox.

 

Limitations

 

Note that this macro is designed for fitting single exponential decays. Multi-exponential decays should use the inverse Laplace options in the NNLS menu. Also, because we provide control over each point in the data set this macro is limited to 200 points. If you have more points than this you should consider log-binning the data first using the following additional commands:

 

x = [2:2:3000]/1000

y = (1-2*exp(-x/1))+ noise(size(x))*0.01

 

(xb,yb) = logbin(x,y,50)

plot(xb,yb)

 

 Note that the x axis must be positive in this case so the first point has been given a small positive value.
