fillInside1D

 

This macro fills a region within the current 1D plot with a constant. The region is defined by the user interactively or using the GUI textboxes. Either the inside or outside of the region may be filled.

The Update button is used to copy the horizontal extent of the selected rectangular region into the GUI textboxes while the Undo button will reverse the last fill operation.
