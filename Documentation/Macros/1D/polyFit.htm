polyFit

 

If you want to fit a polynomial to a function then this macro can be of assistance. Simply run this macro, decide on the likely polynomial order and then press fit.

 

 

 

  Polynomial Fit

  --------------------

 

  Model function: a0 + a1*x + a2*x^2 + a3*x^3

 

  Number of supplied data points = 50

  Estimated standard deviation in data = 0.917

  Specified standard deviation in data =  1

  Chi squared = 38.7

 

  Fit parameters ± SD

  --------------------

  a0: 9.98 ± 0.53

  a1: 0.984 ± 0.094

  a2: -0.0597 ± 0.0045

  a3: 0.001 ± 6e-05

 

The results of the fit will be reported in the CLI. In this example the standard deviation of the data was supplied and so the uncertainties reported in the results will be accurate. If you don’t know the standard deviation then you can ask the program to estimate it for you and then use this in the calculation. A good fit with a realistic value for the standard deviation should have a chi-squared value close to the number of data points.
