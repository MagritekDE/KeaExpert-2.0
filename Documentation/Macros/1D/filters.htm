filters

 

This macro enables the design of apodization filters for use by other macros such as the 2D or nD Fourier transform. The filter design data is stored in a file in the user’s preferences folder and may be accessed from other macros by using the procedure get_filter which is part of this macro.

 

As supplied, the macro comes with 5 filters. Clicking on the filter name in the filter menu displays the filter equation and parameters in the lower part of the window. Two types of filter should be supported – one which peaks at the beginning of the data set (call the FID filter) and another which peaks in the centre (called the Echo filter). You can view your filters by pressing the appropriate Test button.

 

 

If you make changes to the filter design or add a new one, press the “Update” button to copy it to the filter menu. Finally press “Save” to copy the changes you have made to the filter preferences file. Pressing the “Restore” button before the “Save” button will reload the filter data from the preferences file in case you have made a mistake.

 

To access these filter from another macro called the command:

 

vec = filters:get_filter(name,type,N)

 

where

 

     name ... name of filter

     type ... "FTFid" or "FTEcho"

     N ...... number of data points in filter

 

To get the current menu (for use in another macro for example) call the command

 

lst = filters:get_menu()

 

For example:

 

plot(filters:get_filter("gaussian", "FTEcho", 128))

 

 

In this case the Gaussian parameters were p1 = N/2 and p2 = N/10
