calibrate1D

 

This macro allows the x axis of the current 1D data set to be calibrated using a linear scale.

 

There are three options – in the first “By range”, you enter the minimum and maximum values for the x axis and then press the “Apply” button.

 

 

 

Calibration performed by entering minimum and maximum x values

 

In the second option “By selection”, you enter a width and the value of a particular point on the x-axis. When you press the “Apply” button this point is selected interactively using the 1D cursor and mouse.

 

 

 

Calibration performed by entering a width and selecting a reference point (in this case the left most peak)

 

Finally the third option “Indices” removes the user-defined scale and simply calibrates the x axis using the data index.

 

Note that reversed scales cannot be applied at present.
