shiftRotate1D

 

This macro provides an interface to the shift and rotate commands for 1D data. To use the macro, enter the desired shift or rotation in points (positive or negative) and then press the shift or rotate button.

 

 

 

 

Original Data

After a shift

 

Shifting data results in data loss and the replacement of the lost data with zeros, while rotation keeps all data, but moves it to a new position. Pressing the enter key when either the shift or rotate button has focus will cause the action to be repeatedly applied.

 

Original Data

After a rotation

 

Interactive rotations

 

Pressing the Start button allows the selection of a point in the data set to be used as the first point, while the Centre button allows the selection of a new centre point. In both cases an appropriate rotation is applied.

 

The Drag button allows the interactive rotation of the data. After clicking on the data set the chosen point will then move with the cursor as it is dragged.

 

The Undo button will reverse the last shift or rotation operation.
