generalLinearFit

 

This macro allows you to find the best linear fit to a data set. For example in the following case the red dots represent the function obtained with the macro:

 

x = linvec(0,1,100)

y = cos(x*10)+2*cos(5*x)+0.5*noise(100)+5

plot(x,y)

 

Then the macro was run with the following parameters:

 

 

The yellow line represents the best fit – the parameters being listing in the CLI:

 

        Linear Fit

   --------------------

 

   Model function: a0 + a1*cos(10*x) + a2*cos(5*x)

 

   Number of points: 100

   Chi squared: 29.4697

   Estimated standard deviation in data: 0.551191

 

   a0 = (4.96 ± 0.06)

   a1 = (0.95 ± 0.08)

   a2 = (1.97 ± 0.08)

 

Note that a least two terms must be present in the fit for it to work and that the only constant parameter allowed is 1.
