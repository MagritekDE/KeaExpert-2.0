apodizeTime

 

This macro applies an apodization function to time domain data.

 

The macro has several built-in filters. The macro equations are listed below:

 

exponential       exp(-abs(x-p1)*p2)

sinebell          cos(0.5*pi*(x-p1)/(N-p1))

sinebellsquared   cos(0.5*pi*(x-p1)/(N-p1))^2

gauss             exp(-(x-p1)^2*p2^2)

gaussdeconv       exp((x-p1)*p3-(x-p1)^2*p2^2)

 

In these equations p1 = 0, p2 is taken from the first slider (labelled Broadening) and p3 from the second slider (labelled Narrowing).

 

 

The effect of applying this operation to a decay sinusoidal function can be seen by running the command:

 

(x,y) = cosine(1024,1,300,10)

plot(x,y)

 

This generates a 1024 point, 300 Hz  data set lasting 1 second. The exponential decay time is 10 seconds.

 

 

 

The Gaussian deconvolution command can be used to remove exponential broadening from peaks and replace it with the narrower Gaussian broadening:

 

(x,y) = cosine(16384,5,300,0.2)

plot(x,y)
