submatrix1D

Use this macro to replace the current 1D data set with a sub-matrix determined by the currently visible region, by a rectangle drawn over the plot or by manually defined limits.

 

  • First select the region to be extracted - do this by either zooming into the region of interest, by outlining it with a rectangle or by entering the start coordinate and the width of the selection.  Note that the height of the rectangle is ignored.

 

  • Next run the macro. The chosen limits will be written to the macro textboxes automatically.

 

  • Press the Extract button to complete the extraction.

You can choose the region limits using x axis units or with indices by selecting the appropriate option from the Units radio-button.

 

The Update button will (if necessary) update the submatrix range from the data set (although this should be automatic), while the Undo button will reverse the submatrix operation.
