adderrorbars

 

Error bars can be added to a 1D plot using macro commands: for example

 

 

 

x = [0:9]

y = 5*noise(10) + x*10

eb = matrix(10,2)

eb[~,0] = 5

eb[~,1] = 5

plot(x,y,"tracetype","none",

         "symbolshape","circle",

         "ebar_array",eb)

hold("on")

plot(x,x*10,"tracetype","lines",

         "symbolshape","none",

         "tracecolor",[255,0,0])

title("best fit!")

hold("off")

 

Alternatively, you can add them using this macro. Here you add error bars to an existing data set by passing two arrays. One which determines the extent of the error bars above the data point and another below the data point.
