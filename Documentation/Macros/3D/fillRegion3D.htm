fillRegion3D

 

This macro fills a box shaped region inside the specified 3D data set with a constant. The region is selected by drawing a rectangle in one plane of the 3D data set, which will automatically make the third dimension of the box span the complete 3D data set. Alternatively the dimensions of the box may be entered manually.

 

Once selected, the region may be filled with any real (or in the case of a complex data set, a complex) number. Alternatively by selecting the “Fill outside” option this value can be applied to the region outside the box.
