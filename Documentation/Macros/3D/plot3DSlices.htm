plot3DSlices

 

This macro displays planes or slices from a 3D real or complex data set. To use it, select a 3D matrix from the list at the top of the dialog. Then select the plane number and plane direction you wish to view, from the “scan planes” group-box. The selected plane will be displayed immediately using the current 2D display options.

 

 

The axes label range region of this dialog allows the 2D axes ranges to be modified so that they no longer default to indices. 1D vectors with elements [min,max] can be used to set these values rapidly.

 

The buttons on the right of the window allow selection of:

 

  1. Tiers – these are 1D data sets taken at right angles to the current 2D plot. As you move the cursor around the current plane, a tier at position (x,y) will be displayed.
  2. Horizontal slice. If you are viewing in the xy plane and select a horizontal slice then the new 2D plot will be in the xz plane. Likewise for other planes.
  3. Vertical slice. If you are viewing in the xy plane and select a vertical slice then the new 3D plot will be in the yz plane. Likewise for other planes.
