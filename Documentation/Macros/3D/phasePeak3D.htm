phasePeak3D

 

This macro takes a 3D complex matrix and applies an automatic zeroth-order phase correction to each row in each plane. The only user interaction is to select the extent of the peak to phase.

 

Typical operation is as follow.

 

  1. Load a complex 3D data set using importData.mac.
  2. Fourier transform along the x axis using ndFourierTransform.mac
  3. Run this macro.
  4. Select a reference plane with the slider (unnecessary if the plane is the first)
  5. Press the “Auto phase” button and choose points either side of the peak to be phased (these points should be on the baseline and not part of the peak).
  6. The macro will phase all rows so that the selected peak is in absorption mode.

A number of different options are also available:

 

  1. Columns can be phased instead of rows.
  2. A fixed phase can be applied to all rows or columns.
  3. The autophase method can be modified to give the best phasing:

a.       max adjusts the phase so as to maximise the area under the peak.

b.      min adjusts the phase so as to minimise the part of the peak below a baseline draw between the two selected points. (This method often works better with a sloping baseline).
