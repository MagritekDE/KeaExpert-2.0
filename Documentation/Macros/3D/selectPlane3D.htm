selectPlane3D

 

Using the macro selectPlane3D.mac (found in the 3D menu), you can display slices through the 3D plot in the 2D plot. As you move the plane position the 2D plot will update in real time.

 

 

The plane direction, position, colour and transparency can be set from the macro interface.
