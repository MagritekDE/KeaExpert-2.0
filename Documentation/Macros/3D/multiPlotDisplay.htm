multiPlotDisplay

 

This option will display all the slices in a 3D matrix simultaneously – each as a separate 2D plot.

 

When first loaded, the multi-plot macro will default to the xy plane and make a guess at how you want the multi-plots to be organised. If you prefer a different number of plots in each dimension you can enter them manually, however note that the product must equal the total number of slices in that direction. So for example in the adjacent example 5*6 = 30 but you could also use 6*5, 3*10, or 10*3.

By default, each subplot will be auto-ranged, meaning that the planes with low-level data will look as intense as planes with high-level data. However by selecting the subplot with the largest data values and then deselecting “Autorange” in the 2D display parameters macro, selecting “Apply to all plots” and then pressing the “Draw” button, the plot above right can be obtained.
