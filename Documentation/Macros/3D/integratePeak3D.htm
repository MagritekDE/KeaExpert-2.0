integratePeak3D

 

This macro takes a 3D real or complex matrix and integrates the data between two horizontally chosen points for each row in the data set. Each row gives a single integral value, while integrating one plane produces a single line in the output file.

 

Typical operation is as follow.

 

  1. Load a complex 3D data set using importData.mac.
  2. Fourier transform along the x axis using ndFourierTransform.mac
  3. Phase the data set using the macro phasePeak3d.mac.
  4. Run this macro and select a reference plane with the slider (unnecessary if the plane is the first).
  5. Select a name for the 2D output integral file.
  6. Press the “Integrate” button and choose points either side of the peak to be integrated.
  7. The macro will then integrate the data and display the 2D output matrix.
