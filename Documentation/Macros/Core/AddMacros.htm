AddMacros

 

 

 

This option allows the selection of which macros will appear in the main menu.

 

By default certain macro menus are duplicated when first installed. This is done by copying the menu folder to one inside Macros\Libraries. In many cases you may not wish to see all the possible menu options and this utility allows you to edit the menu contents.

 

 

  • The Arrow button allows you to copy macros selected in the library to the output folder (Kea-MRI in this case). 

 

  • The All button will copy all library macros to the output folder.

 

  • The Cross button will delete the selected macro in the output folder.

 

  • The Macro Help button displays help (if it exists) for the selected Library macro.

 

Note that copying a macro from the library to the output folder will overwrite any changes you have made to that macro.
