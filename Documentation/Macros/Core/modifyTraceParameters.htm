modifyTraceParameters

 

 

To modify a plot-trace double-click on it to obtain the following dialog:

 

 

Once the dialog is displayed, then single clicking on different traces will update the contents. The particular trace and it location will be shown in the dialog title bar.

 

The dialog is divided into several regions; the top two sections modify the trace, the third section an associated symbol and the bottom right an optional error-bar. The trace can be attached to either the left or right axes – this is determined by the controls at centre right.

 

The trace name is a text string which will be used for identifying the trace when a legend is displayed (settable in the 1D plot View menu). If the “Show in legend” checkbox is unchecked then this trace will not appear in the legend.

 

Click on the row of Style buttons to display the data points with a line connecting them, as a staircase function, or simply as dots. The right-most button does not display the trace at all. (Useful when you just want to display the symbol).

 

The width of the trace may be set to any number from 0 (the narrowest line) up to 10.

 

The line-dash options are as follows

 

solid line

dash-dash- (-----)

dot-dot-dot (.....)

dash-dot-dash-dot- (-.-.-.-.)

dash-dot-dot-dash-dot-dot (-..-..-..)

 

The trace colour for real and imaginary data may be chosen by left clicking on the colour boxes to the right of the dash menus.

 

The symbol section works in much the same way. Choose the symbol shape you prefer (or none - the last button in the list), its colour and size.

 

In the error bar section you can choose a colour for error bars and whether they should be displayed or not. A fixed height error bar can be chosen or by choosing an array name, a variable height error bar can be displayed. In this case the array should be two dimensional with as many columns as there are data points. The first row contains the height above the data point the second row the height below. Here is an example for a 50 point data set

 

eb = matrix(50,2)

eb[~,0] = abs(noise(50)*10)+10

eb[~,1] = abs(noise(50)*10)+10

 

 

Note that the settings you make in this dialog only apply to the current trace. If you make a new plot or add a new trace to the existing plot then the default trace and symbol properties will be used. You can make the current trace parameters the default by choosing the option “Save plot preferences” from the 1D plot Pref menu. This will then apply to all further plotting – even in new instances of Prospa.
