2D Display Settings

 

This macro allows the selection of different 2D display modes and also allows the adjustment of the 2D display range.

 

 

The default setting is to display the image as an intensity plot with the full data range being used.

 

 

Selecting the Contor plot and Interpolated plot modes are alternatives, the latter just filling in the spaces between the contours

 

Contour mode

Interpolated contour mode

 

The number of contour levels can be set using the Control Levels text box.

 

On the display range side of the control the default is to automatically determine the range of values in the plot.

 

 

Unchecking “Auto range” will enable these controls. Adjusting the base and range sliders will update the plot display in real time as well as updating the maximum and minimum levels.

 

The Full range button will update the max. level and min. level with the full range of data values in the visible data set.
