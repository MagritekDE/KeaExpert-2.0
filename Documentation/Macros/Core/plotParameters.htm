plotParameters

 

 

This macro allows the non-trace attributes in a 1D plot-region or regions to be set. For trace parameters see the ModifyTraceParameters macro. Only one plot parameter macro can be displayed at one time and if you select a different plot region the tab will be updated to reflect the new parameters. Each tab in the macro dialog box controls a different aspect of the plot window. Details are shown below for each tab.

 

The Ticks Tab

 

 

This tab defines the spacing between numbered labels and the number of ticks between these labels. The spacing is in the current x or y units.

 

If the checkbox “Choose tick parameters automatically” then the size of the displayed region will determine these parameters. If it is unchecked then you can override the automatic calculation.

 

The tick length parameters determines the length of the short and long ticks in pixels. They may be positive (point away from the plot) or negative (pointing into the plot).

 

The tick color determine the colour of the ticks.

 

Tick parameters are updated automatically if you press return after entering each number or you can press the Apply button to update using all entered parameters.

 

The Grid Tab

 

 

A grid may be displayed over the plot. The x and y grid lines go from label tick to label tick while the fine x and y graid lines go between the unlabeled ticks. The color for each of these grid lines can be also be set.

 

 

The Range Tab

 

 

By default the complete data set is always drawn when a plot is performed. However you can override this feature by entering x and y ranges into this tab. Pressing return after entering each number will update the plot, or you can press the Apply button to update all the entries.

 

If the “Auto range when plotting” checkbox is unchecked then this range will be used for all subsequent plots in which the hold setting is on. New plots will reset this checkbox.

 

The Title Tab

 

 

This allows the plot title to be set along with its font, size, color etc.

 

 

The Labels Tab

 

 

This allows the plot labels to be set along with their font, size, color etc. Note that plots can have left and right labels but unless a trace is attached to the right axis the right label will not be displayed. (See ModifyTraceParameters macro for how to do this).

 

 

The Axes Tab

 

 

This allows the axes font to be set along with its font, size, color etc. This used for drawing the numbers on both the X and Y axes.

 

 

The Colors Tab

 

 

This tab only controls the colors of the plot border and background. Individual font and tick colors can be set in their appropriate tabs. However there is an option called “All labels and ticks” which will give all these the same color.

 

The Mapping Tab

 

 

If the x and/or y axes do not include zero or negative numbers a logarithmic axis can be displayed by selecting the appropriate option from this tab page. If the axis are not valid for this operation an error message will be display.

 

The “Apply to all subplots” button.

 

This button will apply the current tab-page settings to all subplots in the current plot control.
