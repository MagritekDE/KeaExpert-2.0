Display 3D Isosurface

 

 

This macro displays a surface over a 3D object at a particular surface level – i.e. the surface represents constant values in the data set.

 

When the macro is launched or a new (real) data set is selected from the Name text menu, the minimum and maximum values in the data set will be displayed and an iso-surface level will be chosen 80% of the way between the min and max values.

 

 

 

 

 

The default parameters will cause the plot to be cleared before being drawn and an axis is included with the plot. The axis can be boxed as well by selecting the Draw box option (as was done in this example).

 

The axes range will default to the number of points in each dimension. This can be modified by entering your own range

 

 

 

Alternatively global two column vectors can be defined with the minimum and maximum limits as entries.

 

Multiple surfaces can be displayed by unchecking the Clear plot flag and plotting each level in different colors and with a transparency less than 1.

 

 

 

Finally, if the data surface you are interested in is very small relative to the complete matrix you can choose the fitted axes option which will draw axes only 25% bigger than the display surface dimensions (fitted axes with scales). Note that in this case you will need to add your own range since the dialog box is not aware of the size of the 3D surface until after it is plotted.
