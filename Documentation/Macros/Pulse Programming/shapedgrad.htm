shapedgrad

 

Generate a shaped gradient pulse or pulses

 

Syntax

 

1 gradient

 

shapedgrad(address, table, size, duration)

 

2 gradients

 

shapedgrad(adrs1, tbl1, adrs2, tbl2, size, duration)

 

3 gradients

 

shapedgrad(adrs1, tbl1, adrs2, tbl2, adrs3, tbl3, size, duration)

 

 

address ........ which gradient to send value to. Must store a number with value 0, 1, 2, 3 (0 is the connector at the bottom of the gradient controller). Must have form “nx” where x is an integer greater than 0.

 

table1 ......... a table of gradient amplitude values. Has the form “tx” where x is an integer greater than 0.

 

size2  .........   the number of entries or steps in table. Must have form “nx” where x is an integer greater than 0.

 

duration3 ......  the duration of each step in the table. Has the form “dx” where x is an integer greater than 0 . Duration units should be microseconds.

 

 

1.      The table values should be between -32767 and 32768. (Waveform resolution is 16 bits). The table should be passed in the form of a row vector.

 

2.      The sum of all table sizes must be smaller than 6000 points in total.

 

3.    The stepTime value may range from xx ms to 327,670 ms. xx is 2 ms for the single gradient control, 4 ms for dual gradient control and 6 ms for 3 gradient control.

 

 

Example

 

      "t1 = 32767*exp(-(linvec(-1,1,n))^2*5)"

  

 

t1 is a table which describes a Gaussian pulse. This should be entered as a parameter into the relationships table in the pulse program macro.
