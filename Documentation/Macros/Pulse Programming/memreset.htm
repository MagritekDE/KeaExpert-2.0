memreset

 

This command resets the memory address for saving data on the spectrometer processor.

 

Syntax

 

memreset([address])

 

address......  an optional parameter controlling the memory address.

Parameters have the following form:

 A number variable with name "nx", where x is any valid string. Allowable values for the variable are from 0x10000 to 0x30000)

 

 

 

0x10000 is the normal start memory location for data stored on the Kea and is the default value for this command. If during a pulse program the memory location needs to be reset or changed this command can perform this action. Note that for most pulse sequences this command is unnecessary since the memory address is initialised to 0x10000 automatically.
