endloop

 

Signifies the first statement after the repeated code within the loop command.

 

Syntax

 

endloop(name)

 

name  ........... the loop label. Must have form “lx” where x is an integer greater than 0 and must match the label in the corresponding loop command.

 

Note

 

  1. This command takes 10 ns to complete and is executed once when leaving the loop.
