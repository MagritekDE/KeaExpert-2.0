gradoff

 

This command switches one of the gradient outputs to zero.

 

Syntax

 

gradoff(address)

 

address .........  which gradient to zero. Must store a number with value 0, 1, 2, 3 (0 is the connector at the bottom of the gradient controller) or (from the top) "x/y/z/o". The address parameter has the form “nx” where x is an integer greater than 0 or it can be one of the previously mentioned constants.

 

Notes

 

  1. This command takes 2.2 us to complete.
  2. All gradients are automatically zeroed at the end of each pulse sequence unless the "Leave shims on at the end of the sequence" option is checked in the Kea Preference shims tab.
