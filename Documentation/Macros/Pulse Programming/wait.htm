wait

 

This command pauses the DSP for the specified time. Useful when the delay command discussed above has insufficient dynamic range.

 

Syntax

 

wait(duration)

 

duration ......  the parameter controlling the length or duration of the pulse. It has the form “wx” where x is an integer greater than 0.

 

Notes:

 

  1. The delay duration may vary from 0.2 ms to 167 s.
  2. Wait units should be in microseconds.
  3. This command requires that 3, 24 bit words are sent to the DSP parameter list compared to 1 for the simpler delay command.
