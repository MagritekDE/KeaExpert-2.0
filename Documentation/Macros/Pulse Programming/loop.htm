loop

 

A group of commands can be repeated by putting them between a loop-endloop pair.

 

Syntax

 

loop(name, n)

 

name ............ the loop label. Must have form “lx” where x is an integer greater than 0.

n ............... the number of times to execute the code inside the loop. Must have form “nx” where x is an integer greater than 0.

 

Notes

 

  1. n can be any number from 0 to 224-1.
  2. A loop command must be bracketed with an endloop command.
  3. This command takes 60 ns to complete and is only executed on the first loop.
