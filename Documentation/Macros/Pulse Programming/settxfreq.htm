settxfreq

 

This command sets the transmit frequency i.e. the frequency to use for subsequent RF pulses. It does not affect the receiver reference frequency.

 

Syntax

 

settxfreq(frequency) # channel = 1

 

or

 

settxfreq(channel,frequency)

 

channel .........  which RF output channel to update. Either "i", "e", or "1" for channel 1 and "2" for channel 2.

frequency .......  the name of the new frequency parameter. Must have form “fx” where x is an integer greater than 0.

 

This command updates the default RF transmit frequency and also updates the output frequency directly if placed between txon and txoff commands.

 

Notes

 

  1. The command takes 2 ms to complete with the direct frequency updating occurring at the end of this time.

 

  1. The frequency value should be given in MHz.

 

  1. Do not place this command directly after an RF pulse. There should always been a delay of at least 1 ms between an RF pulse command and settxfreq.

 

 

 See also

 

inctxfreq

 

 

 Examples

 

To update the RF pulse frequency on channel 2

 

    settxfreq(2,f1)

    pulse(2,a1,0,d1)

 

To change the frequency during a pulse on channel 1

   ...

     "f1 = 1",

     "f2 = 2",

     "f3 = 4",

     "f4 = 8",

  ...

    txon(1,a1,0,f1) # Switch on at 1 MHz

    settxfreq(1,f2) # Switch to 2 MHz

    settxfreq(1,f3) # Switch to 4 MHz

    settxfreq(1,f4) # Switch to 8 MHz

    delay(2)

    txoff(1) # Switch off RF
