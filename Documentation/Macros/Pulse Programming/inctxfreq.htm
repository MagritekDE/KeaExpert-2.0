inctxfreq

 

This command increments the transmit frequency i.e. the frequency to use for subsequent RF pulses. It does not affect the receiver reference frequency.

 

Syntax

 

inctxfreq(frequency)

 

frequency .......  the name of the increment frequency parameter. Must have form “fx” where x is an integer greater than 0.

 

 

Notes

 

  1. This command takes 1.25 us to complete.
  2. The frequency increment should be in MHz.
