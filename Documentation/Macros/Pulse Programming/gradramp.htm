gradramp

 

Generate a ramped gradient

 

Syntax

 

gradramp(address, start, end, nSteps, stepDuration)

 

 

This calculates gradient values on the fly according to the (integer) equation

 

g = n*(end-start)/(nSteps-1) + start.

 

where n takes the values 0 to nSteps-1.

 

Arguments

 

address ........ which gradient to send value to. Can be a number with value 0, 1, 2, 3 (0 is the connector at the bottom of the gradient controller) or a string "x", "y", "z", "o" (for other). Alternatively can be a variable of the form “nx” where x is an integer greater than 0.

 

start1 ......... the initial gradient amplitude. Can be any integer should be between -32767 and 32768 (gradient resolution is 16 bits). Alternatively can be a numeric variable of the form “nx”, or a table variable with the form "tx" where x is an integer greater than 0.

 

end1 ........... the final gradient amplitude. Can be any integer should be between -32767 and 32768 (gradient resolution is 16 bits). Alternatively can be a numeric variable of the form “nx”, or a table variable with the form "tx" where x is an integer greater than 0

 

nSteps2  .......   the number steps in the ramp between start and end. Should be a positive integer greater than zero or have the variable form “nx” where x is an integer greater than 0.

 

stepDuration3 ..  the duration of each step in the table. Duration units should be microseconds and can be a floating point number in these units or can take the variable form “dx” where x is an integer greater than 0.

 

 

1.      The gradient values should be between -32767 and 32768. (Waveform resolution is 16 bits). These correspond to the maximum output voltages on the gradient controller (±10V)

 

2.      Because of the 24 bit integer architecture of the DSP there is a limit to the product: (nSteps-1)*(end-start). This number should be less than or equal to 223. This means that if you are going to use the full range of gradient values (16 bits) you must use 129 steps or less.

 

3.      The stepDuration value may range from 2 us to 327,670 us. 

 

Example

 

   relationships = [... 

                    "t1 = [1000:2000:20000]",

                    ...]

   ...

   setindex(t1,0)

   loop(l1,11)

      gradramp(x,0,t1,10,5)

      delay(100)

      gradramp(x,t1,0,10,5)

      delay(20)

      incindex(t1)

endloop(l1)

...
