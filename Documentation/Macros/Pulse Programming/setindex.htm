setindex

 

This sets the specified table index allowing certain commands to access different table entries.

 

Syntax

 

setindex(table,index)

 

table ..... the table to increment index for. It has the form “tx” where x is an integer greater than 0.

 

index ..... a zero based offset which specifies which table entry to access next. Note that there is no checking to see if the index is out of array bounds.

 

 

See also: incindex, decindex.
