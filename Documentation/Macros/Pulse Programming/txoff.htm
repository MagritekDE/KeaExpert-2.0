txoff

 

This command switches off the transmitter output

 

Syntax

 

txoff(destination)

 

destination ……...   "1" for channel 1 – used for Proton and Fluorine.

"2" for channel 2 – used for X channel nuclei.

"1nb" channel 1 no blanking – in this case there is no signal to the probe, but a pulse is still produced on the transceiver output (for debugging).

"2nb" channel 2 no blanking – as for 1nb but on channel 2.

"w1" channel 1 wobble mode (switches to wobble mode on channel 1 duplexer).

"w2" channel 2 wobble mode (switches to wobble mode on channel 2 duplexer).

 

Notes:

 

  1. This command takes 430 ns to complete for a single channel or 740 ns for two channels.
  2. If no argument is passed or the argument is "w" then both channels are switched off.
