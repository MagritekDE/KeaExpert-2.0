endpp

 

This should always be the last statement in the pulse program. It joins together all sections of the pulse program and writes this to the asm file which will subsequently be compiled into DSP machine code. This statement also returns the list of pulse program parameters in the order they appear in the asm file. This list is the mechanism by which the Prospa macro interacts with the pulse program. The list order should not be changed.

 

Syntax

 

lst = endpp()

 

lst ... list of pulse program parameters.
