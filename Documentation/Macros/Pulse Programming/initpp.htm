initpp

 

 Syntax

 

initpp(dir)

 

dir .......  The name of directory where the pulse program will be saved. The name of this directory will also be used as the base name for the interface macro (*.mac), which will be stored outside the directory and the pulse programs (*.p), and parameter files (*.par) stored inside the directory.

 

In addition to defining the various files used when generating the pulse sequence this command also initialises various memory buffers used by the assembler. This should command should always be the first statement in a pulse sequence.
