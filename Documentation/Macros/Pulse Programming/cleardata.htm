cleardata

 

In those cases where data is added to pre-existing data (e.g. in the cpmgAdd macro) this command can be used to initialise the DSP data memory.

 

Syntax

 

cleardata(points)

 

points ........  number of complex points to set to zero.

 

Notes:

 

  1. This command takes (150 + points*40) ns to complete.
