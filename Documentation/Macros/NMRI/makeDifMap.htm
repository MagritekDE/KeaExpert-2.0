makeDifMap

 

This macro calculates a diffusion map by analysing a set of q-space images obtained using a PGSE + imaging sequence in which the PGSE gradient, pulse length or pulse spacing has been incremented between each image.

 

 

Before this macro can be run, the real 2D or 3D input data set must have been either loaded from a file or generated using the nDFourierTransform macro.

 

Once loaded:

 

 

  1. First select the q-space matrix from the pop-up menu to the right of the “Input matrix” field.

 

  1. Then select a name for the 1D or 2D diffusion matrix and the corresponding intensity matrix.

 

  1. Next select the type of experiment performed. The options are:

 

Variable gradient

Variable large delta (inter-gradient spacing)

Variable small delta (gradient pulse length)

 

  1. Select which axis has the q-space data (x, y or z).

 

  1. Enter a calculation threshold from the first vector/plane of the q-space data set. Diffusion coefficients will only be calculated for points larger than this threshold.

 

  1. Enter a maximum percentage error to plot – typically start with 90% and work your way down. (This means that a fit with an uncertainty of 90% will be accepted as ok).

 

  1. Enter the gradient, large and small delta value, in SI units, into the remaining fields. Two parameters will be fixed while the third is represented by a 1D array with the format [value1, value2, value3 ...]. The two small buttons to the right of the variable parameter text box allow you to enter this array by assuming a linear ramp [0:stepsize:maxValue] or from an ASCII data file (a list of numbers – 1 per line).

 

  1. Press “Calculate” to generate the diffusion plot/image.

 

  1. For 3D q space data you can press the 1D view button to display the echo-amplitude vs. gradient curve for a particular point in the image. The local self-diffusion coefficient is also calculated.

 

  1. Don't forget to save the resultant data sets if you wish to keep them. (Save data as ... under the main File menu) Or alternatively you can save the parameters you have entered using the saveParameter macro under the NMRI menu.
