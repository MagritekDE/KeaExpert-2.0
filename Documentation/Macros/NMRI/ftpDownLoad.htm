ftpDownLoad

 

Data can be directly downloaded from a remote ftp server such as a spectrometer using this macro. The first time you use the macro you will need to enter the server address and passwords into the macro preferences file.

 

 

Initial FTP download dialog

 

 

Press the Server Info button to see this dialog

 

You can do this by clicking on the Server Info button in the Bruker FTP Download window and then entering information for up to 3 servers in the Server connection details window. Press the “OK” button and then press the “Connect” button in the download window. The following dialog will appear (with your connection details displayed):

 

 

The default servers you entered above (if any) may be accessed via the drop down-menu or you may select another server by manually entering the required data.

 

Once the ftp login information has been entered, press the “Connect” button. This will establish a connection to the server and display in the main download window the title of the data stored on the spectrometer under your login name. (i.e. /u/data/username/nmr/dataset/number).

 

 

Select the data set you wish to copy from the ftp server and then specify the local directory where the data should be stored (…). Finally press the “Down-load” button to access the data. Once the data has been copied you should terminate the FTP connection by pressing the “Disconnect” button or alternatively just close the window.

 

Pressing the “Import” button will down-load the data set, close the window and then open the importData window with the downloaded data already selected.

 

Note that for non-Bruker spectrometers it will be necessary to make several modifications to this macro – please contact magritek for more information.
