inverseFourierTransform

 

This command provides the reverse functionality of the ndFourierTransform macro. When displayed, the parameters for the current matrix will appear in the dialog window – as shown below (in this example the Kumquat data set has first been loaded and the Fourier transformed.)

 

 

Press the “Transform” button to perform the inverse transform. Note that filters and zero filling are not supplied here since it is supposed that this macro will be used to simply move to the time/k-space domain and that the ndFourierTransform macro will be used to perform filtering and zero-filling.
