makeRelaxMap

 

This macro generates a relaxation and image map from a series of relaxation weighted images.

 

 

Before this macro can be run, the real 2D or 3D input data set must have been either loaded from a file or generated using the nDFourierTransform macro.

 

Once loaded:

 

  1. First select the experiment type. Possibilities are:

 

T2 with variable echo time

T1 with variable time delay between experiments

T1 using inversion recovery

 

  1. Next select the 2D or 3D input matrix from the pop-up menu to the “Input matrix” field.

 

  1. Then select a name for the 1D or 2D relaxation and proton density maps (matrices).

 

  1. Enter, as an array, the list of echo or delay times used in the experiment (in ms) e.g. [10,20,30,40] or [10:10:40]

 

  1. Select the relaxation dimension in the input data set (x, y or z)

 

  1. Enter an maximum uncertainty in the fit as a percentage (e.g. 20% means that all points displayed in the image have a relaxation curve fit error with a standard deviation less than 20% of the corresponding relaxation amplitude)

 

  1. Enter a threshold from the first (T2) or last (T1) plane of the 3D relaxation data set. Relaxation times will only be calculated for points higher than this threshold.

 

  1. Press Calculate to generate the relaxation map.

 

  1. For 3D relaxation data press the “1D view” button to display the echo-amplitude vs. echo-time curve for a particular point in the image. The local relaxation time is also calculated.

 

Note that all data values in the relaxation image represent times in milliseconds.

 

Don't forget to save the 2D data sets if you wish to keep them. (“Save data as” ... under the main Files menu).
