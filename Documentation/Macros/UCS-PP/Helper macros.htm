Kea Helper Macros

 

In addition to the Kea GUI macro there are a series of helper macros which are stored in

 

<appdir>\Macros\Kea-Core

 

These are

 

keaCtrl ... procedures to control the display, initialisation and termination of a pulse program. Normally the end user will not need to use these files unless designing their own interface.

 

keaRun ... miscellaneous procedures commonly called from the execpp procedure.

 

keaFiles ... commands to load and save Kea related files.

 

keaPlot ...  commands to plot the results of an experiment in the 1D or 2D plot windows.

 

A detailed list of the procedures available within these files which are relevant to the end user can be found in appendix A1.

 

Note that previous versions of Kea macros used a single file called keaNMR for all these procedures. This is still supplied for backwards compatibility but all new macros should use the new versions  listed above.
