The Kea pulse-program user interface

 

All Kea pulse program macros use a standard window interface which typically consists of four groups of controls each accessible via a "tab" at the top. The left-most tab contains the user-definable controls specific to each pulse sequence. The remaining tabs allow the setting of acquisition parameters, processing and display options, and file information.

 

 

An example of the Kea interface showing the controls for the Pulse Sequence tab. Other controls can be accessed by pressing the Acquisition, Processing+display, or File settings tabs.

 

 

As you click on each tab different sets of controls become visible. Clicking on the Expand button allows all controls to be visible simultaneously:

 

 

The T1-Inversion recovery macro after the Expand button has been selected. Press the Collapse button to return to the tabbed interface.

 

 

To start an experiment you press the Run button at the bottom. The Abort button stops the experiment after the current scan without saving anything, while Finish completes the current phase cycle and then saves the accumulated data.

 

The Load Par. button loads a set of parameters from the acqu.par file generated by a previous experiment of this type.

 

The Kea Pref. button displays a window with a list of preferences used by all Kea experiments.

 

The Help button displays specific pulse program details.
