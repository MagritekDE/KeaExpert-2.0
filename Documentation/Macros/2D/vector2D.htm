vector2D

This macro controls the display of vector plots. A vector plot displays a 2D array of arrows which represent the magnitude and direction of vectors stored in two real matrices and. i.e.

 

 

The vector2d dialog allows the matrices to be chosen which will be used to form the vector components and  (Ex and Ey in the following dialog).

 

Since some data sets are too dense to enable clear display all vectors, this dialog also allows you to choose the sampling density along each axis. (e.g. 1 implies that all points will be used, 2 every second point and so on).

 

The magnitude of each vector is represented by the length of the associated arrow and its colour. The colour is taken from the current colour-map - the top colour representing the largest magnitude vectors and the bottom colour the smallest. The arrow lengths range from 0 for the smallest vectors to the value chosen from the dialog slider for the largest (34 in this case).
