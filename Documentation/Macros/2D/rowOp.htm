rowOp

 

This macro applies a one-line operation to each row in the current 2D matrix. The dialog contains a text box into which you can type the operation to apply to each row. At your disposal you have the following variables; x a row vector having the value [0: 1: w-1],  y the row number, w  the width of the matrix and h its height.

 

 

In the text box the current row is represented by the variable m1[~,y], m1 being the name used by the dialog for the current matrix. Essentially the macro applies the following macro

 

for (y = 0 to h-1)

   m1[~,y] = expression in text box

next (y)

 

So the default action is to do nothing.

 

Some examples

 

Suppose you want to scale the row data by its column position, then you should type the expression:

 

m1[~,y].*[0:1:w-1]    or more simply  m1[~,y].*x

 

To reverse the direction of the row data you would type

 

reflect(m1[~,y])

 

To multiply the columns by a Gaussian of width h/4

 

m1[~,y] .* exp(-(x-w/2)^2/(w/4)^2)

 

Press the “Update” button to ensure the current plot parameters data are correct (this should be done automatically most of the time).

 

Press the “Undo” button to reverse the last column operation applied.
