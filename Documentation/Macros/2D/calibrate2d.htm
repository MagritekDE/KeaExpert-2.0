calibrate2d

 

By default, a 2D plot displays matrix indices along the x and y axis. With this macro you can modify this range by entering minimum and maximum x and y values. An example is shown below for a 256 by 256 point matrix. Press the “Calibrate” button to update the plot.

Using the “By selection” option you select a width and height for your x and y axes range and then specify the value of a reference coordinate (x0,y0) which you then select interactively using the mouse.

 

In this example the centre of the image is to have the coordinate (4,8), while the width and height must be 10.

 

 

 

Press the Indices button to remove the current calibration and revert to index only labels.

 

Press the Update button to ensure the current calibration data in the GUI controls is correct (this should be done automatically most of the time).

 

Press the Undo button to reverse the last calibration applied.
