integrate2DRegion

 

This macro sums real or complex data in a rectangular region. This region may be interactively selected by zooming into a region, by drawing a rectangle about the region, or you can enter the region to integrate manually.

 

 

 

Pressing the Integrate button will calculate the sum of the data in the selected region.

 

The result of the integration is printing in the CLI and in the GUI status window.

 

Pressing the Update button updates the textboxes with the current region. Normally this won’t be necessary as it is done automatically when a new data set is loaded or when drawing a rectangle.
