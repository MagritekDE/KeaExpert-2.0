baselineCorrection

 

This macro applies an offset to each row or column in a 2D data set using the average value of a region selected by the user in the same row or column. Alternatively the same offset can be applied to every point in the data set using the average value of a rectangular region.

 

 

To use this macro, first select which direction you want to correct in (some data sets have varying amounts of offset in different rows or different columns). Then select a representative region which contains just noise. The macro will then apply the offset to all rows or columns.

In the above example the noise region selected was a horizontal strip below the image. For each column an average was calculated over this region and then this was subtracted off the complete column.

 

To correct using a region average, choose the “Region average” option and then select a noisy region by drawing a rectangle over the 2D plot. This will automatically update the region textboxes as shown below. Finally press the “Correct” button to calculate the average of the selected region.
