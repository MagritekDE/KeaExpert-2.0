ft2D

 

This macro applies a forward or inverse 2D FFT to the current 2D data set.

 

 

The dimensions of the data set will be copied into the zero-fill text boxes when the window is first loaded. If new data is subsequently loaded with a different dimension these new sizes can be copied to the zero fill text boxes by pressing the “Update zerofill” button. By default, the Fourier transform (FT) type will be FTEcho, which means that it expects the centre of k-space to be in the centre of the data set.

 

Press the “Do FT” button to apply the transform. “Do IFT” applies the reverse transform in a similar way.

 

Smoothing can be applied to the data set before applying the transform by selecting an appropriate apodization function in the Filter text boxes. Press the “Modify Filters” button to view and modify the available filter functions. (If you add new functions to the filter list you will have to exit the 2D FFT macro and rerun it again to see these in the filter menus).

 

For more details about using this macro see the help for nDFourierTransform.mac in the NMRI menu.

 

Pressing the Undo button will undo the last FT/IFT operation.
