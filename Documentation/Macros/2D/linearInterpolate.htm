linearInterpolate

 

The interpolate command above uses Fourier transforms to perform the interpolation. This produces smooth curves between the original data points, but you are restricted to powers of 2 in the size of the data set and the process is rather slow. In addition, you can't reduce the size of the data set. The linearInterpolate macro applies simple linear interpolation to enlarge or reduce the 2D plot data size from 0.1 to 10 times its original size ('original' means the size before the macro was run or after pressing the Update button). All this is done by dragging the slider in the GUI window:

 

 

You can modify the slider limits by selecting different numbers from the text menus. Note that the step size as you move the scaling slider is equal to the minimum scaling value (so in the above example the scaling steps are 0.1, 0.2, 0.3 ... 5).
