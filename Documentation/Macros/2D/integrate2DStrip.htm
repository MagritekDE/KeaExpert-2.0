integrate2DStrip

 

This macro integrates a group of rows or columns along the orthogonal axis and projects  the result into a 1D data set which is displayed in the 1D window.

 

 

 

To use this macro, simply choose the integration direction (either rows  or columns) and then press the “Use mouse” button. This will allow you to select the rows or columns to integrate from the current 2D image. An example is shown below:
