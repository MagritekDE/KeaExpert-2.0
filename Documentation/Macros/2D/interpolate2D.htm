interpolate2D

 

This macro allows a data set which is a power of 2 in size to be interpolated by the process of inverse FFT, zero filling and then forward FFT.

 

 

 

Typically you may wish to enlarge one part of a data set. This is most conveniently done by zooming the region while the shift key is held down. This causes the selection rectangle to have dimensions which are a power of 2. Once the region is selected, run the macro submatrix2d and then apply the interpolation macro. In the following example a 64 by 64 region has been interpolated to 512 by 512 data points.

If the region you have chosen has edges which do not fall to zero then it may be worth applying a filter to reduce “sinc” wiggles from appearing in the interpolated image.

 

Press the Undo button to reverse the interpolation process.
