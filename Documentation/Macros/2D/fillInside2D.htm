fillInside2D

 

This macro can fill the interior or exterior of a rectangular region within the current plot with a constant. The region is selected by entering the coordinates and dimensions into the GUI window’s text boxes, by drawing a rectangle over the plot, or by using the “Update” button to read the dimensions of the currently visible 2D plot.
