multiPlot2D

 

If you have a series of (single) image files (extension .pt2), and you want display them all together in a single multiplot this macro will enable you to do this quickly. (This can also be done using the 2D plot load-menu option but it is rather more inconvenient.)

 

 

 

 

 

 

 

The first step when using this macro is to choose the directory in which the plot files are stored (note that they all need to be in the same directory). Do this by selecting the button with label “...”. Once chosen, all the 1D plot files in this folder will be displayed in the left-hand list box (the extension having been removed).

 

Next, choose those plots to be displayed by either selecting them and pressing the button labelled "=>", or by double clicking on them. They will then appear in the right-hand list box.

 

The right-hand list can then be reorganised using the Delete, Up or Down buttons which work on the currently selected line in this list box.

 

The plot organisation text-menus specify the number and layout of the 1D multiplot. The first value is the number of plots in the x direction (i.e. the number of columns) while the second number is the number in the y direction (i.e. the number of rows). The plots in the right hand listbox will be copied to the multiplot in reading order i.e. left to right and top to bottom.
