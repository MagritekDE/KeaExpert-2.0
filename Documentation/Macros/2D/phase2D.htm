First select the direction of the phase correction - either rows or columns.

 

If you wish to apply an automatic zeroth order phase correction you will need to select a sample peak in the data set after clicking on the “Auto phase” button. (Click on either side of the peak).

There are two methods for autophasing the data. Either the area under the data set between the two selected points is maximised (max option) or the area under a line connected the two selected points is minimised (min option). The first method works best for a single peak while the second works best when there are multiple peaks, as it is less susceptible to baseline error. However, the max option gives more consistent results on a single peak. If the peak is broad, ensure that the selected points are well away from the peak.

 

If you wish to apply a fixed phase correction to all rows or columns in the data set, then just enter the required phase shift in degrees into the window’s single text-box and then press the “Fixed phase” button.

 

Press the Undo button to reverse the phase correction.
