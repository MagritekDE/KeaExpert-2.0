fillOutside2D

 

This macro can enlarge the current data set and then fill the extra space with a user defined constant. The extra space can be evenly placed around the original data set or appear in the lower left or upper right corners.

 

To use the macro, select the fill position mode and then the new data set dimensions. Powers of 2 are available from the pull down menus. Finally choose the fill value to appear in the new region.

 

 

 

Original 32 by 32 data set

After zero-filling to 64 by 64

 

 

Press the Undo button to reverse the filling operation and the Update button to update the width and height text boxes with the current plot dimensions.
