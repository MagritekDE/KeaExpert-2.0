columnOp

 

This macro applies a one line operation to each column in the current 2D matrix. The dialog contains a one line text box into which you can type the operation to apply to each column. At your disposal you have the following variables; x the column number, y a column vector having the value [0: 1: h-1]’, w the width of the matrix and h its height.

 

 

In the text box the current column is represented by the variable m1[x,~], m1 being the name used by the dialog for the current matrix. Essentially the macro applies the following macro

 

for (x = 0 to w-1)

   m1[x,~] = expression in text box

next (x)

 

So the default action is to do nothing.

 

Some examples

 

Suppose you want to scale the column data by its row position, then you should type the expression:

 

m1[x,~].*[0:1:h-1]’    or more simply  m1[x,~].y

 

To reverse the direction of the column data you would type

 

reflect(m1[x,~])

 

To multiply the columns by a Gaussian of width h/4

 

m1[x,~] .* exp(-(y-h/2)^2/(h/4)^2)

 

 

After loading a new 2D plot press the Update button to ensure the current plot parameters data are correct (this should be done automatically most of the time).

 

Press the Undo button to reverse the last column operation applied.
