blur2d

 

This macro applies a Gaussian blur to a 2D matrix by applying an inverse 2D FFT, multiplying by a Gaussian window in both dimensions and then applying a forward 2D FFT to this data set. Note that the matrix must have dimensions which are a power of 2.

 

 

 

The Blur slider controls the amount of blurring while the Reset button returns the image to its original form. Press the Update button when a new data set has been loaded.
