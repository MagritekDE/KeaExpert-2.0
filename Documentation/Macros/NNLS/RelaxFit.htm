RelaxFit

 

This is a simplified version of NNLS1D which just uses the plot data to perform a relaxation fit.  The user interface allows the selection of an input and output subplot. These are integers numbered from 0 increasing from left to right and top to bottom. The input and output subplot can be the same.

 

The Fitting Parameters are as described in the NNLS1D manual.

 

The Undo button will restore the output file to its original condition.
