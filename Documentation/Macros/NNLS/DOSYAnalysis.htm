NNLS DOSYAnalysis

 

This macro provides an interface for inverting stacked one-dimensional diffusion data with spectroscopic resolution sets using non-negative least squares analysis. The data set should have been collected using the PGSTE types experiments or DOSYsimulation script.

After the experiment is completed, the following interface appears. Start by clicking on the stacked spectra plot. Then press the DOSY button on the Processing control panel on the right side of the interface.

The following window will appear.

The next step includes parameters setup.

 

Preprocessing:

Stacked plot broadening in Hz units is used to apply broadening to stacked spectra before the fitting procedure.

Noise threshold sorts out the peaks with amplitude larger than this value multiplied by the standard deviation of the noise value. The corresponding level is displayed on the plot with reference spectra as a red horizontal line.

Peak prominence in % will only consider the peaks with prominence greater than the corresponding value.

 

After pressing the Apply button, reference spectrum with a horizontal line appears to fine-tune Noise threshold and Peak prominence

With mouse Scroll/Shift+Scroll carry out the fine adjustment of corresponding values.

 

Type of fit:

Inverse Laplace transform fit uses Lawson and Hanson's (L&H) algorithm to invert relaxation data.

Fitting parameters for single/bi-exponential fit:

Min. D and Max. D parameters define the borders of the relaxation time axis of processed DOSY data. The number of DOSY data points is related to Diffusion steps parameter.

Fitting parameters for inverse Laplace transform:

Similarly to exponential fit parameters, the corresponding parameters for ILT are set up.

The I.L. smoothing parameter is active only when the Inverse Laplace fit type is chosen.

The larger smoothing parameter means a smoother DOSY curve.

Display parameters:

Peak width parameter defines the width of resulting peaks in DOSY plot.

Peak height parameter is available for exponential fits and defines the fraction of the diffusion coefficient used as a peak height.

Data display

The resulting DOSY data is displayed in the main window after pressing the Apply button.

Saving the data:

When all needed data are obtained, close the DOSY processing window and, if necessary, save each plot by clicking on the particular plot and pressing the Save button in the Processing control panel. Use Save all plots in the File menu to save all plots simultaneously.
