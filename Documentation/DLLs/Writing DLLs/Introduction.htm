DLL Help - Introduction

 

 

New commands can be added to Prospa via dynamic linked libraries (DLL). A folder called "DLLs" in Prospa's home directory (available via the “appdir” variable), can contain any number of DLLs, thereby extending the functionality of the core program. These DLLs can be written using a C++ development environment such as Microsoft’s Visual C++. The advantage of this system is that it allows the program to be customised for different applications without having to modify the original code.
