dspgetmsdata

 

 

Read data produced by a pulse program while running in multi-scan fast mode

 

Syntax

 

result = dspgetmsdata(points [,port])

 

points ... the number of data points to read.

port ..... an optional USB port (0,1,2 ...). Default is 0.

result ... the returned data vector

  

 

Description

 

In fast mode only a small data set can be read each scan. This is limited to 128 complex data points (i.e. 256 data points in total).
