dspwrite

 

 

Send a float vector containing DSP program data from Prospa to the DSP.

 

Syntax

 

dspwrite(memory, address, vector)

 

memory ... which DSP memory to write to: x/y/p (STR)

address .. memory address to start writing to. (INT)

vector ... data to be sent to the DSP. (VEC)

 

Description

 

This command is the companion of dspread. It takes a 1D float vector – converts each element to a 24 bit integer word and then sends these via the USB port to the specified DSP memory location.
