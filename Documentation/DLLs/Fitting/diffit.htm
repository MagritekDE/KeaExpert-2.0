diffit

 

 

Perform a single exponential fit characteristic of PGSE diffusion experiments.

 

Syntax

 

(E, E0, D, errE0, errD) = diffit(type, x, y, noise, var1, var2 )

 

E ........ best fit to x,y data (FLOAT)

E0 ....... amplitude of exponential (FLOAT)

D ........ Self diffusion coefficient (FLOAT)

   errE0 .... error in amplitude of exponential (FLOAT)

errD ..... error in self diffusion coefficient (FLOAT)

type ..... type of fit: either "grad","sdel" or "ldel".

x ........ abscissa data; either G, D or d (VEC)

y ........ amplitude attenuation data (VEC)

noise .... rms noise of supplied amplitude data (0 if not known) (FLOAT)

var1 ..... first independent parameter (G, D or d) (FLOAT)

var2 ..... second independent parameter (G, D or d) (FLOAT)

 

Description

 

Attempts to fit an exponential function of the form to the supplied x,y data

 

 

where one of the parameters G ("grad"), D, ("ldel") or d,  ("sdel")  is varied and the other 2 are kept constant.

 

Standard deviations in fit parameters are also returned.

 

 

See also

 

t1fit, t2fit.
