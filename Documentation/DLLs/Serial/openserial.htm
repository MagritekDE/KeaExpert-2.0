openserial

 

 

This command opens a serial port connection.

 

Syntax

 

openserial([port, baudrate, databits, parity, stopbit])

 

port ...... serial port, one of: com1, com2 ... (STR)

baudrate .. transfer speed: e.g. 2400, 4800, 9600, 19200, ... (INT)

databits .. number of bits per data word, one of 6,7,8 (INT)

parity .... none, even, odd (STR)

stopbit ... 1, 1.5 or 2 (FLOAT)

 

Default values for these arguments are

 

com1, 9600, 8, none, 1

 

The current defaults can be determined by typing

 

pr openserial("getargs")

 

At present there the serial flow control defaults to “none”

 

Following is an example macro which uses these DLL commands to control a hypothetical stepper motor. In this case the motor controller returns OK after each successfully completed command.

 

Example

 

openserial("com1",9600,7,"even","1")

writeserial("initialise")

if(readserial() != "OK") # Wait for motor controller to initialise

  abort("initialisation error")

endif

 

for(s = 0 to 10)

  writeserial("move(10,0)")  # Move 10 steps along the x axis

   if(readserial() != "OK")  # Wait for motor to complete movement

     abort("move error")

   endif

next(s)

 

closeserial()
