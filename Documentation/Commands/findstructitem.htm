findstructitem

 

Returns the index of an item inside a structure

 

Syntax

 

index = findstructitem(structure, item)  

 

Description

 

Normally you can access a structure item using the arrow syntax, e.g. s->b. However it is also possible to use an index using the getstructitem command. Use this command to find a specific index when you know the name of the item.

 

Example

 

> s = struct(a=23, b="Hi", c=[1,2,3,4])

 

 > findstructitem(s,"b")

 

   findstructitem(s,"b") = 1

 

> getstructitem(s,1)

 

   "b"   "Hi"
