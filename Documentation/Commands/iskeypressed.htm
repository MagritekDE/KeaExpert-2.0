iskeypressed

 

Check to see if a particular key is pressed or not

 

Syntax

 

(procedures,argumnts) = iskeypressed(key)

 

key ....... which key to check (STR)

 

Description

   

key can be one of the following strings: "shift", "control", "enter", "up", "down", "left", "right" or the keyboard letter (e.g. "A" or "1"). This command is useful if you want to modify the action of a standard button, for example by holding down the shift key when it is pressed.

 

Example

 

In this example the program sits in a while loop printing zeros until the number 1 is pressed. If shift+A is pressed it aborts.

 

procedure(iskeypressedTest)

 

   while(1)

  

      pr  iskeypressed("1")

     

      if(iskeypressed("A") & iskeypressed("shift"))

       exitwhile

      endif

 

   endwhile()

 

endproc()
