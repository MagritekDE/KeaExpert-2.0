logspace

 

Generate a vector with logarithmically spaced element (same as logvec)

 

Syntax

 

result = logspace(first,last,number)    (eee:fff)

 

first .... first value in vector (FLOAT/DOUBLE)

last ..... last value in vector  (FLOAT/DOUBLE)

number ... number of elements in vector (INT)

result ... vector of number elements (VEC/DVEC)

 

Description

 

This command is equivalent to the Prospa procedure

 

   procedure(logSpace,mn,mx,N)

 

   stp = (log10(mx)-log10(mn))/(N-1)

   x = [log10(mn): stp: log10(mx)]

   xl = 10^x

 

 endproc(xl)

 

and produces numbers which are equally spaced when displayed on a log axis.

 

 

 

Example

 

x = logspace(1,100,10)

pr x'

plot(x)

 

   x' =

 

         1

    1.6681

   2.78256

   4.64159

   7.74264

   12.9155

   21.5443

   35.9381

   59.9484

 100

 

 

 

 

Note

 

The start and end points should be > 0.
