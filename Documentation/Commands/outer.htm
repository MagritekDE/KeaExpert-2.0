outer

 

Form the outer product between two matrices.

 

Syntax

 

result = outer(mat1, mat2)     (ee:m/cm,m/cm)

 

mat1 ..... first matrix (dimensions m*n) (MAT/CMAT)

mat2 ..... second matrix (dimensions t*s) (MAT/CMAT)

result ... outer product (dimensions m*t by n*s) (MAT/CMAT)

 

Description

 

This command forms the outer product of two matrices or vectors.

 

Example

 

v1 = [1,2,3]

v2 = [1,2,3]

pr outer(v1,v2')

 

outer(v1,v2) =

 

       1   2   3

2   4   6

3   6   9

 

v1 = [1,2;3,4]

v2 = [1,1;1,1]

pr outer(v1,v2)

 

outer(v1,v2) =

 

1   1   2   2

1   1   2   2

3   3   4   4

3   3   4   4
