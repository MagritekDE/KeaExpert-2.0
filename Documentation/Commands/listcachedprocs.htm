listcachedprocs

 

Lists all the procedures which are currently cached.

 

Syntax

 

listcachedprocs(mode)

 

mode ........ which cache to list: "window" or "local" (STR)

 

Description

 

This is typically used as a debugging tool.

 

Example

 

procedure(listcachedprocsTest)

 

   cd("$appdir$\\Macros\\1D_Macros")

   cachemacro("linfit.mac","local")

   cachemacro("stats1d.mac","local")

   cacheproc("true")

   listcachedprocs("local")  

 

endproc()
