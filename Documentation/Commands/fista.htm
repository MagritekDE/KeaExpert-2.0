fista

 

Performs a 2D inverse Laplace transform using FISTA (Fast-Iterative-Shrinking Threshold Algorithm).

 

Syntax

 

mOut = fista(nmrData, xKernel, yKernel, spectraGuess, smoothing, numberInterations, [mode, plotRegion, progressControl])

 

nmrData ........ the NMR data to invert (DMATRIX)

xKernel ........ the kernel function for the x (horizontal) dimension (DMATRIX)

yKernel ........ the kernel function for the y (vertical) dimension (DMATRIX)

spectralGuess .. an initial guess to the result (DMATRIX)

smoothing ...... the smoothing parameter (DOUBLE)

nrIteractions .. how many iterations to try before exiting (INTEGER)

mode ........... 0 – no intermediate output (default), 1 – update image each 100 iterations, 2 - update progress control each 100 iterations (INTEGER) 3- update image and progress control each 100 iterations.

plotRegion ..... plot control for display of results (PLOT_OBJECT)

progressCtrl ... control number to display progress (INTEGER)

mOut ........... the inverted data (DMATRIX)

 

Example

 

This example inverts a data set consisting of 3 two dimension exponential functions with time constants (0.1,0.33), (0.33,0.1) and (0.033,0.033)

 

procedure(fistaTest)

 

   szIn = 100

 

# Generate input axes and data

   xAxis = linvec(1, 99, szIn)/100

   yAxis = linvec(1, 99, szIn)/100

   mIn = (exp(-xAxis*10))'*(exp(-yAxis*3)) + @

         (exp(-xAxis*3)) '*(exp(-yAxis*10)) + @

         (exp(-xAxis*30))'*(exp(-yAxis*30))

 

# Generate output axes

   szOut = 50

   xSpec = logvec(1, 99, szOut)/100

   ySpec = logvec(1, 99, szOut)/100

   solution = dmatrix(szOut,szOut)

   smth = 0.001

   nrIterations = 5000

 

# General kernels

   xK = exp(-xAxis'*(1/xSpec))

   yK = exp(-yAxis'*(1/ySpec))

 

# Do the inversion

   mInvert = fista(double(mIn),double(xK),double(yK),double(solution),

                   double(smth),nrIterations)

 

# Display the result

   plt = curplot("2d")

   plt->image(mInvert,[xSpec[0],xSpec[-1]],[ySpec[0],ySpec[-1]])

   plt->axes->xmapping("log")

   plt->axes->ymapping("log")

 

endproc()
