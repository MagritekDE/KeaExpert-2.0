requestlicense

 

Saves a license request file to the desktop.   

 

Syntax

 

requestlicense()

 

 

Description

 

Normally the file ‘prospaLicenseRequest.txt’ is generated when Prospa is first run. However if you want to extend your license to other products you may need to regenerate this file. This should then be sent to Magritek with the request to add additioal products to the license.
