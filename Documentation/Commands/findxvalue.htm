findxvalue

 

Interpolate to find the x axis value which matches specific y values. Starts the search from a specific x axis index in either left or right directions

 

Syntax

 

(FLOAT x1, x2 ...) = findxvalue(VEC xAxis, VEC yAxis, INT startXIndex, STR direction, FLOAT y1, FLOAT y2, ...)  ("eeeeee..","vvltff..")

 

xaxis ....... x axis vector (VEC)

yaxis ....... y axis vector (VEC)

start ....... index to start searching from (INT)

direction ... which direction to search from: "left"/"right" (STR)

yi .......... value in y array for which the x value is desired. (FLOAT)

 

xi .......... value in x array corresponding to yi. (FLOAT)

 

Description

 

This command is useful when you have a y-value in a 1D dataset and you would like to find the corresponding x-value  when searching from a specified start point and in a certain direction. For example if you have data set which contains multiple peaks and you would like to find the linewidth of one of the peaks you would call the command twice with startXIndex set to the top of the peak and y1 to half the peak height. The first call would have direction equal to "left" and the second call direction equal to "right". Linear interpolation is used to obtain a reasonably accurate value for the x values.

 

Example

 

This example generates 3 spectral lines each with a full-width half maximum linewidth of  about 6 Hz.  It then uses the findxvalue commands to determine this linewidth.

 

N = 512

t = [0:N-1]/N

f = ([0:N-1]-N/2)

y = 10*exp(-i*2*pi*100*t-t*20)

y = y + 5*exp(-i*2*pi*50*t-t*20)

y = y + 15*exp(-i*2*pi*-50*t-t*20)

y[0] = y[0]/2

s = real(ft(y))

plot(f,s)

 

(py,px) = max(s) # find index of -50 Hz peak

 

left =  findxvalue(f,s,px,"left",py/2) # find index of left half height

right =  findxvalue(f,s,px,"right",py/2) # find index of left half height

pr "\n   FWHM = $(right-left)$ Hz\n" # multiply width in points by the frequency resolution

 

(right-left)*(f[1]-f[0]) = 6.42 Hz

 

The correct value is 6.37 Hz, the small difference is because linear interpolation is not ideal for a Lorenzian lineshape. However the result is closer to the ideal than using the simpler findindex2 command.
