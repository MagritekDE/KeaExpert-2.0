pr

 

Print something to the command line interface.

 

Syntax

 

pr(expression)   (e:s,f,d,l,c,m,cm,v)

pr(global)

pr(hidden)

pr(local)

pr(winvar)

 

expression is evaluated and the result is printed to the CLI or to a file (if the printto command was applied first).

 

The second option lists all visible global variables currently defined, the third hidden global variables, the fourth lists the local variables defined within the current macro, and the fifth lists all window variables.

 

There are some predefined hidden global variables which are always present. Some of these are listed below:

 

appdir ..................   location of Prospa executable, macro, documentation and DLL folders.

colorscales .............   a list of colorscales to display in the 2D plot.

dllsearchpath ...........   a list with the directories to search to find DLL commands.

macrosearchpath .........   a list with the directories to search to find macro names.

prefdir .................   location of preferences folder.

usermenus ...............   a list with the names and directories to use for the main menu.

workdir .................   a general purpose directory which is saved on exit.

 

copper ..................   colorscales for use in 2D and 3D plots.

grey

hot

rainbow

spectrum

 

ans .....................   the answer variable – updated after each command has been run or an expression has been evaluated

 

It is often useful to be able to print strings with embedded numerical values. In Prospa this is done by surrounding the numerical expression with dollar signs and optionally following the expression with a formatting statement.

 

Examples

 

v = [1,2,3]

pr(v*2)

  v*2 = [2,4,6]

 

a = "The value of pi = $pi,1.3f$"

pr (a)

   The value of pi = 3.142

 

The format specifier is the same as that used in the C printf statement except that the initial % symbol is not used. The same technique can also be used in the title, xlabel and ylabel commands. Other symbols which can be embedded in the string include \n to move to the next line, \r to move to the start of the line, \t for tab, and +- for ±.

 

Note that when printing large matrices you can halt the printout by pressing the escape key.
