shufflelist

 

Randomly reorder a list.

 

Syntax

 

result = shufflelist(list)   (e:v)

 

 

result ... modified list (LIST)

list ..... source list (LIST)

 

 

Description

 

Randomly changes the order of entries in a list and copies this to a new list.

 

Example

 

lst = ["a = 1", "b = 2", "c = 3", "d = 4"]

 

pr shufflelist(lst)

 

 

   shufflelist(lst) = {

       c = 3

       d = 4

       a = 1

       b = 2

   }
