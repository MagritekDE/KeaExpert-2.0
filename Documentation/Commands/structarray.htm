structarray

 

Make an array of structures

 

Syntax

 

result = structarray(dimension) (e)

 

dimension ... the number of entries in the array (INTEGER)

result ...... the resultant empty structure array (STRUCTURE_ARRAY)

 

Description

 

Sometimes you might want as series of structures with the same keys but different values. The structarray allows this

 

Example

 

people = structarray(2)

people[0] = struct(name = "Josh", number = 12345)

people[1] = struct(name = "Kevin" number = 67890)

 

pr people

 

   people[0] = {

          name = "Josh"

          number = 12345

   }

 

 

   people[1] = {

          name = "Kevin"

          number = 67890

   }

 

pr people[0]->number

 

   people[0]->number = 12345
