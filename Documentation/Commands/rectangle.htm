rectangle

 

Add a rectangle to a 2D matrix.

 

Syntax

 

matOut = rectangle(matIn, x, y, w, h, a)

 

matIn .... input matrix to add rectangle to.

x,y ...... coordinates of centre of rectangle.

w,h ...... half the width and height of the rectangle.

a ........ value to fill rectangle with.

matOut ... matrix with filled rectangle added.

 

Note that all coordinates/dimensions are matrix indices.

 

Description

 

This command adds a filled rectangle to a matrix.


Example

 

m = matrix(1000,1000)

 

for(k = 0 to 50)

  x = rand(1000)

  y = rand(1000)

  a = rand(100)

  w = rand(100)

  h = rand(100)

  m = rectangle(m,x,y,w,h,a)

  image(m)

next(k)
