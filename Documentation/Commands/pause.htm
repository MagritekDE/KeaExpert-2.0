pause

 

Wait a predefined length of time.

 

Syntax

 

pause(time [, mode, event]))   (eee:fss)

 

time ... the pause time in seconds

mode ... whether to sleep during the pause (sleep/nosleep)

event .. whether to consume windows events during pause (events/noevent)

 

Description

 

This statement will halt the macro for the specified time (in seconds). The smallest value of time is 0.001s. It is sometimes useful when trying to see rapidly changing graphics. The 'sleep' option allows other threads to work during the pause, however this limits the minimum pause time to 1 ms. The 'events' option allows windows to resized and moved during the pause option. If the 'nosleep' and 'noevent' options are chosen then pauses as short as a 10 microseconds are possible (depending on computer speed). For long pauses it makes sense to include at least the 'events' option to keep Prospa interactive.

 

Example

 

# Draw a moving spiral

 

t = [0:0.1:30*pi]

 

for(p = 1 to 20 step 0.5)

   x = cos(t).*exp(-t/p)

   y = cos(t+ pi/2).*exp(-t/p)

   draw1d("false")

   plot(x,y)

   title("p = $p,1.1f$")

   draw1d("true")

   pause(0.1) # Pause for 1/10 sec.

next(p)
