threadcleanup

 

Delete any memory associated with a thread

 

Syntax

 

result = threadcleanup(threadID)

 

threadID ...... the thread identifier (as returned by the thread command (INTEGER)

result ........ returns 1 if the memory was deleted 0 if the thread has already been cleaned.

 

Description

 

If possible this command should run when a thread finishes, as it cleans up some residual memory allocated to the thread. It is not necessary to call this if the threadwait or threadabort commands have been used.

 

See also

 

thread, threadabort, threadwait
