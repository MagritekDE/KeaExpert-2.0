findmacro

 

Find the first instance of the passed macro in the Prospa search path.

 

Syntax

 

(path, name) = findmacro(macro_name)    (e:s)

 

macro_name .... The macro name with or without an extension (STR)

path .......... The folder where the macro can be found.

name .......... The macro name with extension

 

Description

 

Finds the first instance of the passed macro in the Prospa search path returning the path and full file name. If not provided the extensions searched are .mac and then .pex.

Note that you should avoid having multiple macros with the same name.

The command will return nulls if it can’t find the macro.

 

Example

 

 > findmacro("editorWin")

 

   "C:\Users\Craig\Macros\Windows_Layout"   "editorWin.mac"
