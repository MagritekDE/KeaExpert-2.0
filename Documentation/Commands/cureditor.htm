cureditor

 

Get or set the current text editor

 

Syntax

 

obj = cureditor([winNr, objNr])   (ee:dd)

 

winNr ... The window the editor belongs to (INT)

objNr ... The object number for the editor (INT)

obj ..... Returned object if no arguments passed (OBJ)

 

Description

 

If winNr and objNr are defined, then the current text editor is set to the specified object. If winNr is not defined, then the CLI object is returned.
