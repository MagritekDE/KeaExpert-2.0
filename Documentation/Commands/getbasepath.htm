getbasepath

 

Returns the parent path of the current folder or file

 

Syntax

 

result = getbasepath(str filename/path)

 

 

filename/path ...... a full or partial filename path (filename may be included)

result ............. the parent path

 

Description

 

Where as getbasedir returns the just the directory name directly above the filename or path passed this command returns the complete path of that directory

 

Example

 

This simple example shows the difference between getbasedir and getbasepath

 

pr appdir

pr getbasedir(appdir)

pr getbasepath(appdir)

 

appdir = C:\Users\Craig\Documents\Projects\Prospa 3.x Release Current

 

getbasedir(appdir) = Prospa 3.x Release Current

 

getbasepath(appdir) = C:\Users\Craig\Documents\Projects
