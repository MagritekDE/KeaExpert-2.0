funcedit

 

Provides access to a number of text editor functions which are not accessible using normal commands.

 

Syntax

 

funcedit(command)  (c:s)

 

command ... Name of function to perform (STR)

 

Description

 

When Prospa 3 was formed many menu and toolbar functions were not accessible using the normal command or macro interface. With time most of these will be made more accessible in this way but in the meantime this command provides the only external access to these facilities. Note that all these commands work on the current text editor.

 

Typically these commands would be used in a menu or toolbox interface.
