setstruct

 

Initialise the values in a structure.

 

Syntax

 

result = setstruct(structure, parameter_1_value, parameter_2_value, parameter_3_value, ...)

 

structure............. the name of the structure to initialise (STRUCT)

parameter_n_value .... a pariable which will be assigned to the nth parameter (VARIANT)

result ............... the resultant structure after initialisation

 

Description

 

Rather than setting each member in a structure with a separate statement you can do it in one go with this command.

 

Example

 

procedure(setstructTest)

 

  customer = struct("name","age","sex","extension")

  customer = setstruct(customer,"John",23,"male",12345)

  pr customer

 

endproc()

 

Output:

 

   customer = {

          name = "John"

          age = 23

          sex = "male"

          extension = 12345

   }
