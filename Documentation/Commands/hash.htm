hash

 

Take a vector and convert it to a single integer value. Can be used to determine the integrity of data which has been transferred

 

Syntax

 

result = hash(data)

 

data ........ the vector to have its hash value calculated (VEC)

result ...... the resultant hash value (INT)

 

Example

 

 

   procedure(hashTest)

 

   a = [1:1000]

   result = hash(a)

   pr result

   a[999] = a[999]-1

   result = hash(a)

   pr result

 

endproc()
