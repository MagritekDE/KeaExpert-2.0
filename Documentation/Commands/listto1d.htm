listto1d

 

Converts a 2D list into a 1D list.

 

Syntax

 

list1d = listto1d(2dlist)

 

list2d........ the list to convert (LIST2D)

list1d........ the resultant 1D list (LIST1D)

 

Description

 

This command converts a 2D list to a 1D list to allow for easier interation (since 2D lists can be jagged) if the 2D information is not important.

 

 

Example

 

l2 = list(2,2)

l1 = listto1d(l2)

 

pr l2

 

   l2 =

       "item 0,0",   "item 1,0"

       "item 0,1",   "item 1,1"

 

pr l1

 

   l1  = {

       item 0,0

       item 1,0

       item 0,1

       item 1,1

   }
