sound

 

Play a vector as a sound.

 

Syntax

 

sound(v, [sr, [scale]])   (eec:mfs)

 

v ....... vector to play (VEC/MAT)

sr ...... sampling rate in Hz (default 8192) (INT)

scale ... one of "fixed" or "scale".  (STR)

 

Description

 

Vector v (1D row matrix) is sent to the sound output as a 16 bit number using the sampling rate specified by sr. If scale is "scale" (the default), the vector is scaled so that the maximum value is 32767 and the minimum value is -32768. If the scale is "fixed" the data is left untouched except to be clipped to this range.

 

If v is a 2D matrix with 2 rows then the first row will play to the left channel and the second to the right channel assuming the sound card supports stereo.

 

Example

 

sps = 16384

f = 2300

t = linvec(0,sps*50-1,sps*50)/sps

f = 500*sin(t*30) + 2300

yl = 30000*sin(2*pi*f.*t/8)

yr = 30000*sin(2*pi*f.*t)

y = matrix(sps*50,2)

y[~,0] = yl

y[~,1] = yr

sound(y,sps/2,"fixed")
