query

 

Display a pop-up message with “yes/no” response options.

 

Syntax

 

response = query(title,message,[[default, mode])  (eeee:ssss)

 

title ...... text to appear in title bar. (STR)

message .... message to appear in window. (STR)

default .... default selected choice, either “yes” (default) or “no”. (STR)

mode ....... display mode, either “yes/no” (default) or “yes/no/cancel”. (STR)

response ... returns user’s choice of "yes", "no" or (depending on mode) “cancel”. (STR)

 

Examples

                                                     

r = query("Warning","This will delete all data - continue?", "yes")

 

 

r = query("Warning","There are unsaved files.\r\rDo you want to save them before exiting?","yes","yes/no/cancel")

 

 

r will take the value yes, no or cancel on return. As in the above example, if your message is very long you can force it to appear on multiple lines by including carriage returns \r.
