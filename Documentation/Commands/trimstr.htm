trimstr

 

Removes leading and/or trailing spaces from a string

 

Syntax

 

result = trimstr(source, [method[, character]])

 

source ........ the string to be trimmed (STR)

method ........ which parts of the string to trim ("front"/"back"/"both") (STR)

character ..... characters to be removed (default is " " i.e. spaces) (STR)

result ........ the trimmed string (STR)

 

 

Example

 

txt = " Hi there "

txt = trimstr(txt, "front")

pr "\"$txt$\"\n"

 

"Hi there "

 

txt = trimstr(txt, "back")

pr "\"$txt$\"\n"

 

" Hi there"

 

txt = trimstr(txt, "both")

pr "\"$txt$\"\n"

 

"Hi there"
