min

 

Finds the minimum value in a 1D, 2D, or 3D real matrix.

 

Syntax

 

(m,x) = min(expression)      (e:m)   # 1D version

(m,x,y) = min(expression)    (e:m)   # 2D version

(m,x,y,z) = min(expression)  (e:m)   # 3D version

 

expression ... an expression resulting in a matrix. (MAT)

m ............ minimum value found in matrix (FLOAT)

x,y,z ........ indices of minimum value (INT)

 

Description

 

This command searches through the matrix produced by evaluating expression and returns the smallest value m. It also returns the position of the minimum value in (x,y,z).

 

Example

 

A = [1,2;3,4;5,6]

pr(A)

   A =   1    2

         3    4

         5    6

pr(min(A))

   min(A) = 1 0 0
