listto2d

 

Converts a 1D list to a 2D list

 

Syntax

 

list2d = listto2d(list1d)

 

list1d........ the list to convert (LIST)

list2d........ the resultant 2D list (LIST2D)

 

Description

   

This command is useful if you have a macro which expects a 2D list but you have only a 1D list.

 

Example

 

This example prints out the elements of a 2D and then a 1D list. If the list is 1D it is first converted to a 2D list

 

procedure(listto2dTest)

 

   print("\n2D list")

   lst = ["1";"2","3";"4","5","6"]

   :printList(lst)

   print("\n1D list")

   lst = ["1","2","3","4","5","6"]

   :printList(lst)

 

endproc()

 

procedure(printList,lst)

 

   if(vartype(lst) == "list")

      lst = listto2d(lst)

   endif

   (w,h) = size(lst)

   for(y = 0 to h-1)

      for(x = 0 to w[y]-1)

         pr "\nRow = $y$, Col = $x$, Value = $lst[x,y]$"

      next(x)

   next(y)

 

endproc()

 

Output

 

2D list

Row = 0, Col = 0, Value = 1

Row = 1, Col = 0, Value = 2

Row = 1, Col = 1, Value = 3

Row = 2, Col = 0, Value = 4

Row = 2, Col = 1, Value = 5

Row = 2, Col = 2, Value = 6

1D list

Row = 0, Col = 0, Value = 1

Row = 0, Col = 1, Value = 2

Row = 0, Col = 2, Value = 3

Row = 0, Col = 3, Value = 4

Row = 0, Col = 4, Value = 5

Row = 0, Col = 5, Value = 6
