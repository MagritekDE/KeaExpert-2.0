relpath

 

Express a directory path relative to a variable.

 

Syntax

 

path = relpath(path_of_interest, variable_name[, delimiters]) (eee:svs)

 

path ............... relative path to path_of_interest from variable_name (STR)

path_of_interest ... a directory reference (STR)

variable_name ...... a variable name (VAR)

delimiters ......... two characters, prefix and suffix to variable_name in resulting path. Defaults to “$$” (STR)

 

Description

 

This command expresses path_of_interest, a directory or folder reference, in terms of variable_name.

 

The delimiters around the result literal representation of variable_name can be set using the delimiters parameter. delimiters[0] will then appear immediately before variable_name in the result, and delimiters[1] will appear immediately after variable_name.

 

This syntax allows folder names connected with the Prospa directory to be recorded in a compact way, and in addition to remain valid even after the folder is moved.

 

Example

 

dir = "C:\Documents and Settings\Craig\My Documents\data"

pr relativepath(dir,appdir)

 

   $appdir$\..\..\data\

 

pr relativepath(dir,appdir,"<>")

 

   <appdir>\..\..\data\
