plotpref

 

Set one or more preferences for new 1D or 2D plots.

 

Syntax

 

plotpref({parameter, [value]})

 

parameter ..... plot parameters to modify (see below) (STR)

value ......... new value for plot parameter (VARIOUS)

 

PARAMETER               VALUE

 

antialiasing ........ "true", "false"

axescolor ........... [R,G,B]

axesfontcolor ....... [R,G,B]

axesfontname ........ string

axesfontsize ........ integer

axesfontstyle ....... integer

axestype ............ "corner"/"box"/"cross"/"xaxis"/"yaxis"/

                      "xaxis_box"/"yaxis_box"/"xaxis_cross"/"yaxis_cross"

barcolor ............ [R,G,B]              

bkcolor ............. [R,G,B]

bordercolor ......... [R,G,B]

errbarcolor ......... [R,G,B]

errbarfixed ......... "true", "false"

errbarshow .......... "true", "false"

errbarsize .......... integer

finegridcolor ....... [R,G,B]

gridcolor ........... [R,G,B]

imagcolor ........... [R,G,B]

imagstyle ........... [0-4]

labelcolor .......... [R,G,B]

labelfontname ....... string

labelfontsize ....... integer

labelfontstyle ...... integer

majorticklength ..... integer

minorticklength ..... integer

realcolor ........... [R,G,B]

realstyle ........... [0-4]

symbolcolor ......... [R,G,B]

symbolshape ......... "opensquare", "square", "opencircle", "circle", "opentriangle", "triangle", "openinvtriangle", "invtriangle", "opendiamond", "diamond", "cross", "plus", "none"

symbolsize .......... [1-19]

titlecolor .......... [R,G,B]

titlefontname ....... string

titlefontsize ....... integer

titlefontstyle ...... integer

tracetype ........... "lines", "stairs", "dots", "none"

tracewidth .......... integer

 

Description

 

This command modifies the default setting for new 1D and 2D plots. Existing plots are not affected by this command.

 

If a single parameter is requested and a value is not entered, then the command will return the current default value for that parameter.

 

Example 

 

 

plotpref("bkcolor",[100,100,100])

plotpref("realcolor",[255,0,0])

multiplot("1d",1,1)

x = [0:0.1:10]

plot(x,x^2)

 

Note the multiplot command, which replaces the current plot with a new plot. Without it, the plotpref commands would do nothing.
