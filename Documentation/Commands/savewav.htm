savewav

 

Save a vector as a Microsoft .wav sound file 

 

Syntax

 

savewav(fileName, vector, samplingRate)

 

fileName .......... name of wav file to write (STR)

vector ............ vector to convert to a wav file (REAL 1D MATRIX)

samplingRate....... samples per second for wav file  (INT)

 

Description

 

This command converts a real vector into a file which can be played using most audoplayers.  Note that the vector range should be between -32768 to 32767.

 

Example

 

This example generates a two frequency FID and copies it to a .wav file. This can then be opened using (e.g.) Windows Media Player.

 

procedure(savewavTest)

 

  t = linspace(0,2,20000)

  f1 = 200

  f2 = 205

  y = cos(2*pi*f1*t).*exp(-t*2)

  y = y+cos(2*pi*f2*t).*exp(-t*2)

  plot(t,y)

  savewav("$desktop$\\test.wav",y*15000,10000)

  sound(real(y),10000)

 

endproc()
