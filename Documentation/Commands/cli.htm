cli

 

Generate a command line interface window.

 

Syntax

 

OBJ obj = cli(INT nr, INT x, INT y, INT w, INT h)

 

nr ............ control number ID

x,y ........... top left coordinates of control in pixels

w,h ........... width and height of control in pixels  

obj ........... access to cli internal functions and parameters (optional)

 

Description

 

This command produces a command line interface (CLI) window. This can be used for entering Prospa commands directly.

 

The CLI object has a large number of underlying functions which can be used for modifying how it appears and behaves. These functions can be accessed using the returned object obj (also accessible using the getobj() command). The object functions are listed below.

Class function list for cli object

 

Note : All functions should be prefixed with the cli object name e.g. obj->copy()

 

copy() .......................... copy selected text to clipboard.

paste() ......................... copy clipboard to current CLI insertion point.

cut() ........................... cut selected text and copy to clipboard.

undo() .......................... undo last modification to CLI text.

menu(STR location, STR menu) .... set cli contextual menu.

OBJ menu = menu(STR location) ... get cli contextual menu.

 

For generic object parameters which may be set or get see commands setpar/getpar
