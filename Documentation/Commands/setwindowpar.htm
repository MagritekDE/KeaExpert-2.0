setwindowpar

 

Modify parameters for the specified GUI window.

 

Syntax

 

setwindowpar(window_number, parameter, value)  eee:dtv

 

window_number ... window to modify (INT)

parameter ....... parameter to change – see below (STR)

value ........... new value for this parameter (VARIOUS)

 

 

Parameter

Value

 

"bgcolor"

 

1x3 matrix – RGB vector (0-255) sets background colour.

 

"constrained"

 

 

"debug"

 

"dimensions"

"true"/"false" - Whether the window dimensions are constrained by a parent window.

 

"true"/"false" – Enable or disable the debugger.

 

[x,y,w,h] (vector or list)

If arg is a vector then the dimensions are fixed. If arg is  a list it can include parent window relative dimensions e.g. ww/2+10.

(Format [ww|wh][/|*]float[+|-]float)

 

"exit_procedure"

string - Command to run when window is closed.

 

"focus"

 

"gridspacing"

integer - Object to gain focus (via object number).

 

float – Set grid spacing.

 

"height"

 

"keepinfront"

 

 

"mainwindow"

integer - window height. 

 

"true"/"false" (string) - Keep window in front of normal GUI windows.

 

Sets this window to be the main Prospa window. 

 

"menu"

 

integer – Adds a menu and accelerator table to the window

 

"menubar"

 

1D matrix - Array of menu objects to display in window menu bar. 

 

"name" 

 

string – Set user definable tag for the window

"permanent"

 

 

"position"

 

"true"/"false" - Whether window can be deleted (closed) by user.

 

1x2 matrix – Set the window’s x,y position.

"resizable"

 

"showgrid"

"true"/"false" – Enables and disables resizability.

 

"true"/"false" – Enables and disables grid visibility.

 

"show_menu"

 

 

"sizelimits"

"true"/"false" – Displays and hides the GUI editor contextual menu.

 

1x4 matrix – Set the minimum width, maximum width, minimum height, and maximum height for a window.

"snaptogrid"

 

"true"/"false" (string) – Enables and disables grid snapping.

 

"statusbox"

 

"title"

 

"titleupdate"

integer - Status box to display by object number.

 

string - Window title.

 

"true"/"false" (string) – Enables and disables title updating.

 

"toolbar"

 

"type"

 

integer - Toolbar to display, by object number.

 

string - Type of window. Options are "dialog".

 

"width"

integer - Window width 

 

"x"

integer - Window x-position

 

"y"

integer - Window y-position

 

Description

 

Sets a parameter for a specified window. window_number is the number returned by the window command, parameter is the name of the parameter (listed above) and value is the value to be assigned to this parameter.

 

If window_number is zero then the current GUI window will be modified.

 

Example

 

n = window("", 1, 1, 1, 1)

button(1,10,10,40,20,"OK",closewindow(0);)

setwindowpar(n,"x",100)

setwindowpar(n,"y",100)

setwindowpar(n,"width",200)

setwindowpar(n,"height",200)

setwindowpar(n,"title","My square window")

setwindowpar(n,"exit_procedure","pr \"Exit\"")

setwindowpar(n,"show_menu","false")

setwindowpar(n,"focus",1)

showwindow(n)

 

This displays a square blank window at position (100,100) of size 200 by 200. When the window is closed using the close button in its title bar it prints “Exit” in the CLI. Pressing <Enter> will also close the window since the OK button has the focus. The window does not show a contextual menu.
