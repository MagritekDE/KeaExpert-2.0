getparentpath

 

Given a directory path this will strip off the base filename or base directory and return the rest of the path

 

Syntax

 

output = getparentpath(input)

 

input ... The directory to process; full or relative (STR)

output .. The parent directory to input

  

Example

 

> dir = "C:\\Users\\user_name\\Applications\\SpinsolveExpert\\Macros"

 

> pr(getparentpath(dir))

 

    C:\Users\user_name\Applications\SpinsolveExpert

 

> file = "C:\\Users\\user_name\\Applications\\SpinsolveExpert\\Macros\\X\\1Pulse-X"

 

> pr(getparentpath(dir))

 

    C:\Users\user_name\Applications\SpinsolveExpert\Macros\X

 

 

 Note that it is important to escape the directory backslashes on explicit pathname strings to prevent certain escaped characters being replaced (e.g. \t, \n, \r)
