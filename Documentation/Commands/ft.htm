ft

 

Fast Fourier transform (radix 2).

 

Syntax

 

result = ft(expression)

 

expression ... vector to transform (VEC/CVEC)

result ....... transformed vector (CVEC)

 

Description

 

This command calculates the fast Fourier transform of expression and places it in the result vector. expression must resolve into a 1D row or column, real or complex matrix with a size equal to a power of 2. If the data is real it is copied into a complex matrix first with the imaginary part zeroed. The returned result is always a complex row or column matrix.

 

Example

 

x = [0:1:1023]/10

y = exp((i-0.2)*x)

plot(x,ft(y))
