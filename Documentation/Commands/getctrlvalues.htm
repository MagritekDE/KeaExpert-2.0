getctrlvalues

 

Copy GUI control names and values to a parameter list.  

 

Syntax

 

result = getctrlvalues(window_nr, [output, [which, variable]]) (eeee:dssv)

 

result ..... parameter list receiving gui data. (LIST | STRUCT)

window_nr .. specifies which gui window to examine. (INTEGER)

output...... Type of output. One of ["list" | "struct"] (default = list). (STRING)

which ...... Which controls to look at ["all" | "list" | "prefix"] (default = all). (STRING)

variable ... Name of list variable if which == "list". (LIST/STRING)

 

Description

 

This command extracts from the GUI window specified by window_nr (0 if it’s the current window) a list or structure representing the names and values of controls in the window.

 

The parameter output specifies the data structure of the result. If left unspecified, or set to "list", the result is returned in the form of a list of strings, with each string taking the form of an assignment operation. If set to "struct", the result is returned in the form of a Prospa struct, with attribute names and values corresponding to the names and values of the controls in window window_nr.

 

The optional parameters which determines which GUI parameters will be considered. The default is "all". i.e. all GUI controls will be looked at. If which = "list" then a list of GUI parameter names should be supplied in the variable parameter. If which = "prefix" then only those GUI parameters with names which start with the prefix defined by variable, will be looked at.

 

This command is normally combined with setctrlvalues to provide a convenient way to get and set the state of a GUI window.

 

Example

 

From the NMRI/import macro with T2_3D data set selected:

 

a = getctrlvalues(1)

pr a

 

   a =

 

   nrType = "complex"

   3dtype = "T2"

   2dtype = "ky"

   1dtype = "kx"

   experiment = "Tecmag"

   matrixIn = "relaxT2"

   3dim = 8

   2dim = 256

   1dim = 256

   fileType = "Prospa"

   title = "Mandarin - T2 weighted"

 

 

lst = ["1dim","2dim","3dim"]

a = getctrlvalues(1,"list","list",lst)

pr a

 

   a =

 

   3dim = 8

   2dim = 256

   1dim = 256
