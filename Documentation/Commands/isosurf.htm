isosurf

 

Draw a 3D iso-surface from a 3D matrix.

 

Syntax

 

isosurf(matrix, level, colour, [range])  (eeee:vfvv)

 

matrix ... the 3D matrix containing the data to plot (MAT3D)

level .... the data value which defines the surface to draw (FLOAT)

colour ... the colour to use when drawing the surface (RGB 0-1)

range .... scale the data fit inside this box (VEC)

 

Description

 

This command renders a surface plot in the 3D window which corresponds to an iso-surface in the 3D matrix which has value level. The range vector can be used to scale and shift the data so that it is rendered at a convenient place in 3D space. In the following example the 3D matrix is shifted so it ranges from -50 to 50 along each axis.

 

Example

 

 

 

 

m3d = load("$appdir$\Example         Data\native\small.3d")

clear3d()

isosurf(m3d,30000,[.8,.2,0],

        [-50,50,-50,50,-50,50])
