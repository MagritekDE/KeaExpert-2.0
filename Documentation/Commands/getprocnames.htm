getprocnames

 

Returns a list of procedure names and argument lists for the specified macro file.

 

Syntax

 

(procedures,argumnts) = getprocnames(macro_path,macro_file)

 

macro_path....... the location of the macro in the filesystem (STR)

macro_name ...... the name of the macro (STR)

procedures ...... a list of procedures in the macro (LIST)

argument ........ a list of argument strings – one for each procedure (LIST)

 

Example

 

procedure(getprocnamesTest)

 

   path = "$appdir$\\Macros\\NMRI"

   file = "importData.mac"

 

   (procs,args) = getprocnames(path,file)

  

   for(k = 0 to size(procs)-1)

      print("\n$procs[k]$($args[k]$)")

   next(k)

 

endproc()
