addprefix

 

Add a prefix to list keys

 

Syntax

 

result = addprefix(list,prefix)     (ee:vs)

 

result ..... new list with prefixed entry keys. (LIST/PLIST)

list ....... list to be modified. (LIST/PLIST)

prefix ..... prefix to remove from list keys. (STR)

 

Description

 

This command scans through list adding the prefix to each list key copying the entries to result.

 

Example

 

  l1 = ["var1","var2","var3","test"]

  l2 = addprefix(l1,"p_")

pr l2

 

 l2 = { 

   p_var1

   p_var2

   p_var3

   p_test

 }
