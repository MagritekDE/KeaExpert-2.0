setlistvalues

 

Update all parameter value entries in one list with those in a second list

 

Syntax

 

result = setlistvalues(list1, list2)

 

list1 ...... the list to modify (LIST)

list2 ...... the list whose values will be applied to list1 (LIST)

result ..... the modified version of list1 (LIST)

 

Description

 

This allows the parameter values in one list to be applied to another.

 

Example

 

procedure(setlistvaluesTest)

 

  list1 = ["a = 23","b = \"Hi there!\"", "c = [1,2,3,4,5]"]

  list2 = ["c = \"A new value has been placed here\""]

  pr list1,list2

  newList = setlistvalues(list1,list2)

  pr newList

 

endproc()

 

 

 

Output

 

   list1 = {

       a = 23

       b = "Hi there!"

       c = [1,2,3,4,5]

   }

 

   list2 = {

       c = "A new value has been placed here"

   }

 

   newList = {

       a = 23

       b = "Hi there!"

       c = "A new value has been placed here"

   }
