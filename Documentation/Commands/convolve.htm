convolve

 

convolve a kernel with a matrix.

 

Syntax

 

R = convolve(A,K)

 

A .... input matrix or vector (MATRIX/VECTOR)

K .... kernel matrix or vector (MATRIX/VECTOR)

R .... convolution of matrix with kernel (MATRIX/VECTOR)

 

 

Description

 

This command calculates the following R = A Ä K i.e.

 

 

Restrictions

 

The kernel must be smaller than the input matrix in both dimensions.

The kernel dimensions must be odd.

 

Examples

 

The simplest useful application of this command is to perform a moving average of a vector.

 

R = A Ä [1,1,1]

 

In two dimensions many filters are possible such as the Sobel convolution which can be used to perform edge detection

 

q1 = [-1,-2,-1;0,0,0;1,2,1]

q2 = [-1,0,1;-2,0,2;-1,0,1]

g1 = convolve(m,q1)

g2 = convolve(m,q2)

image(sqrt(g1^2+g2^2))

 

 

With just one of the filters applied we obtain a relief image

 

 

Or a simple high pass filter:

 

convolve(m,[-1,-4,-1;-4,26,-4;-1,-4,-1])

image(n)

 

 

Note that the edges of the matrix or vector are unaffected by the convolution. (The unaffected region will depend on the size of the Kernel. For an n by n kernel the region will be (n-1)/2 pixels.
