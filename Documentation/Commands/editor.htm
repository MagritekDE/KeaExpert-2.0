editor

 

Generate a text editor window.

 

Syntax

 

obj = editor(nr, x, y, w, h)    (eeeee:dvvvv)

 

nr ............ control number ID     (INT)

x,y ........... top left coordinates of control in pixels (INT)

w,h ........... width and height of control in pixels      (INT)

obj ........... access to editor internal functions and parameters (optional)  

 

Description

 

This command produces a text editor window. This can be used for entering Prospa macros or editing any other text file.

 

The text editor object has a large number of underlying functions which can be used for modifying how it appears and behaves. These functions can be accessed using the returned object obj. obj is also accessible using the getobj command. The object functions are listed below.

Class function list for editor object

 

Note : All functions should be prefixed with the editor object name e.g. obj->copy()

 

filename ....................  name of file the text is associated with.

pathname ....................  path of file the text is associated with.

debug .......................  whether the editor is to be used as part of a debugger ("true"/"false").

modified ....................  whether the current text has been modified or not ("true"/"false").

readonlytext ................  whether the current text is readonly or not ("true"/"false").

showContextualMenu ..........  whether a contextual menu will be displayed in the editor ("true"/"false").

showSyntaxColoring ..........  whether a text will be colored according to its syntax menu will be displayed in the editor ("true"/"false").

showSyntaxDescription .......  whether a short description of the currently selected command will be displayed in the editor staus bar ("true"/"false").

wordwrap ....................  whether the text will automatically wrap at the end of the line ("true"/"false").
