panel

 

Generate a control panel.

 

Syntax

 

pnl = panel(objNr, x, y, w, h) (eeeee:ddddd)

 

objNr ......... control number ID  (INT)

x,y ........... top left coordinates of control in pixels (INT)

w,h ........... width and height of control in pixels(INT)

pnl ........... access to panel internal functions and parameters (optional) (OBJ)

 

Description

 

This command produces a panel object. This can be used as a container for a large group of controls which can then be scrolled to bring them into view.

 

A panel object has a number of underlying functions which can be used for modifying how it appears and behaves. These functions can be accessed using the returned object pnl, also accessible using the getobj command. The object functions are listed below.

 

 

Class hierarchy for panel

 

parent window

      │

panel control

      │

panel children    

 

Class function list for panel object

 

Note: All functions should be prefixed with the plot object name e.g. pnl->panelupdate

 

panelscrollset ...... set scroll position (0-100).

panelupdate ......... initialise the panel control after registering children

 

related functions

 

panelparent ......... any control can be a child of a panel – just use this command to register the parent object number

 

Note that panel children must have x,y coordinates relative to the top left hand corner of their panel parent, not to the window parent. Usually panel children are define in another macro and are imported to the panel on creation of the user interface. Alternatively they can be automatically generated as in the example given below.

 

See also

 

setpar, getpar

 

 

Example

 

This generates the panel shown at the top of this help page

 

 

# Display a panel with automatically generated child controls

procedure(panelTest)

 

   n = :windowdefinition()

   :initPanel(n)

   showwindow(n)

 

endproc()

 

# Define main controls

procedure(windowdefinition)

 

   n = window("Panel test", -1, -1, 230, 311)

 

      # Define all controls with basic parameters

      button(1, 20, 268, 51, 23, "OK",closewindow(0))

      button(2, 170, 270, 51, 23, "Close",closewindow(0))

      panel(3,10,10,200,250)

 

endproc(n)

 

# Add child controls to panel

 

procedure(initPanel,n)

 

   xoff = 80

   yoff = 20

   for(k = 0 to 20)

      statictext(k*2+4,xoff,k*25+yoff+3,"right","parameter $k$")

      setpar(n,k*2+4,"panelparent",3)

      textbox(k*2+5,xoff+10,k*25+yoff,60)

      setpar(n,k*2+5,"panelparent",3)

   next(k)

 

   setpar(n,3,"panelupdate","")

 

endproc()
