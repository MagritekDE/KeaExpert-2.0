mkparstruct

 

Make a structure from the passed list of variables or the local variables

 

Syntax

 

result = mkparlist([keys, [type]]) (ee)

 

keys ... an optional list of variable names (LIST)

type ... an optional type variable (STR)

 

Description

 

This command is similar to mkparlist and takes a list of local variables and packs them into a structure. The command name isn’t really correct, but reflects that it can be used as a drop-in replacement for mkparlist if a structure is required rather than a parameter list.

If the key list is not included, then all local variables are added except the default procedure variables nrArgs and parentCtrl. If type is included then only variables of that type will be included. Valid type names are:

 

all, string, float, complex, list matrix, cmatrix, struct, float32, float64.

 

 

Examples

 

procedure(test)

 

   a = 23

   b = 45

   c = 56

  

   s = mkparstruct(["a","b","c"])

  

   pr s

 

endproc()

 

Output

 

   s = {

          a = 23

          b = 45

          c = 56

   }

 

This example takes all the local variables but filters out the matrix type (variable c)

 

procedure(test)

 

   a = "test"

   b = 45

   c = [1:10]

  

   s = mkparstruct(varlist()),"matrix")

  

   pr s

 

endproc()

 

   s = {

          c = (matrix with 10*1 elements)

   }
