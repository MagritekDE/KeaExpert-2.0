plot2d

 

Generate a 2D plot window.

 

Syntax

 

OBJ plt = plot2d(INT objNr, INT x, INT y, INT w, INT h)

 

nr ............ control number ID

x,y ........... top left coordinates of control in pixels

w,h ........... width and height of control in pixels  

plt ........... access to plot internal functions and parameters (optional)

 

Description

 

This command produces a 2D plot object. This can be used for displaying image type data. The plot can be divided into multiple regions. Each region can display a single image.

 

A plot2d object has a large number of underlying functions which can be used for modifying how it appears and behaves. These functions can be accessed using the returned object plt (also accessible using the getobj command) or by using the current-plot commands (as provided with versions of Prospa before 3.0). The object functions are listed below along with their current-plot alternatives.

Class function list for plot object

 

Note: All functions should be prefixed with the plot object name e.g. plt->region(0)

 

OBJ rg = region(INT index) ...... get region using zero based region index.

OBJ rg = region(VEC x, VEC y) ... get region using zero based x, y indices.

OBJ rg = curplot() .............. get region which is currently selected.

OBJ win = parent() .............. get parent window object.

menu(STR location, STR menu) .... set plot contextual menu.

OBJ menu = menu(STR location) ... get plot contextual menu.

 

For generic object parameters which may be set/get see commands setpar/getpar.

 

Once the region object has been obtained, there are a number of functions which can be used to get and set the information within the region.

 

Class function list for plot-region object

 

Note: All functions should be prefixed with the region object name e.g. rg->ylabel("my label")

 

INT d = dim() .......................... get data dimension (1 or 2)

(MAT m, [VEC x VEC y]) = curdata() ..... get current region image data and axes range

OBJ plt = parent() ..................... get plot parent object

OBJ lbl = title() ...................... get the title object

OBJ title(STR text) .................... set the title text

OBJ lbl = xlabel()...................... get the x-label object

xlabel(STR text) ....................... set the x-label text

OBJ lbl = ylabel()...................... get the y-label object

ylabel(STR text) ....................... set the y-label text

OBJ ax = axes() ........................ get the axes object

Current-plot access

 

There are a number of commands which modify the current 1D or 2D plot. The current plot can be set by clicking on it with the mouse or by using the following commands

 

curplot(winNr, objNr, x_index, y_index)

 

or

 

curplot(winNr, objNr, region_number)

 

Current-plot commands equivalent to the above are listed below

 

plot() ......... draw (x,y) on current plot region.

getplotdata() .. extract current image data

trace() ........ set or get plot current trace parameters

xlabel() ....... set or get current plot region xlabel

ylabel() ....... set or get current plot region ylabel

title() ........ set or get current plot region title

axes() ......... set or get current plot region axes
