varlist

 

Returns a list of variable names of specified type and scope.

 

Syntax

 

list = varlist([type[, scope]])     (ee:s/f,s/f)

 

type .... a value representing variable types returned (INT/STR)

scope ... variable scope (see below) (INT/STR)

list .... returned list of variables (LIST)

 

The 'type' value can be one or a combination of the following:

 

String type

 

string

float

complex

list

matrix1d

cmatrix1d

matrix2d

cmatrix2d

matrix3d

cmatrix3d

matrix4d

cmatrix4d

all

Numeric type

 

1

2

4

8

16

32

64

128

256

512

1024

2048

4095 (default)

 

If the numeric version is used than more than one type can be returned by adding the numbers together – e.g. 7 would return all strings, floats and complex numbers.

 

The 'scope' argument can take one or a sum of the following values

 

String type

 

local

global

window

all

Numeric type

 

1

2

4

7 (default)

 

If varlist is run without any arguments all variables will be returned in the result list.

 

Description

 

This command returns a list of variable names of the specified type and scope.

 

Example

 

pr varlist("float")

         

     varlist(float) = {

gamma

pi

     }

 

pr varlist(5,6)      # all strings and complexes at either local or window scope

 

  varlist(5,6) = {

local_string

local_complex_1

local_complex_2

window_string

  }
