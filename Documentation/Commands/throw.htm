throw

 

Throw an exception

 

Syntax

 

throw(exception_string)

 

exception_string ...... the message returned by the throw command (STRING)

 

Description

 

Causes the interpreter to jump to the current catch block  (if it exists) where the throw message can be inspected.

 

Example

 

This example tests for divide by zero

 

procedure(throwTest)

 

   try

     

   for(k = -10 to 10)

  

      if(k == 0)

         throw("divide by zero")

      endif

      pr "k = $1/k$\n"

  

   next(k)

  

   catch

 

     print("Caught\n")

     err = lasterror()

     print(err->description)

  

   endtry

 

endproc()

 

 

Output

 

k = -0.1

k = -0.111111

k = -0.125

k = -0.142857

k = -0.166667

k = -0.2

k = -0.25

k = -0.333333

k = -0.5

k = -1

Caught

 

   err->description = divide by zero
