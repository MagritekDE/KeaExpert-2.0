showprocedures

 

Show all the procedures in the currently selected macro name in a popup blocking dialog,

 

Syntax

 

menuObj = showprocedures(mode)

 

mode ........ "editor"/"cli"  (STR)

 

Example

 

This is the result of selecting ctrl+M in the editor when the cursor is place somewhere on the function name (underlined)

 

      fft:1dDig(nameIn,nameOut,1dzf,1dfltr,1dft,magnitude,trunc(-digShift))
