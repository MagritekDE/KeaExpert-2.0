getmessage

 

Receive an event message from the Prospa application or a GUI window.

 

 Syntax

 

getmessage(objNr,commandList)

 

objNr ......... control number (INT)

commandList ... list of commands when message is received.

 

 Description

 

There are a variety of messages which are broadcast from Prospa to GUI windows and this command can be used to respond to them. In this way the macro interface can be modified to reflect the current Prospa state.  In addition it is possible to send message from one GUI window to another.

 

Typically commandList will simply consist of a call to a macro procedure and then within this procedure the message details will be extracted and acted on. To extract the message sent by Prospa just use getpar to obtain the source of the message and any additional details:

 

   (src,cmd) = getpar(winNr,objNr,"text")

 

Where winNr is window number for the current interface (typically 0 to get the current gui window number) and objNr is the control number defined in the getmessage command.

 

The returned variables src and cmd are strings having the following possible values:

 

src = "1D Plot"

cmd:

   LoadPlot .............. A new plot has been loaded using the Load Plot menu command.

   SelectPlot ............ A different plot in a multiplot display clicked on with mouse.

   Shift ................. A plot has been shifted (e.g. using the arrow keys)

   Scale ................. A plot has been scaled (e.g. using the arrow keys)

   Centre ................ A plot has been centred.

   Zoom .................. A region has been zoomed

   LastPlot .............. The last zoomed region has been selected.

   DisplayAll ............ The full data set has been displayed

   SelectRow ............. A row has been selected and drawn from a 2D plot

   SelectCol ............. A column has been selected and drawn from a 2D plot.

   DrawRectangle  ........ A rectangle has been draw over the plot.

   Autorange ............. The autorange parameter has been toggled using the menu.

 

 src = "2D Plot"

cmd:

   LoadImage ............. A new image has been loaded with the Load Plot menu.

   SelectImage ........... A different image in a multiplot display clicked on with mouse.

   Shift .................  A plot has been shifted (e.g. using the arrow keys)

   Scale .................  A plot has been scaled (e.g. using the arrow keys)

   Centre ................ A plot has been centred.

   Zoom .................. A region has been zoomed

   LastImage ............. The last zoomed region has been selected.

   DisplayAll ............ The full data set has been displayed

   DrawRectangle  ......... A rectangle has been draw over the plot.

   Autorange ............. The autorange parameter has been toggled using the menu.

 

A typical example of usage is to update GUI text boxes when a rectangle is drawn in the 1D or 2D plot window.

 

 Example

 

n = window("Test getmessage",10,10,100,100)

   statictext(1,10,10,"left","0")

   statictext(2,10,30,"left","0")

   getmessage(3,(src,cmd) = getpar(0,3,"text");

                if(src == "1D Plot" & cmd == "DrawRectangle");

                   (x,y,w,h) = getrect();

                   setpar(0,1,"text",w);

                   setpar(0,2,"text",h);

                endif;)

showwindow(n)
