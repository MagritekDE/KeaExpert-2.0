structappend

 

Appends the items in one structure to another.

 

Syntax

 

structappend(modified_structure, structure_to_append) (ve)

 

modified_structure ... the structure to be modified (STRUCTURE)

structure_to_append .. the structure to append

 

Description

 

Any entries in the structure to append which are not already in the modified structure will be added to the modified structure. Note that there is no return value, the modification happens in place.

 

Example

 

> s1 = struct(a = 23, b = "Hi there")

> s2 = struct(c = 123.45, d = [1:10])

> appendtostruct(s1,s2)

> pr s1

 

   s1 = {

          a = 23

          b = "Hi there"

          c = 123.449997

          d = (matrix with 10*1 elements)

   }
