histogram

 

Generate histogram data from a vector

 

Syntax

 

(x,y) = histogram(data, bins, range)

 

 

data ... data to be histogrammed (VECTOR)

bins ... number of bins in histogram (INT)

range .. range of x axis over which to bin (VECTOR)

 

Description

 

This commands counts up the number of entries in the data which fall in each bin and returns these in the y vector. The x vector covers range with size bins.

 

Example

 

This example checks the distribution of the noise command

 

Np = 1e4 # Number of points

m = noise(Np)

Nb = 200 # Number of bin

xlimit = 5

(x,y) = histogram(m,Nb,[-xlimit,xlimit])

yf = exp(-x^2/2) # Theoretical fit

it1 = integfunc(exp(-x^2/2),-xlimit,xlimit)

it2 = integfunc(exp(-x^2/2),-xlimit/Nb,xlimit/Nb)

fac = Np/(it1/it2) # Scale factor

# Plot histogram data

plot(x,y)

hold("on")

# Plot theoretical data

plot(x,fac*yf,"tracecolor",[0,255,0])

title("Histogram of noise data")

ylabel("amplitude")

hold("off")
