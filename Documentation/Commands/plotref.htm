plotpref

 

modify the plot display parameter preferences.

 

Syntax

 

plotpref(parameter, value, parameter, value, …)

 

parameter ... name of parameter [STR]

value ..... . new value for this parameter [VARIANT]

 

Possible parameters and their respective data types:

 

PARAMETER        VALUE

 

   axescolor ........... [R,G,B]

   axesfontcolor ....... [R,G,B]

   bkcolor ............. [R,G,B]

   bordercolor ......... [R,G,B]

   finegridcolor ....... [R,G,B]

   gridcolor ........... [R,G,B]

   labelcolor .......... [R,G,B]

   titlecolor .......... [R,G,B]

   minorticklength ..... integer

   majorticklength ..... integer

  

This command acts on the all 1D and 2D plots and any new ones which are generated.  Parameter and value pairs can be entered in any order.

 

Example

 

x = [0:1:99]

y = 200*noise(100) + x^2

plot(x,y)

grid("xgridstatus","on",

     "ygridstatus","on",

     "xfinegridstatus","on",

     "yfinegridstatus","on")

 

plotpref("axescolor",[0,0,255],

         "axesfontcolor",[0,0,255],

         "bkcolor",[0,50,0],

         "finegridcolor",[80,0,0],

         "gridcolor",[150,0,0],

         "labelcolor",[0,0,0],

         "titlecolor",[255,0,255],

         "minorticklength",5,

         "majorticklength",10)
