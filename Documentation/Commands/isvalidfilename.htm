isvalidfilename

 

Check if a filename conforms to the Windows requirements.

 

Syntax

 

result = isvalidfilename(file_name)

 

file_name ........ filename to check (STR)

result ........... whether the name is valid or not (0/1) (INT)

 

Description

 

Certain characters should not appear in a Windows file name. These are:

 

'\', '/', ':', '*', '?', '\"', '<',  '>', '|'

 

If these are found in the filename this command will return 0, otherwise 1.

 

 

 

Example

 

This example will return 0

 

pr isvalidfilename("Invalid:name")
