dec

 

Convert a hexadecimal number to signed decimal

 

Syntax

 

result = dec(hex_number, nr_of_bits)    (ee:dd)

 

hex_number .... The number to be converted (HEX)

nr_of_bits .... The number of bits used for convertion (INT)

 

Description

 

Converts a hexadecimal number into signed decimal using the specified number of bits to determine the corresponding range of decimal numbers

 

Example

 

> print dec(0xFF,16)

 

255

 

> print dec(0xFF,8)

 

-1

 

In the first case the range of numbers is -32768 to -32767 and 0xFF (or 0x00FF) is identified as 255 while in the second is it only -128 to 127 and 0xFF is identified as -1.
