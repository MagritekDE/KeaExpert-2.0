time

 

Return the current time in seconds.

 

Syntax

 

time(set_time)    (e:f)   # Set the current time

result = time()           # Return the current time

 

set_time .... current time to set (FLOAT)

result ...... current time returned (FLOAT)

 

The first option allows the time in seconds to be set, while the second option returns the time since the time was set. The resolution is 1 ms. By default the current time will be the time since Prospa was started. This command can be used to determine the time required to perform various functions within Prospa.

 

Example

 

time(0)

x = [0:1:100000]

for(n = 0 to size(x)-1)

   a = sqrt(x[n])

next(n)

t1 = time()

 

time(0)

for(n = 1 to 100)

   x = [0:1:100000]

   a = sqrt(x)

next(n)

t2 = time()

 

pr ("\n   Normal time = $t1$s\n   Vectorised time = $t2/100$s")

 

Normal time = 1.544s

Vectorised time = 0.00296997s

 

This example also highlights the advantage of vectorizing your code where possible!
