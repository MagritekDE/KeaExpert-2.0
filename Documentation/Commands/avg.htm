avg

 

Take the mean or average of a 1D or 2D matrix.

 

Syntax

 

result = avg(input)  (e:v/m)

 

result ....... average of input

input ........ any real or complex vector or matrix (EXP)

 

Description

 

Sums all the elements of the input matrix and return the result divided by the number of elements.

 

Example:

 

> x = [0:100]

> pr avg(x)

 

   avg(x) = 50
