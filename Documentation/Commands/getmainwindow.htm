getmainwindow

 

Returns the window number of the root Prospa window

 

Syntax

 

mainWinNr = getmainwindow()

 

mainWinNr....... the root window number, -1 if not found (INT)

 

Description

 

The first window generated by Prospa is the root window and when this window is closed the program will exit. You can force a window to be the main window using the setwindowpar option mainwindow.
