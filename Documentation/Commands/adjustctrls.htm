adjustctrls

 

Forces all controls in a window to resize and reposition as if the user had resized the window manually

 

Syntax

 

adjustctrls(winNr)

 

 

winNr ... the number of the window to adjust (0 means the current window).

 

Description

 

This command should be applied after moving a divider programmatically when that divider controls the position and or size of other controls.

 

Example

 

This example places two buttons in a window separate by a divider. As the divider is moved the buttons move. However if you press on the buttons the divider is moved programmatically. In this case we need to call adjustctrls to force the buttons to move as well.

 

 

 

 

procedure(adjustctrlsTest)

 

   n = :windowdefinition()

   showwindow(n)

 

endproc()

 

procedure(windowdefinition)

 

   n = window("AdjustCtrls test", -1, -1, 600, 110, "resizable")

 

      # Define all controls with basic parameters

      button(1, "ww/2-25",44, 50, 20, "Left",:adjustSize(1))

      button(2, "ww/2-25",44, 50, 20, "Right",:adjustSize(0))

      divider(3,"ww/2",0,4,"wh","vertical")

      setpar(0,1,"region",[-1,3,-3,-4])

      setpar(0,2,"region",[3,-2,-3,-4])

 

endproc(n)

 

procedure(adjustSize, par)

 

   if(par == 1)

     getobj(0)->draw("false")

     setpar(0,3,"x","ww/3")

     adjustctrls(0)

     getobj(0)->draw("true")

 

   else

     getobj(0)->draw("false")

     setpar(0,3,"x","ww*0.66")

     adjustctrls(0)

     getobj(0)->draw("true")

   endif

 

endproc()

 

Note the inclusion of the draw commands. This ensures that that all resizes or repositions occur simultaneously.
