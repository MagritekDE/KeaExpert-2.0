macadrs

 

Return the ethernet adapter media access control (MAC) address for the current computer.

 

Syntax

 

result = macadrs()

 

result ........ the MAC address (VEC)

 

Description

   

This command is used by the licensing software to determine the identifying address of the computer. Note that the MAC address is returned in decimal format. To display it in hex (more traditional) apply the hex command.

 

Example

 

This example opens this help page in internet explorer

 

 procedure(macadrsTest)

 

   print(hex(macadrs()))

 

 endproc()
