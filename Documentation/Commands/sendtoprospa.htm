sendtoprospa

 

Search for a running Prospa window with the specificed title and then send it a file to open. 

 

Syntax

 

result = sendtoprospa(windowTitle, pathToFile, fileToOpen)

 

windowTitle ...... title of Prospa window to search for (STR)

pathToFile ....... location of file to send to Prospa (STR)

fileToOpen ....... the file to send to Prospa  (STR)

result ........... returns 1 if Prospa window is found, 0 otherwise

 

Example

 

This example Prospa starts a second Prospa instance, running as a help viewer, and then requests that it opens the help file for the acos function. Note that the documentation viewer does not expect a full path to the file – just the parent folder in the documentation folder.

 

It then opens another version off Prospa – in this case the plotting calculator example and loads an FID plot file into it.

 

   procedure(sendtoprospaTest)

 

# Load a help file into Prospa help viewer

   exec("$appdir$\\Prospa.exe","\"$appdir$\\Macros\\coreMacros\\ProspaHelpViewer.pex\"")

   pause(1)

   sendtoprospa("Prospa Help Viewer Application","Commands","acos.htm")

 

# Load a 1D plot file into Prospa plotting calculator

   exec("$appdir$\\Prospa.exe","\"$appdir$\\Macros\\PEX_macros\\PlottingCalculator.pex\"")

   pause(1)

   sendtoprospa("Plotting calculator","$appdir$\\Example Data\\Plots","fid.pt1")

 

endproc()
