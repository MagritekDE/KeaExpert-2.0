printtofile

 

Send print output to a file rather than to the CLI.

 

Syntax

 

printtofile(filename) (c:s)

 

Description

 

Normally the pr command sends output to the CLI. However, using printtofile, you can redirect output to a file. This is useful for generating parameter lists.

 

Example

 

a = 10

b = 20

c = 30

 

printtofile("test.par")

  pr("a = $a$\n")

  pr("b = $b$\n")

  pr("c = $c$\n")

closeprint()

 

p = load("test.par")

pr p

 

  p = {

a = 10

   b = 20

     c = 30

        }

 

See also

 

closeprint, load, pr
