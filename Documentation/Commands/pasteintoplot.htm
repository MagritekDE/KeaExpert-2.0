pasteintoplot

 

This will copy a plot on the clipboard into the current plot without deleting the current plot.

 

Syntax

 

pasteintoplot([plot_region])

 

plot_region ........ The plot region you want to paste into. (Default is the current plot). (CLASS)

 

Description

 

Sometimes you want to overlay several plots or traces. The command allows you to do this.
