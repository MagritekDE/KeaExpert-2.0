func1d

 

Provides access to a number of 1D plot functions which are not accessible using normal class commands.

 

Syntax

 

func1d(command)  (c:s)

 

command ... Name of function to perform (STR)

 

Description

 

Some 1D plot commands are not accessible from the plot object. In this case this general function can be used instead. However, note that all these commands work on the current 1D plot within the calling window. You can change the current plot using the curplot command. If you need to access a 1D plot in another window it is important that you change the current window first using the curwin command.

 

Command names:

 

axes font ............... display axes font dialog and apply results to plot

border axes ............. display axes all around border

copy all plots .......... copy all plots to internal buffer

copy plot ............... copy plot to internal buffer

corner axes ............. display axes only on x and y axes of region

crossed axes ............ display axes on x and y axes of data set.

cut plot ................ delete plot copying to internal buffer

delete all plots ........ remove all plots

display data ............ display data cursor over plot

drag plot ............... all the plot to be dragged by user

enlarge horizontal ...... expand all plot traces horizontally

enlarge vertical ........ expand all plot traces vertically

full region ............. display complete data sets

hide borders ............ remove plot object borders

last region ............. display the last zoomed region

linear x scale .......... use a linear x axis scale

load plot ............... interactively load a plot file

log x scale ............. use a log x axis scale

move down ............... shift the current plot downwards

move left ............... shift the current plot leftwards

move right .............. shift the current plot rightwards

move up ................. shift the current plot upwards

multiplot 1*1 ........... set the multiplot to have the 1*1 plot layout

multiplot 1*2 ........... set the multiplot to have the 1*2 plot layout

multiplot 2*1 ........... set the multiplot to have the 2*1 plot layout

multiplot 2*2 ........... set the multiplot to have the 2*2 plot layout

multiplot m*n ........... set the multiplot dimensions via a dialog

overlapping plots ....... toggle plot overlapping (equivalent to toggling hold)

paste plot .............. paste copied plot(s) into (multi)plot

plot title .............. open a dialog to set the plot title

plot x,y labels ......... open a dialog to set the plot x and y labels

print plot .............. open a dialog to print the plot

reduce horizontal ....... reduce the horizontal scale of the plot

reduce vertical ......... reduce the vertical scale of the plot

remove all data ......... remove all data from the plot

remove current data ..... remove the latest plot data from the current plot

remove other subplots ... remove all plot data from the current plot

save emf ................ save the plot as an EMF (extended metafile)

save plot ............... save the plot

select region ........... enable region selection tool

show all plots .......... show all plots

show borders ............ show plot object borders

swap plots .............. replace current plot with copied plot

toggle border ........... display/undisplay plot borders

toggle fixed range ...... enable/disable fixed range

toggle imaginary ........ enable/disable display of imaginary component

toggle real ............. enable/disable display of real component

view one plot ........... hide other plots in multiplot

zoom region ............. zoom to current region
