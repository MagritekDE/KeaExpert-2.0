rms

 

Calculates the rms (root mean square) of a matrix.

 

Syntax

 

result = rms(source)

 

source ........ the data to be processed (MATRIX/DMATRIX/CMATRIX)

result ........ the root-mean-square value (FLOAT/DOUBLE)

 

Description

 

This command calculates the RMS value of an N by M matrix. If the source is a double precision matrix the output will be double precision. If it is a complex matrix the magnitude of the data is used.

 

Real matrix:  

 

Complex matrix:

 

 

Example

 

l2 = list(2,2)

l1 = listto1d(l2)

 

pr l2

 

   l2 =

       "item 0,0",   "item 1,0"

       "item 0,1",   "item 1,1"

 

pr l1

 

   l1  = {

       item 0,0

       item 1,0

       item 0,1

       item 1,1

   }
