findindex2

 

Find the array indices closest to the array values. Non-single valued functions are supported.

 

Syntax

 

(index1, index2, ...) = findindex(array, start, direction, value1, value2, ...)  (eeee...:vdsf...)

 

array ....... vector to search (VEC)

start ....... index to start searching from (INT)

direction ... which direction to search from: "left"/"right" (STR)

valuei ...... value in array for which the index is desired. (FLOAT)

indexi ...... array entry which contains a number closest to value1 (INT)

 

Description

 

This command is useful when you have a value in a 1D dataset and you would like to find the closest index when searching from a specified start point and in a certain direction. For example if you have data set which contains multiple peaks and you would like to find the linewidth of one of the peaks you would call the command twice with start set to the top of the peak and valuei to half the peak height. The first call would have direction equal to "left" and the second call direction equal to "right".

 

Example

 

This example generates 3 spectral lines each with a full-width half maximum linewidth of ~6 Hz.  It then uses the findindex commands to determine this linewidth.

 

N = 512

t = [0:N-1]/N

f = [0:N-1]-N/2

y = 10*exp(-i*2*pi*100*t-t*20)

y = y + 5*exp(-i*2*pi*50*t-t*20)

y = y + 15*exp(-i*2*pi*-50*t-t*20)

s = real(ft(y))

plot(f,s)

ind1 = findindex(f,-50) # find index of -50 Hz peak

height = s[ind1]

left =  findindex2(s,ind1,"left",height/2) # find index of left half height

right =  findindex2(s,ind1,"right",height/2) # find index of right half height

pr (right-left)*(f[1]-f[0]) # multiply width in points by the frequency resolution

 

(right-left)*(f[1]-f[0]) = 6
