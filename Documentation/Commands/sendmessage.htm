sendmessage

 

Send a message to a GUI window.

 

Syntax

 

sendmessage(source, message)      (ee:ss)

 

source ........ a string representing the source of the message (STR)

message ....... a string containing the message (STR)

  

Description

 

Occasionally it is useful to be able to send a message to a functioning GUI window. When combined with the getmessage command, sendmessage is able to provide “backdoor” communication with a functioning window. For example, you could use it to modify the text in several windows by adjusting a slider in another window, as shown in the following example.

 

Example

 

n1 = window("Master window",10,10,100,100)

  slider(1,10,10,100,20,"horiz",

         sendmessage("Master","$getpar(0,1,\"value\")$"))

showwindow(n1)

 

n2 = window("Slave window 1",10,210,100,100)

    statictext(1,10,10,"left","")

    getmessage(2,(src,cmd) = getpar(0,2,"text");

                 if(src == "Master");

                     setpar(0,1,"text",cmd);

                 endif;)

 

showwindow(n2)

 

n3 = window("Slave window 2",10,420,100,100)

    statictext(1,10,10,"left","")

    getmessage(2,(src,cmd) = getpar(0,2,"text");

                 if(src == "Master");

                     setpar(0,1,"text",cmd);

                 endif;)

 

showwindow(n3)

 

setfocus(n1)

 

 

The advantage of this method over directly writing to the controls using setpar() is that you don’t need to know the relevant control numbers or which windows contain them.
