searchdlls

 

Whether DLLs will be searched for a command

 

Syntax

 

searchdlls(mode)

 

mode ........ true/false (STR)

 

 Description

 

When deciding which command should be run in Prospa the following search path is used: internal, DLL, cached macros, file macros. If this command is run with mode set to "false" then the DLL search will be skipped. This allows certain DLL commands to be ignored or replaced with alternative macro based commands.

 

 

Example

 

Using default DLL:

 

   pr square(2)

 

      square(2) = 4

 

Not using DLL:

 

   searchdlls("false")

 

   pr square(2)

 

     Error: invalid or unknown expression (command = 'square' argument = '2')

     Line reads : square(2)

     Error occurred in line #1 of macro ''

 

Replacing with a macro:

 

   square = "procedure(square, n); print(\"\n\n   Result is $$n^2$$\n\"); endproc();"

   square(2)

 

   Result is 4
