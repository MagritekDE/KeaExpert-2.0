textmenu

 

Make a textmenu object.

 

Syntax

 

textmenu(nr,x,y,w,h,[commands]) (eeeeeec:dddddss)

 

nr ............ control number (INT)

x,y ........... top left coordinates of text in pixels (INT)

w ............. text box width in pixels (INT)

h ............. height of menu in pixels (INT)

command ....... command list to run when menu option selected or return is pressed.

 

Description

 

The textmenu is a combination of a text box and a pull-down menu. This command is useful when you need a control which has only certain defined states (such as powers of 2).

 

To fill the combobox with menu entries and select one of them you need to use the setpar command as well, as illustrated in the following example. Note that the h entry must be at least large enough to accommodate the complete menu. Calculate h using the following formula:

 

h = 20 + 20*number_of_menu_entries

 

Example

 

n = window("test",100,100,150,100)

    textmenu(1,20,20,80,100);

    setpar(n,1,"menu",["option 1","option 2","option 3","option 4"])

    setpar(n,1,"text","option 2") # Specify displayed value

    setpar(n,1,"index",2)  # Same as above but uses index in list

showwindow(n)

 

Note that the menu items are stored in an array which can contain numbers or strings. To extract the current value of the textmenu you need to use one of the getpar commands:

 

selection = getpar(1,1,"text")  # Item is returned as a string

selection = getpar(1,1,"value") # Item is evaluated first
