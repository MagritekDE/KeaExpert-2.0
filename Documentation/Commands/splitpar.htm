splitpar

 

This takes a parameter entry stored in a string and returns the parameter name and the parameter value as strings.

 

Syntax

 

(name,value) = splitpar(parameter_string)  (e:s)

 

 

name ... parameter name (STRING)

value .. parameter value (STRING)

 

 

Description

 

Splits a parameter string having the form

 

“name = value”

 

into name and value strings. Spaces around the equals sign are ignored. If a string of this form is not passed then an error will be returned. If value is a string with embedded spaces it should be quoted e.g.

 

str = "a = \"my value\""

 

Example

 

  > str = "a = 23"

  > (name,value) = splitpar(str)

> pr name,value

 

     name = a

 

   value = 23
