showdialog

 

Show a user defined dialog.

 

Syntax

 

result = showdialog(winNr)     (e:d)

 

winNr ... window to display as a dialog (INT)

result .. optional variable returned by command closedialog (VARIANT)

 

Description

 

This command is useful when you wish to force the user to enter some data before proceeding with the rest of the macro.

 

Example

 

This procedure displays a dialog into which you can enter some text. When you press OK it returns with this text. Try typing the following in the CLI.

 

> pr gettext("Enter text")

 

where gettext is defined as follows (this macro is supplied with Prospa)

 

procedure(gettext,prompt)          

 

# Define dialog

n = window(prompt, -1, -1, 249, 70)
setwindowpar(n,"type","dialog")

edittext(2, 9, 12, 110)

button(1, 130, 8, 47, 26, "OK",

result = getpar(0,2,"text");

closedialog(result);)

button(3, 184, 8, 47, 26, "Cancel",

closedialog("cancelled");)

setpar(n,1,"mode","default")

 

# Display dialog and wait for text to be returned

   text = showdialog(n)

 

endproc(text)
