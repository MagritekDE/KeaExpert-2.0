sortlist

 

Sort a list of strings in alphanumerical order.

 

Syntax

 

out = sortlist(in)   (e:l)

 

out ... sorted list. (LIST)

in .... unsorted list. (LIST)

 

Description

 

in is a list of strings.

 

out is a list sorted in alphanumeric order. Strings starting with numbers come first in numerical order, followed by strings in alphabetical order.

 

Note that case is ignored in this comparison.

 

Example

 

lst = getdirlist(".")

pr sortlist(lst)

 

   sortlist(lst) = {

       Classes

       DLLs

       General Information

       Internal

       Macros

       User Interface

       Writing macros

   }
