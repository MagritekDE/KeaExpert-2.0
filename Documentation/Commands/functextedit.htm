functextedit

 

Provides access to a number of text edit functions which are not accessible using normal commands.

 

Syntax

 

functextedit(command) (c:s)

 

command ... Name of function to perform (STR)

 

Description

 

These are commands specific to the textedit control (not to be confused with the larger more complex text editor). They all apply to the textedit control which has the current keyboard focus. These commands are typically used when defining a user interface and specifying the menu options for the textedit controls.

 

Command names:

 

clear all

copy

cut

paste

select all

undo
