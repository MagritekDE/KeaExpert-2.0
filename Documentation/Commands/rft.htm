rft

 

Calculate the real Fourier transform.

 

Syntax

 

r = rft(expression)

 

expression ... mathematical expression or variable (VEC/CVEC)

result ....... the real Fourier transform. (CVEC)

 

Description

 

This command calculates the real fast Fourier transform of the real part of expression and places the complex result in the vector r. expression must resolve into a real or complex 1D row matrix with a size equal to a power of 2.

 

 Example

 

t = [0:127]/128

y = cos(2*pi*20*x).*exp(-x*10)

y[0] = y[0]/2

plot(rft(y))

xlabel("f(Hz)")

 

Note that the resultant data set has half as many points as the original.
