statusbox

 

Generate a status-box window object.

 

Syntax

 

statusbox(nr) (e:d)

 

nr ... object number (INT)

 

Description

 

The status box is a strip at the bottom of a window into which information may be written. It is often used to report the status of a macro while it is running.

 

Example

       

n = window("test",100,100,200,100)

statusbox(1)

showwindow(n)

setwindowpar(n,"statusbox",1)

setpar(n,1,"text","window displayed")
