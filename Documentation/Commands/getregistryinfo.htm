getregistryinfo

 

Read a Windows registry entry

 

Syntax

 

result = getregistryinfo(hive, key [,value_name, value_type])

 

hive ........ which part of the registry to search (STR)

key ......... which key to check (STR)

value_name .. value to extract (default is "")

value_type .. only "REG_SZ" is supported.

 

Description

   

This command is useful if you wish to see if an application is a installed and where it is installed. It returns null if the regsitry entry cannot be found.

 

Example

 

This example opens this help page in internet explorer

 

 procedure(readRegistryTest)

 

   cmd = getregistryinfo("HKEY_CLASSES_ROOT","Applications\\iexplore.exe\\Shell\\open\\command")

   if(cmd != null)

      (file,arg) = scanstr(cmd,"\"%1\" %2")

       exec(file,"$appdir$\\Documentation\\commands\\getregistryinfo.htm")

   endif

 

endproc()

 

 

Note that this example should be pasted into a Prospa text editor before running.
