jn

 

Calculate the nth-order Bessel function

 

Syntax

 

result = jn(order,expression) (e:f/m)

 

order ........ Bessel function order (INT)

expression ... argument to process (VARIANT)

 

Description

 

Calculates the nth-order Bessel function.

 

Example

 

x = [0:0.1:10]

hold "on"

c=[255,0,0;255,255,0;0,255,0;0,255,255]

for k = 0 to 3

   plot(jn(k,x),"tracecolor",c[~,k])

next k

hold "off"
