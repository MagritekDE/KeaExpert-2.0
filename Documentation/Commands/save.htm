save

 

Save a Prospa data file or plot file from memory to disk.

 

Syntax

 

save("fileName.1d",variable) # Save vector to file

save("fileName.2d",variable) # Save 2D matrix to file

save("fileName.3d",variable) # Save 3D matrix to file

save("fileName.4d",variable) # Save 4D matrix to file

save("fileName.pt1")         # Save current 1D window to file

save("fileName.pt2")         # Save current 2D window to file

save("fileName.*")           # Save current editor contents to a text file

save("fileName.pt1",x,y)     # Save 1D data in region (x,y)

save("fileName.xxx","1d")    # Save 1D plot to an image file xxx = emf/png/jpg/gif/tif/bmp.

save("fileName.xxx","2d")    # Save 2D plot to an image file xxx = emf/png/jpg/gif/tif/bmp.

save("fileName.xxx","3d")    # Save 3D plot to an image file xxx = emf/png/jpg/gif/tif/bmp.

 

Description

 

This command has several forms depending on the file name extension and the number of variables. The "1d", "2d", "3d" and "4d" extensions save just the data in "variable" (along with minimal dimensional information), while the "pt1" and "pt2" extensions save the contents of the complete 1D or 2D window (i.e. all subplots) with all formating information. All other extensions (e.g. "mac" or "txt") save the contents of the current edit window to a text file.

 

The last 3 options allow plots to be saved to and image file.  In this case the filename should have the appropriate extension.

 

Example

 

A = noise(100,100)

save("test.2d",A)     # Save a fabricated 2D data set

 

B = load("test.2d") # Load the 2D data set and display it

image(B)
