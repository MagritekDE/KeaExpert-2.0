isproc

 

Check if a procedure exists

 

Syntax

 

result = isproc(macropath, macroname, procedure)

 

macropath ... The path to the macro containing the procedure (STR)

macroname ... The macro containing the procedure (STR)

procedure ... The procedure name to search for (STR)

result ...... Returns 1 if found, 0 if not found.

  

Description

 

If the presence of a procedure is not definite you can use a try-catch statement however this requires searching for the procedure which if not present may take some time.  Because this command has a definite macro path and name it should be quicker. Note that this command will search the procedure cache first before checking the file system.

 

Example

 

Using try-catch

 

try

   testMacro:testProc()

catch

   pr("Macro not found\n")

endtry

 

Using isproc

 

if(isproc("macroPath","testMacro","testProc"))

   testMacro:testProc()

else

   pr("Macro not found\n")

endif
