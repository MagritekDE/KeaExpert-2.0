execandwait

 

Run a program and wait for it to complete

 

Syntax

 

execandwait(str filename, str arguments)

 

 

filename ...... the name of the application to run (full path or relative)

arguments ..... a space delimited set of quoted arguments to pass to the application

 

Description

 

Unlike the exec command which returns immediately this command will wait until the started application exits.

 

Example

 

This simple example starts up Prospa and waits for it to exit:

 

cd(appdir)

execandwait("prospa.exe")

pr "Finished\n"

 

where as exec returns immediately:

 

cd(appdir)

exec("prospa.exe")

pr "Finished\n"

 

 

Note that you have to first move to the application folder before running this command unless you supply a full absolute path or one relative to the current directory, otherwise it will return with a error.

 

To run with arguments surround each argument with escaped quotes e.g. to run chrome and load one of these help files:

 

path = "$appdir$\\Documentation\\Macros\\1D\\multiPlot1D.htm"

execandwait("C:\\Program Files (x86)\\Google\\Chrome\\Application\\chrome.exe","\"$path$\"")

pr "Finished\n"

 

Do not attempt to call Prospa using multiple arguments from within Prospa as the program will call itself and will hang waiting for it to exit (use the exec command in this case).
