cmap3d

 

Draw a colour-scale on the 3D plot.

 

Syntax

  

cmap3d(scale,[min, max],font_color, font_size)      (eeeee:mffmf)

 

scale ........... RGB matrix defining colours in scale (MAT)

min, max ........ data value range represent by colorscale  (FLOAT)

font_color ...... RGB color for font (0-1) (VEC)

font_size ....... size of font in current display units (FLOAT)

 

Description

 

Draw a colour-scale on the 3D plot.

 

scale is a matrix with 3 columns representing red, blue and green values (0-1) with n rows – the more rows the finer the steps in the colour scale.

 

Example

 

The following macro code generates the red-blue colour scale shown on the right. The left scale has N = 101 the right N = 11

 

N = 101

cs = matrix(3,N)

cs[0,~] = linvec(0,1,N)'

cs[1,~] = 0

cs[2,~] = linvec(1,0,N)'

cs[0,N-1] = 0

cmap3d(cs,[0,1],[1,1,0])

showcmap3d("yes")

 

 

Note that N should contain an extra entry at the end which specifies whether the colour-scale is normal or plus-minus. A normal colour-scale has 0 in the red N-1th position while a plus-minus colour-scale has a 1 in the red N-1th position.

 

To make a new colour-scale available add it to the global variable colorscales e.g.

 

colorscales = colorscales + "cs"

 

The new colour-scale "cs" will now appear in the 2D colors menu.

 

The 3D colorscale is currently only used by the 2D surface plot.
