exitthread

 

Stop a procedure running in a separate thread.

 

Syntax

 

exitthread(id)

 

id .......... thread id – needed for stopping thread. (INT)

 

Description

 

Once a thread has been started it will return an identifying number which can be used to stop it at later time (e.g. when the parent window is closed). If a thread is not exited but allowed to continue after a window has been closed then there is the chance of a Prospa crash.
