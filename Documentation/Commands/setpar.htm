setpar

 

Set a parameter in a user-defined control.

 

Syntax

 

setpar(winNr,ctrlNr,parameter,value) (eeee:ddsv)

 

winNr ....... parent window of control. (INT)

ctrlNr ...... number of this control. (INT)

parameter ... name of parameter to modify (STR)

value ....... new value for this parameter (VARIANT)

  

ctrlNr is the number specified when the control was made, while parameter is a string which is dependent on the type of object being modified. value is the updated parameter value.

 

Description

 

Each user-defined object belongs to a window specified by a number winNr. If winNr is zero then the current GUI window is used.

 

For a list of parameters, please see the documentation for generic window object parameters, along with the documentation for the specific type of window object of interest.

 

 

Generic options

 

Applies to all controls:

 

Parameter Type

 

 

"active"

"true"/"yes"/"on" – Set window active. Any other argument sets window inactive.

 

"ctrlid"

text – Set the control id.

 

"ctrlnr"

int – Set control number.

 

"draw"

"true"/"yes"/"on" - Enable changes in the window to be redrawn. Any other argument prevents redrawing.

 

"enable"

"true" - Enable control.

"false" - Disable control.

 

"event"

string – Set the callback event name for the control.

 

"eventactive"

"true"/"yes"/"on" – Set event active. Any other argument sets event inactive.

 

"focus"

"" - Set keyboard focus to control.

 

"label"

text - Set the label for the window.                                         

 

"menubar"

Row vector - Set the menubar and accelerator table for the window.

 

"name"

text – Set the control name.

 

"procedure"

string – Set the callback procedure for the control.

 

"range"

Two-element array – Set expected data range as [lower_extreme, upper_extreme]

 

"readonly"

"true"/"yes"/"on" – Set window read-only. Any other argument enables writing.

 

"region"

1x4 matrix – Set current region for object.

 

"statusbox"

int – Set the status box for the control.

 

"tab_number"

int – Set position of control in tab list.

 

"tabparent"

Two-element array – Set current tab parent and tab page.

 

"tag"

string – Set the tag for the control.

 

"toolbar"

int – Set the toolbar for the control.

 

"type"

"integer", "float", "double", "string" – Set expected data type.

 

"visible"

"true" - Show control.

"false" - Hide control.

 

"width/height"

int - Set width or height of control.

 

"x/y"

value - Set location of control.
