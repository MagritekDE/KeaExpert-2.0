setfont

 

Set some of the Prospa fonts

 

Syntax

 

result = setfont(prospa_font_name, system_font_name, font_size)    (eee:ssd)

 

prospa_font_name..... the name of the Prospa font to change (see below) (STR)

system_font_name..... the name of a supplied system font. (STR)

font_size ........... the size of the font in points. (INT)

 

Description

 

This command replaces the specified font when the relevant window is regenerated (i.e. it won’t change the font in existing windows).

 

Prospa fonts which can be changed with this command:

 

“Control font” .... the font used in controls (static text, text boxes etc).

“Edit font” ....... the font used in the editor window.

“CLI font” ........ the font used in the command line interface.

 

The default fonts are:

 

Control font : MS Sans Serif 8

Edit font : Courier New 10

CLI font : Courier New 10  

 

The default startup fonts can be changed by placing setfont command in the preferences Startup folder e.g. in the macro nmr.mac.

 

Example

 

A not very practical example:

 

  setfont("Control font","Symbol",8)
