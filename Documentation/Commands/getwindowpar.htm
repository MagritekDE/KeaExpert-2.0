getwindowpar

 

Get parameters for the specified window.

 

Syntax

 

result = getwindowpar(winNumber,parameterName)  (ee:ds)

 

winNumber ...... the window to extract parameters from (INT)

parameterName .. the parameter to extract (STR)

      

Description

   

This command returns a parameter for the window specified by winNumber. The parameter is specified by the string parameterName.

 

Possible values for parameterName are:

 

bgcolor

 

bordersize

 

ctrllist

 

dimensions

 

dimensionstxt

 

gridspacing

 

height

 

macroname

 

macropath

 

mainwindow

 

menubar

 

name

 

permanent

 

position

 

resizable

 

showgrid

 

sizelimits

 

snaptogrid

 

title

 

titlebarheight

 

titleupdate

 

visible

 

width

 

winnr

 

x

 

y

 

RGB colour of the window background

 

Border width in pixels (integer)

 

List of the numbers of the controls in the window (list of integers)

 

[x,y,w,h] as an array

 

[x,y,w,h] as a list.

 

Pixel width of grid spacing (integer)

 

Height of window in pixels (integer)

 

Name of macro used to define window (string)

 

Directory of macro used to define window (string)

 

Whether the window is the main Prospa window ("true"/"false") (string)

 

List of menu object numbers for this window (list of integers)

 

User definable tag for the window (string)

  

Whether the window can be deleted ("true"/"false") (string)

 

[x,y] as a vector

 

Whether the window can be resized ("true"/"false") (string)

 

Whether the grid is currently visible ("true"/"false") (string)

 

The size limits for this window (vector)

 

Whether grid snapping is currently enabled ("true"/"false") (string)

 

Window title (string)

 

Height of title bar in pixels

 

Whether the title may be updated

 

Whether the window is visible ("true"/"false") (string)

 

Width of window in pixels (integer)

 

The number of the window (integer)

 

screen coordinate of window’s left edge (integer)

 

screen coordinate of window’s top edge (integer)

 

If winNr is zero then the current window will be used.

 

Example

 

procedure(getwindowpar_test)

 

   n = :windowdefinition()

   showobjects(n)

 

   m = getwindowpar(n,"ctrllist")

   for(k = 0 to size(m)-1)

     pr ("\n  Object number = $m[k]$")

   next

 

endproc()

 

procedure(windowdefinition)

 

   n = window("test", 100, 100, 200, 200)

       button(7, 26, 24, 40, 20, "button")

       slider(6, 34, 61, 100, 20, "horizontal")

       radiobuttons(2, 39, 105, 20, "horizontal", "yes,no", "yes")

 

endproc(n)

 

When run, this macro produces the following output:

 

>

  Object number = 7

  Object number = 6

  Object number = 2

> 

 

With this information it is then possible to access each control in a window regardless of the numbers used to identify each control.
