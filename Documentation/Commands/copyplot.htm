copyplot

 

Copy the current or specified plot.

 

Syntax

 

copyplot([plot_region]) (e:c)

 

plot_region ... an optional plot region parameter. The default actions is to use the current plot.

 

This command copies the specified or current 1D or 2D plot to internal storage. This can later be pasted into another plot.

 

Examples

 

multiplot("1d",2,1) # Generate a 2 by 1 plot

curplot("1d",1,1)   # Specify the current plot

plot(noise(100))    # Generate some data        

copyplot()          # Copy the current plot

curplot("1d",2,1)   # Change the current plot

pause(1)            # Wait 1 second

pasteplot()         # Paste the plot into a different plot region

___________________________________

 

pr1 = getobj(4,1)->subplot(1,1)   # Get standard 2D plot region

pr1->plot(noise(100,100))         # Plot some noise

n = plot2DWin()                   # Make a new 2D plot

pr2 = getobj(n,1)->subplot(1,1)   # Get the new 2D plot region

copyplot(pr1)                     # Copy the first plot

pause(1)                          # Wait 1 second

pasteplot(pr2)                    # Paste into the new region
