plotscale

 

Scales a plot when it is being sent to a file or to the clipboard.

 

Syntax

 

plotscale(scale_factor)

 

 

scale_factor ... factor by which the plot is to be scaled (default 1) FLOAT

 

Description

 

This command allows copied or saved plots to have a higher (or lower) resolution than the displayed version. Note that this command is also accessible from the user interface preference dialog (Prospa menu; Files->Preference->Copy Plot)

 

Example

 

procedure(plotScaleTest)

 

  x = [0:0.001:1]*2*pi*4

  plot(cos(x).*exp(-x/10),sin(x).*exp(-x/10))

    plotscale(1) # Adjust argument before copying

 

endproc()
