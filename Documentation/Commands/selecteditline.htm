selecteditline

 

Selects the specified line number in the current editor. This will also cause the editor to scroll to make this line visible

 

Syntax

 

selecteditline(lineNumber)

 

lineNumber ...... the zero based line to select (INTEGER)

 

Example

 

If copied into a Prospa editor, this example, if saved and run, will highlight each of the :test() lines for 0.2 seconds before moving onto the next one

 

procedure(selecteditlineTest)

 

  :test()

  :test()

  :test()

  :test()

  :test()

  :test()

  :test()

  :test()

 

endproc()

 

procedure(test)

 

   selecteditline(getcurrentline("parent"))

   pause(0.2)

 

endproc()
