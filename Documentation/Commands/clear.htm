clear

 

Clear a plot or a command entry window.

 

Syntax

 

clear("cli | edit | plot") (e:s)

 

Example

 

clear("cli")    # clears the command line interface window

clear("edit")   # clears the current edit window

clear("plot")   # clears the current plot window

 

See also

 

clear1d, clear2d, clear3d
