sortrows

 

Sort the rows of a real matrix based on one column

 

Syntax

 

result = sortrows(matrix, column, mode)

 

matrix ...... matrix to be sorted (REAL MATRIX)

column ...... column to sort on (0 based) (INT)

mode ........ "ascending"/"descending"  (STR)

result ...... sorted matrix

 

Example

 

This example sorts a randomly generated matrix by its first column (shown in bold italics in the output)

 

procedure(sortrowsTest)

 

   m = noise(4,4)

   pr m

   mSorted = sortrows(m,1,"ascending")

   pr mSorted

   mSorted = sortrows(m,1,"descending")

   pr mSorted

 

endproc()
