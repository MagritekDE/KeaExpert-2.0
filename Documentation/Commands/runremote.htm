runremote

 

Run a macro on a second instance of Prospa.

 

Syntax

 

response = runremote(macro, winID)

 

macro ...... the macro to run with arguments (STR)

winID ...... the window id of the other Prospa instance (INT)

response ... the response (STR)

 

Description

 

This command allows macros to be run on other instances of Prospa which exist. These are identified by the winID which can be enumerated using the command prospawinnr which returns all current Prospa window handles (apart from the parent of this command).

 

Example

 

Here we run a macro called testRemote on all external Prospa instances.

 

    w = findprospawin()

    for(k = 0 to size(w)-1)

       r = runremote("testRemote(w[k])",w[k])

       print("$r$\n")

  next(k)

 

Here is testRemote which just returns the ID in a string. This macro should be in the Prospa search path.

 

    procedure(testRemote, ID)

 

 

    endproc("ID = $ID$")
