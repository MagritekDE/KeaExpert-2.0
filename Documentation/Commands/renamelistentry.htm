renamelistentry

 

Rename a parameter list entry

 

Syntax

 

result = renamelistentry(list, oldname, newname)  (ee:lss)

 

 

result ... modified list (LIST)

list ..... source list (LIST)

oldname ... old name of a parameter key in the list (STR)

newname ... new name for the parameter key (STR)

 

Description

 

Renames an entry within a parameter list (list) depending on the second and third parameters.

 

Example

 

a = ["a = 1", "b = 2", "c = 3", "d = 4"]

 

pr renamelistentry(a, "c", "q")

 

 renamelistentry(a, "c","q") = {

 

   a = 1

   b = 2

   q = 3

   d = 4

 }
