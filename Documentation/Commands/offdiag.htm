offdiag

 

Zeroes the diagonal values of a square 2D matrix (i.e. returns the off diagonal elements only)

 

Syntax

 

mOut = offdiag(mIn)

 

mIn........ the matrix to modifiy (SQUARE REAL MATRIX)

mOut....... the resultant matrix (SQUARE REAL MATRIX)

 

 

Example

 

   procedure(offdiagTest)

 

   mIn = noise(4,4)

   print(mIn)

   mOut = offdiag(mIn)

   print(mOut)

 

endproc()
