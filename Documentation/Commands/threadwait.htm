threadwait

 

Wait for the specified thread to finish

 

Syntax

 

threadwait(threadID)

 

threadID ...... the thread identifier (as returned by the thread command (INTEGER)

 

Description

 

After a thread has started you can enter this command to wait for the thread to finish. While waiting the program will still respond to window resizing and moving events.

 

Example

 

This example counts up to 1000 in the CLI in a separate thread. When the thread is finished it prints "finished" to the CLI.

 

procedure(threadWaitTest)

 

   id = thread(":counter")

   threadwait(id)

   print("finished!\n")

 

endproc()

 

procedure(counter)

 

 for(k = 0 to 1e3)

    print(k)

  next(k)

 

endproc()
