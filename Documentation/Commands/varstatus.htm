varstatus

 

Set a global variable’s status (hidden or visible).

 

Syntax

 

varstatus(name [, visibility [, readstatus [, lifetime]]])    (eeee:ssss)

 

or, to return the status of a variable,

 

(visibility,readstatus,lifetime) = varstatus(name)

 

name ........... variable name (STR)

visibility...... "hidden" or "visible" (default) (STR)

readstatus ..... "readonly" or "readwrite" (default) (STR)

lifetime ....... "permanent" or "transient" (default) (STR)

 

To display hidden variables type pr hidden.

 

Description

 

If a variable is hidden then it will not appear when you type pr glo. This is useful if the variable is to be used by a number of different macros, or if it has a constant value.

 

Example

 

Visible status

 

a = "my variable"

pr glo

 

###### Global Variables ######

 

   a = "my variable"

   gamma = 2.675e+008

   pi = 3.14159

 

###### End global variables ######

 

varstatus("a","hidden")

pr glo

 

###### Global Variables ######

 

   gamma = 2.675e+008

   pi = 3.14159

 

###### End global variables ######

 

varstatus("a","visible")

pr glo

 

###### Global Variables ######

 

   a = "my variable"

   gamma = 2.675e+008

   pi = 3.14159

 

###### End global variables ######

 

If a variable is readonly you can't reassign it.

 

Read status

 

a = 23

a = "Hi there"

varstatus("a","visible","readonly")

a = 45

 

   Error: variable 'a' is read only

   Line reads : a = 45

   Error occurred in command #1 of macro 'CLI'

 

If a variable is permanent then you can’t delete it.

 

Lifetime status

 

a = 23

rmvar(a)

a = "test"

varstatus("a","visible","readonly","permanent")

rmvar(a)

 

   Error: Can't delete permanent variable 'a'

   Line reads : rmvar(a)

   Error occurred in command #1 of macro 'CLI'
