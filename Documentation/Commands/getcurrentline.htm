getcurrentline

 

Get the current line number in the executing macro.

 

Syntax

 

lineNumber = getcurrentline(location)

 

location ..... either the "current" macro or the "parent" macro. (STR)

lineNumber.... which line we are on (zero based). (INTEGER)

 

Description

 

Returns the zero based line number in the currently executing macro or the line number in the parent macro.

 

Example

 

0  procedure(getcurrentLineTest)

1

2    :testit()

3

4  endproc()

5

6  procedure(testit)

7

8     print(getcurrentline("parent"))

9

10 endproc()

 

With the argument equal to "parent" the print output is 2 while with "current" as the print output is 8.
