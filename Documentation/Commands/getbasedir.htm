getbasedir

 

Get the base directory in a path name.

 

Syntax

 

dir = getbasedir(path)

 

dir .... the base or last directory in  a pathname (STR)

path ... a full or partial pathname. (STR)

 

Description

 

This command returns the base directory in a path name.

 

Note that the last directory must not end in a '\' otherwise the command will return an empty string.

 

Example

 

pr workdir

 

  workdir = C:\Program Files\Prospa\Example Data\Import

 

pr getbasedir(workdir)

 

  getbasedir(workdir) = Import
