submatrix

 

Extract a 2D, 3D or 4D submatrix from a larger matrix, or a substring from a string.

 

Syntax

 

result = submatrix(matrix,x1,x2,[y1,y2[,z1,z2[,q1,q2]])  (eeeeeeeee:m/cm,llllllll)

 

matrix ... source matrix or string. (MAT/STR)

x1,x2 .... range of 1st dimension values (INT)

y1,y2 .... range of 2nd dimension values (INT)

z1,z2 .... range of 3rd dimension values (INT)

q1,q2 .... range of 4th dimension values (INT)

result ... submatrix extracted (MAT/STR)

 

For a column vector you will need to set the x values to zero and specify the range in the y values (since in Prospa vectors are also 2D matrices).

 

Description

 

For a 1D matrix the inclusive data range indexed by x1 to x2 in matrix is extracted and stored in the smaller matrix result. In the case of a 2D matrix the rectangular region inclusively bounded by (x1,y1) and (x2,y2) with x2>x1 and y2>y1 is extracted and stored in result. For 3D matrices the parameters z1 and z2 are used. q1 and q2 are used for 4D matrices.

 

 Example

 

a = noise(100,100)

b = submatrix(a,30,79,30,79)

pr(size(b))

image(b)

 

Note that submatrix also works for strings:

  

a = "this is a test string"

pr submatrix(a,5,13)
