parentctrl

 

Returns the control which called the current procedure.

 

Syntax

 

parentctrl()

  

Description

 

When the callback procedure for a control is executed it is often useful to have access to the control itself so that information about that control can be extracted. For example, if return is pressed in a textbox, the current value of the textbox is probably going to be used in the callback procedure. There are a number of ways of extracting this information.

 

1.  Use the object/control number and getpar(winNr, objNr, "parameter") or getobj(winNr,objNr)->parameter

2.  Use a control-object variable defined using setpar(winNr,objNr,"ctrlID","name")

3.  Use a value-object variable defined using setpar(winNr,objNr,"valueID","name")

4.  Use parentctrl to return the control-object variable.

 

Option 1 requires that you know the object number for the control – this may vary if the interface is redesigned.

 

Option 2 requires the addition of a control id name when defining the control, and the assignment of this control to a window variable with the command assignctrls.

 

Option 3 requires the addition of a value id name when defining the control and the extraction of this information using the command getctrlvalues whenever the value has changed.

 

Option 4 can be used locally and does not require any other commands – however it is limited to one control.

 

Example

 

This example displays a window from which you can select a color.

 

 

 

procedure(parentctrl_test)

 

   n = :windowdefinition()

   showwindow(n)

 

endproc()

 

procedure(windowdefinition)

 

   n = window("parentctrl test", -1, -1, 220, 62, "resizable")

       button(1, 140, 15, 50, 30, "Close",closewindow(0))

       colorbox(2, 80, 20, 22, 22, [255,0,0],:process();)

       statictext(3, 70, 24, "right", "Select color")

 

endproc(n)

 

procedure(process)

 

   ctrl = parentctrl()  # Get colorbox control

   oldcol = ctrl->color # Get current color

   newcol = getcolor(oldcol) # Get a new color

   ctrl->color(newcol) # Set the new colorbox color

 

endproc()
