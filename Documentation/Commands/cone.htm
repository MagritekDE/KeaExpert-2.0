cone

 

Draws a cone to the 3D plot

 

Syntax

 

cone(vec base, vec top, float radius, rgb color)

 

 

base ...... the x,y,z coordinate of the center of the base of the code.

top ....... the x,y,z coordinate of the top of the code.

radius .... the x,y,z coordinate of the top of the code.

Color ..... the r,g,b color of the cone (each number is from 0 to 1)

 

Description

 

This command can be used alone or as part of a more complex diagram

 

Example

 

This simple example draws a cone

 

cone([0,0,0],[0,30,0],10,[1,0,0])

 

 

 

 

This next example draws an arrow by combining a cone and a cylinder

 

cone([0,0,0],[0,20,0],10,[1,0,0])

cylinder([0,-30,0],[0,0,0],3,[1,0,0])
