while

 

Control statement for implementing while loop

 

Syntax

 

while(expression)

   ...

   [exitwhile]

   ...

endwhile

 

Description

 

While expression is true this command will continually execute the commands between the while and endwhile statements.

 

Example

 

a = 10

while(a > 0)

  pr("a = $a$\n")

  a = a -1

endwhile

 

If you wish to exit from the while loop immediately then call the exitwhile statement.

 

a = 10

while(a > 0)

  pr("a = $a$\n")

  if( a == 20)

     exitwhile

  endif

  a = a + 1

endwhile
