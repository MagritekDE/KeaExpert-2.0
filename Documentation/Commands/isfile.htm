isfile

 

Check a filename for existence.

 

Syntax

 

result = isfile(filename) (e:s)

 

filename ... name of file to verify (STR)

result ..... result of check (INT)

 

Description

 

Checks to see whether the file filename exists in the current folder.

 

If filename is found, result = 1 (true) otherwise result = 0 (false).

 

Example

 

ls

 

     ls = {

       helplist.lst

       noisy.dat

       outfile.txt

       test.dat

}

 

isfile("noisy.dat")

 

     isfile("noisy.dat") = 1

 

isfile("notafile.dat")

 

     isfile("notafile.dat") = 0
