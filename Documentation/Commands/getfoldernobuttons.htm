getfoldernobuttons

 

Loads the file selection dialog without special Prospa buttons

 

Syntax

 

(foldername, relative) = getfolder([cur_folder[,title[,top_folder[,allow_relative_path]]]])  (eeee:ssss)

 

foldername ....... the selected folder name, or “cancel” if the dialog is cancelled (STR)

relative ......... "relative" or "absolute" depending on selected path type (STR)

cur_folder ....... the starting folder for the dialog (STR)

title ............ the title to appear at the top of the dialog (STR)

top_folder ....... topmost navigable and selectable folder in dialog (STR)

allow_rel_path ... "true" or "false", defaults "false"; allow selection of relative or absolute path result (STR)

 

Description

 

Presents a dialog from which you can select a folder/directory. Does not modify the current working folder. Note that the returned folder name does not contain a trailing backslash.

 

If the cancel button is pressed then the foldername will be "cancel"

 

Example

 

getfoldernobuttons("C:", "Select a data folder", "C:", "true")
