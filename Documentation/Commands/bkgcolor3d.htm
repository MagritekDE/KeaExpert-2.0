bkgcolor3d

 

Sets the background colour for the 3D plot.

 

Syntax

 

bkgcolor3d(colour)

 

colour ... RGB colour [VEC]

 

Description

 

Sets the background colour for the current 3D plot. Note that RGB colours for this command are in the range 0 to 255.

 

Example

 

clear3d()

axes3d([0,0,0],5,[1,0,0])

bkgcolor3d([0,80,0])
