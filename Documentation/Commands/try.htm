try

 

Try running a block of code anticipated to raise an otherwise-fatal error, instead catching and handling any such error.

 

Syntax

 

try

... # code to test

...

catch

... # jump here on error

...

endtry

... # jump here if no error

 

Description

 

Blocks of code that are anticipated to encounter errors may be marked with try/catch commands. Fatal errors between try and catch do not terminate their containing macro; instead, the error is caught and handled by the block of code marked by catch.

 

Example

 

for k = -10 to 10

   try

      v = 1/k

      pr "k = $k$ v = $v$\n"

   catch

      pr "divide by zero ignored\n"

   endtry

next k

 

See also

 

catch, endtry, lasterror
