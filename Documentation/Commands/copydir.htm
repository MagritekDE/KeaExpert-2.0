copydir

 

Copies the contents of the source directory to the destination directory.

 

Syntax

 

copydir(source_dir, destination_dir) (ee:ss)

 

 

 

The directories can be relative or absolute. The copy is recursive (subdirectories are also copied). Note that the destination directory should not exist otherwise is will be placed inside the existing directory.

 

 

See also

 

copyfile, rmdir, rmfolder
