getargnames

 

Returns the names of the arguments used to call the current procedure.

 

Syntax

 

names = getargnames()

 

names ... A list of names (LIST)

 

Description

 

Occasionally it is necessary to know the argument names used in the call to the current procedure. This command will recall that information as a list of strings.

 

Example

 

procedure(test)

 

   a = 23

   b = [1:10]

   c = "Hi"

 

   :testfunc(a, b, c)

 

endproc()

 

procedure(testfunc, u, v, w)

 

  print(getargnames())

 

endproc()

 

 

Output

 

   getargnames() = {

       a

       b

       c

   }

 

 

Note that this function will return a null string for an argument name which is an expression (e.g. sin(a) will return a null string).
