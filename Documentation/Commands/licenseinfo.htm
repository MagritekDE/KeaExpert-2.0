licenseinfo

 

Returns information about the current Prospa license.

 

Syntax

 

(serialNr, cmdList) = licenseinfo()

 

 

serialNr ........ serial number of this license (INT)

cmdList ......... list of commands for which this license is valid (LST)

 

 

Description

 

From version 3.2 onwards Prospa requires a license. In addition certain commands require an additional license. This information is also listed in the About Prospa dialog.

 

serialNr is a unique 10 digit number with leading digit = 1. e.g. 1000000001

 

cmdList is a list of strings each the format cmdName : version : description : license info.

 

where cmdName is the Prospa command and license info is the patent or author information which will be displayed in the About Prospa dialog.
