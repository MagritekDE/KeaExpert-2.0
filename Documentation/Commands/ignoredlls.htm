ignoredlls

 

Load all DLLs except those in the supplied ignore list.

 

Syntax

 

ignoredlls(dll_list)

 

dll_list ........ list of DLLs to ignore (LIST)

 

Description

 

If you wish to replace DLL commands with procedure of the same names include the name of the DLLs in the supplied list.

 

Example

 

This example will ignore all pulse program DLLs

 

cd(appdir)

cd("DLLs")

files = ls("*PP*.DLL")

ignoredlls(files)

 

See also

 

ignoredll, listdlls, unloaddlls, usedlls
