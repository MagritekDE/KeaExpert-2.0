clip3dstatus

 

Turn 3D clipping on or off.

 

Syntax

 

clip3dstatus(status)  (e:s)

 

status ... either "on" or "off" (STR)

 

Description

 

This command can be used to control which part of the 3D plot is subject to clipping and which is not. Call with clip3dstatus ("on") before drawing objects to be clipped, and then switch clipping off afterwards.

 

Example

 

In the first example clipping is turned on for cylinder drawing and 3D box drawing.

 

clear3d()

color3d([1,1,0])

clip3dstatus("on")

cylinder([0,-10,0],[0,10,0],4,[1,1,0])

color3d([1,0,0])

box3d(-15,15,-15,15,-15,15)

clip3d(1,0,"xy","plus")

 

 

 

 

 

In the second case clipping is turned off when drawing the box.

 

clear3d()

color3d([1,1,0])

clip3dstatus("on")

cylinder([0,-10,0],[0,10,0],4,[1,1,0])

clip3dstatus("off")

color3d([1,0,0])

box3d(-15,15,-15,15,-15,15)

clip3d(1,0,"xy","plus")
