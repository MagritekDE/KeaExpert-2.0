join

 

Join two matrices together.

 

Syntax

 

result = join(m1,m2,[edge])

 

result ... result of concatenating m1 and m2 (MAT/CMAT)

m1 ....... first matrix (MAT/CMAT)

m2 ....... second matrix (MAT/CMAT)

edge ..... edge to join on "vert"/"horiz" (STR)

 

Description

 

Matrices m1 and m2 (1D or 2D) are joined or concatenated together. If their widths are the same they are joined at top and bottom (horizontal edge). If their heights are the same they are joined at the sides (vertical edge). In the case where the widths and heights are the same, the default mode is to join at the sides. You can override this by adding an edge string which is either "vert" or "horiz".

 

Note that 3D matrices are not supported.

 

Example

 

a = [1,2,3]

b = [4,5,6]

pr join(a,b,"horiz")

pr join(a,b,"vert")

 

   join(a,b,"horiz") =

 

   1   2   3

   4   5   6

 

 

   join(a,b,"vert") =

 

   1   2   3   4   5   6
