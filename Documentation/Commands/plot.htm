plot

 

Plot a 1D data set.

 

Syntax

 

n = plot(y)                  (e:m/cm)

n = plot(y,parameter_list)     (ee:m/cm,l)

n = plot(x,y)                (ee:m,m/cm)

n = plot(x,y,parameter_list) (eee:m,m/cm,l)

 

n ................ trace number for this plotted 1D data (INT)

x ................ x data positions (MAT)

y ................ y data positions (MAT/CMAT)

parameter_list.... list of parameters (LIST)

 

Description

 

Generates a 1D plot in the current 1D region (as defined by curplot or the mouse). In the first form it just plots the 1D matrix y as a function of its array index. y may be real or complex. In the third case the x axis is explicitly provided and is used to define the x scale and the positions for the data in the plot. The other two options allow various trace parameters to be set as well (such as the line colour, symbol shape etc). The plot command returns a unique integer identifier n for the plotted data which may be used later with the command curtrace.

 

All plot commands return the trace index number. This zero based index can be used to set the current trace using the command curtrace or it can be used with curplot to extract a handle to the current trace.

 

Note that the trace parameters are not saved between plots. (To do this, use the command tracepref). To modify the rest of the plot after it has been plotted use the current-plot commands axes, draw1d, grid, title and zoom1d. Alternatively, extract a handle to the current plot using the command curplot. To modify the trace after it has been plotted (e.g. change line or symbol colour) you will need to extract a handle to the trace information using the command curtrace.

 

For a list of the parameters available for parameter_list, please see plotpref.

 

 Example

 

x = exp([0:0.1:3])-1

y = exp(-x/10)

plot(x,y,"symbolshape","opensquare")
