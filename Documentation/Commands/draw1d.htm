draw1d

  

Sets the redraw mode for 1D plot. If true, redraws the plot.

 

Syntax

 

draw1d(action) (e:s)

 

action ... either "true" or "false". "on" or "off" also work. (STR)

  

Description

 

Normally, 1D plot commands affect the display instantly. However if you have a series of such commands this can result in undesirable display updates. This command allows all changes to occur at the same time.

 

If action is true then the plot will be redrawn, and subsequent commands which affect the 1D plot will automatically update it.

 

If action is false then the plot will not be redrawn, and subsequent commands which affect the 1D plot will not automatically update it.

 

Example

 

 

 

x = [0:1:1000]*pi/100

for(m = 1 to 400)

   draw1d("false")

   plot(cos(x*2),sin(m*x/200))

   title("Lissajous Figures: m = $m$")

   xlabel("x axis")

   ylabel("y axis")

   draw1d("true")

   pause(0.05)

 next(m)

 

In this example the plot command displays some default labels and a title; these are overridden by the explicit title and label commands. Try running the same macro but with the drawplot commands removed.
