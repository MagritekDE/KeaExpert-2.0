getpar

 

Extract a parameter from a user-defined window object.

 

Syntax

 

result = getpar(winNr, ctrlNr, parameter)    (eee:dds)

 

winNr ....... parent window of control. (INT)

ctrlNr ...... number of this control. (INT)

parameter ... name of parameter to get (STR)

 

ctrlNr is the ID number specified when the object was made, while parameter is a string which is dependent on the type of object the data is being extracted from. 

 

Description

 

Each user-defined object belongs to a window specified by a number winNr. If winNr is zero then information is extracted from the current GUI window, if this is relevant.

 

For a list of parameters, please see the documentation for generic window object parameters, along with the documentation for the specific type of window object of interest.

Get object id of the control.

 

“ctrlnr/objnr”

Get object number of the control.

 

“datatype”

Get the data type for this control, one of

"integer"

"float"

"double"

"string"

"unknown"

 

“enable”

Returns "true" or "false" depending on the enabled   state of the control.

 

“event”

Returns the name of the control’s callback event.

 

“eventactive”

Returns "true" or "false" depending on the event-  active state of the control.

 

“focus”

Returns "true" or "false" depending on the focus   state of the control.

 

“label”

Get text label for control

 

“menubar”

Returns a list of the items in the menu bar as strings.

 

“name”

Get control name

 

“parent”

Returns a class reference to the parent of the control.

 

“procedure”

Returns the name of the control’s callback procedure.

 

“range”

Returns the upper and lower bounds of the control as a vector.

 

“readonly”

Returns "true" or "false" depending on the readonly state of the control.

 

“region”

Get the region for this control as a vector.

 

“statusbox”

Returns the number of the status box object associated with this control.

 

“tabnr/tab_number”

Get the tab number for this control.

 

“tabparent”

Get the current tab parent and tab page as a vector.

 

“tag”

Get the user-definable tag of the control.

 

“toolbar”

Returns the number of the toolbar object associated with this control.

 

“valueid”

Get control name

 

“visible”

Returns "true" or "false" depending on the visibility state of the control.

 

“width/height”

Get width or height of control

 

“winnr”

Get the parent window number.

 

“x/y”

Get location (x or y) of control
