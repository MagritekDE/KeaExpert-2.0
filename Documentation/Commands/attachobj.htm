attachobj

 

Attach selected objects to one side of the parent window.

 

Syntax

 

attachobj(mode) (e:s)

 

mode .. how to attach the objects (STR)

left – keep a constant distance between the window left edge and object.

right – keep a constant distance between the window right edge and object.

top – keep a constant distance between the window top edge and object.

bottom – keep a constant distance between the window bottom edge and object.

 

Description

 

Attach the selected GUI objects to one side of the parent window by passing a mode string (left, right, top, bottom).

 

When the parent window is resized the object will move so it keeps a constant distance between the object and the selected edge. An object can be attached to more than one window edge.

 

This command is used in the GUI designer window. (See its attach button).
