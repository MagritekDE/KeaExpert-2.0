axis3d

 

Draw a single 3D axis with labels.

 

Syntax

 

axis3d(direction, plot_range, intercept1, intercept2, label_range, label, label_position)

 

direction ...... which axis to plot ("x", "y" or "z") (STR)

plot_range ..... minimum and maximum values for axes extent along plot axis (in plot units) (FLOAT)

intercept1 ..... intercept of axis on second axis  (in plot units) (FLOAT)

intercept2 ..... intercept of axis on third axis (in plot units) (FLOAT)

label_range .... minimum and maximum values to be used for labelling the axis. (VEC)

label .......... label to be printed on axes (STR)

label_position . position of label; either "centre", "positive_end" or "negative_end"

 

Description

 

This command contains information needed to draw a single axis for a 3D plot. Note that the intercepts are cyclic i.e.

 

direction

 

axis 1

axis 2

X

y

z

Y

z

x

Z

x

y

 

 

Examples

 

 

 

 

clear3d

axispar3d(0.5,1,1,2)

axis3d("x",[0,20],0,0,[0,1],"x")

axis3d("y",[0,20],0,0,[0,2],"y")

axis3d("z",[0,20],0,0,[0,5],"z")

 

 

 

clear3d

axispar3d(0.5,1,0.8,1)

axis3d("x",[0,20],10,10,[-1,1],"x","positive_end")

axis3d("y",[0,20],10,10,[-1,1],"y","positive_end")

axis3d("z",[0,20],10,10,[-1,1],"z","positive_end")

shift3d(-10,-10,-10)
