getlistnames

 

Return all the entry names in a parameter list

 

Syntax

 

result = getlistnames(list)  (e:l)

 

 

result ... modified list (LIST/PLIST)

list ..... source list (LIST/PLIST)

 

 

Description

 

Extracts all the entry names from a parameter list and returns them in a new list. If the source list is not a parameter list then result will just be a copy of the source list.

 

Example

 

lst = ["a = 1", "b = 2", "c = 3", "d = 4"]

 

pr getlistnames(lst)

 

 getlistnames(lst) = {

 

   a

   b

   c

   d

 }
