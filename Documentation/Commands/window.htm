window

 

Generate a user-modifiable window object.

 

Syntax

 

nr = window(title,x,y,w,h) (eeeee:sdddd)

 

title ... Title of the window. (STR)

x ....... Top left hand corner x component. (INT)

y ....... Top left hand corner y component. (INT)

w ....... Width in pixels of the window. (INT)

h ....... Height in pixels of the window. (INT)

 

Description

 

This command is included to allow window-based user-interfaces to be added to macros. It generates a window into which other controls can be added. The window has a text title, a position relative to the top left hand corner of the screen "x,y", and a width and height "w,h" (in pixels). If x = -1 and y = -1 then the window will be centred in the screen.

 

The command returns a unique identifier for this window, which is an incremented integer starting from 1. This identifier must be used to identify the window when modifying its sub-window objects such as statictext, sliders, and other controls with the setpar and getpar commands.

 

Note that the window is initially hidden, allowing controls to be added to it while it is still hidden so that the display is consistent. Once you have finished populating the window, you will need to call showwindow to make it visible.
