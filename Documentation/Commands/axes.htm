axes

 

Modify the axes parameters for the current plot.

 

Syntax

 

axes(parameter, value, parameter, value, …)

   

Parameter

Type

 

Example

 

autoscale

axescolor

Description

 

This command acts on the current plot axes (either selected by the mouse or by using the curplot command).  parameter and value pairs can be entered in any order, and pairs may be placed on separate lines.

 

Example

 

To set the current plot’s axes color to green, fontsize to 10, x tickspacing to 50, x label spacing to 200, y tickspacing to 10, and y label spacing to 50:

 

axes("axescolor",[0,255,0],  

"fontsize",10,          

"xtickspacing", 50,

"xticksperlabel",200,

"ytickspacing",10,

"yticksperlabel",50)
