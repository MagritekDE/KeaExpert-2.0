cumsum

 

Calculate the cumulative sum of a matrix.

 

Syntax

 

result = cumsum(matrix, [dimension])    (ee:vt)

 

result ...... the cumulative sum (MAT)

matrix ...... any valid expression which results in a 1D or 2D matrix. (MAT)

dimension ... which dimension or axis to sum over (STR)

 

Description

 

Sums each row (dimension = "y") or column (dimension = "x") placing the cumulative result in the output matrix result.

  

Example

 

v = [1,2,3;4,5,6]

pr (cumsum(v,"x"))

 

   cumsum(v) =

 

   1    3    6

   4    9   15

 

v = [1,2,3;4,5,6]

pr (cumsum(v,"y"))

 

   cumsum(v,"y") =

 

   1   2   3

   5   7   9
