mergelists

 

Merge two parameter lists together.

 

Syntax

 

result = mergelists(list1, list2) [ee:vv]

 

result ... merge of list1 and list2 (LIST)

list1 .... first list (all included) (LIST)

list2 .... second list (unique entries included) (LIST)

 

result is a parameter list formed by merging parameter lists list1 and list2. All entries in list1 appear in result, but only entries in list2 which have names which do not appear in list1 will be added.

 

Example

 

l1 = ["a = 23", "b = 45", "c = 56", "d = 78"]

l2 = ["e = 23", "f = 45", "c = 18", "d = 98", "q = 123"]

pr mergelists(l1,l2)

 

 mergelists(l1,l2) = {

 

   a = 23

   b = 45

   c = 56

   d = 78

   e = 23

   f = 45

   q = 123

    }
