rand

 

Generate a random integer or matrix   

 

Syntax

 

output = rand(max)

 

or

 

output = rand(width,height, max)

 

max........ maximum random value (INT)

width ..... matrix width

height .... matrix height

result .... random number or matrix

 

Description

 

Generates a random integer between 0 and max or fills a matrix with such numbers. (Probability is equal for each value).

 

Example

 

Simple example

 

> rand(10)

 

   rand(10) = 4

 

> rand(10)

 

   rand(10) = 2

 

Probability test

 

N = 1000000

m = matrix(N)

for(k = 0 to N-1)

m[k] = rand(10)

next(k)

(x,y) = histogram(m,30,[-10,20])

plot(x,y)

 

 

Simpler version using 1D matrix option

 

m = rand(100000,1,10)

 

(x,y) = histogram(m,30,[-10,20])

plot(x,y)
