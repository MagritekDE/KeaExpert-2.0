autophase

 

Determine the zeroth order phase shift for a peak.

 

Syntax

 

phase = autophase(y [, left, right [,method]])   (eeee:vllt)

 

phase .... phase correction in degrees (FLOAT)

y ........ data to phase (VEC)

left ..... left index limit for correction (INT)

right .... right index limit for correction (INT)

method ... either 'minimise' or 'maximise' (default)

 

Description

 

y is a 1D matrix containing the data to investigate, while left and right define the range of data in this matrix to consider (supplied as vector indices). If left and right are not supplied, the complete vector index range is used instead.

 

The command internally applies the following function to vector y

 

y’ = y * exp(i*phi)

 

There are two methods supplied for autophasing a peak. The first, “maximise”, adjusts the phase so that the area under the real spectrum between the left and right indices is maximised. The second method, “minimise”, adjusts the value of phi so that in the real part of y the integral of the spectrum below the baseline (defined by a line between points left and right), is minimised.

 

The quality of the autophasing depends on the shape of the peaks and the quality of the baseline. The maximise method works best if the baseline is flat while the minimise method works with sloping baselines but tends to be more sensitive to noise than the maximise method.

 

If the spectrum has only one peak in it then left and right can be set to the extremes of the spectrum although this will make the function run slower. Otherwise they can be chosen interactively as in the macro 1D>autophase1d.

 

Example

 

# Generate spectrum with single peak at 100 Hz

# peak has a phase shift of 60 degrees

t = [0:1:1023]/1023

w = 2*pi*100

p0 = 60/180*pi

T2 = 0.1

sigt = 100*exp(-i*(w*t-p0) - t/T2)

# Fourier tranform and generate frequency axis

sigf = ft(sigt)

f = ([0:1:1023]-512)/1023*(1/t[1])

# Plot data

plot(f,sigf)

# Pause 1 second

pause 1

# Calculate p0 phase correction between 50 and 150Hz

(l,r) = findindex(f,50,150)

p0 = autophase(sigf,l,r)

# Correct the spectrum

sigfp = sigf*exp(-i*p0/180*pi)

# Replot corrected data

plot(f,sigfp)

# print correction

pr p0
