subplot

 

Specify which 1D or 2D subplot to keep and display.

 

Syntax

 

subplot(which, column, row))

 

which ..... 1D/2D (STR)

column .... horizontal (x) position (1 based) (INT)

row ....... vertical (y) position (1 based) (INT)

 

Description

 

If the multiplot command has been used to sub-divide the 1D or 2D plot then this command can be used to extract one of these plots and remove all the rest.

 

Example

 

x = [0:0.1:10]

multiplot("1d",1,2)

curplot("1d",1,1)

plot(x,x)

curplot("1d",2,1)

plot(x,x^2)

pause(2)

subplot("1d",2,1)

 

See also

 

multiplot, curplot, plot
