classinfo

 

Provides a link between a class instance name and the class file name.

 

Syntax

 

classinfo(association_list) (e:list)

 

Example

 

This example is taken from SpinsolveExpert.

 

   assoc = ["gView   = seView",

            "gBatch  = seBatch",

            "gExpt   = seExpt",

            "gParam  = seParam",

            "gPlot   = sePlot",

            "gProc   = seProc",

            "gData   = seData",

            "gDebug  = seDebug",

            "gSample = seSample",

            "gMenus  = seMenus",

            "gSeq    = seSequence",

            "gImport = seImport",

            "gFX3    = seFX3",

            "gDSP    = seDSP",

            "gLock   = seLock",

            "gFFT    = seFFT",

            "quick   = parabolic",

            "power   = simplex"]

 

   classinfo(assoc) 

 

Once this association has been established you can navigate around the class and get information about class methods by control-double/single clicking on the class variable name. Without the classinfo command this will not work.
