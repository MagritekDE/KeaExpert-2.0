unloaddlls

 

Unload all DLLs from Prospa.

 

Syntax

 

unloaddlls()

 

Description

 

Unloads all the dynamic linked libraries from the Prospa application. This is useful while developing a DLL, since it is then unnecessary to exit and rerun Prospa to load the modified or new DLL. The steps you should take are:

 

1.      unload the DLLs.

2.      regenerate the modified or new DLL.

3.      reload the DLLs.
