picture

 

Make a picture control.

 

Syntax

 

picture(nr,x,y,w,h)   (eeeee:ddddd)

 

nr ............ control number ID (INT)

x,y ........... top left coordinates of control in pixels (INT)

w,h ........... width and height of control in pixels (INT)

 

Description

 

This command allows the display of a bitmap or jpeg image within a window.

 

A variety of picture formats are supported, including JPEG, TIFF, GIF, BMP, and PNG.

 

Picture Parameters

 

Use setpar to access this:

 

file .......... Name of bitmap or jpeg file to load and display (STR)

 

Example

 

This example displays a window with a small picture loaded. Note that the picture will be scaled to fit the control size but for best image quality the number of pixels in the two should be matched.

 

 

procedure(Picture test)

 

   n = window("Picture test", -1, -1, 350, 163, "resizable")

 

      # Define all controls with basic parameters

       button(1, 280, 20, 50, 31, "Close", closewindow(0))

       picture(2, 5, 5, 260, 150)

 

   setpar(n,2,"file","test.jpg")

   showwindow(n)

 

endproc()
