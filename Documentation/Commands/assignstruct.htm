assignstruct

 

Extract the contents of a structure and store it as either local or window variables.

 

Syntax

 

assignstruct(variable, vartype) (e:s)

 

variable ... Input structure. (STRUCT)

vartype .... Which type of variables to produce (STR)

 

Description

 

This command unpacks all variables in a structure to either the local (vartype = “local”)  or window (vartype = “winvar”) space.

 

Example

 

procedure(strucTest)

 

  a = 23.3

  b = 2+3i

  c = "Test string"

  d = [1:0.1:10]

 

  s = struct("local")

 

  :showStruct(s)

 

endproc()

 

procedure(showStruct,a)

 

  assignstruct(a,"local")

  pr local

 

endproc() 

 

 

Output in this case is:

 

###### Local Variables ######

 

   a = 23.3

   b = 2+3i

   c = "Test string"

   d = (matrix with 91*1 elements)

   nrArgs = 0

 

###### End local variables ######
