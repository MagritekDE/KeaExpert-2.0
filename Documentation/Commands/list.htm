list

 

Make a list variable.

 

Syntax

 

a = list(n)   (e:l)

 

a ......... list of n elements (LIST)

n ......... number of elements in new list (INTEGER)

 

Description

 

This makes a list with n entries. List entry j is initialized to the string value "item $j$".

 

Example

 

a = list(4)

for (c = 0 to 3)

   a[c] = "file$c$.1d"

next c

 

pr a

 

   a = {

       item 0

       item 1

       item 2

       item 3

   }

 

pr a

 

 a = {

     file0.1d

     file1.1d

     file2.1d

     file3.1d

          }
