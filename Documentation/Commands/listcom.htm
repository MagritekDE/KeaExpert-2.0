listcom

 

List commands built into Prospa.

 

Syntax

 

result = listcom([start, [filter])    (ce:ss)

 

start ... list commands starting with these characters.

filter .. filter list by the category name,

result .. the results as a list.

 

Description

 

Lists all commands built into Prospa. With the optional start argument, then only those commands with start as a name prefix will be listed. Note that start is not delimited by quotation marks. If start = "*" then all commands will be considered. This is the default.

 

If filter is added then only those commands which match the start and filter arguments will be listed.

 

Possible filters are:

 

1d ......... commands relating to 1D plots

2d ......... commands relating to 2D plots

2d ......... commands relating to 3D plots

color ...... commands for manipulating color

control .... commands which control the flow of a procedure

dlls ....... commands relating to loading and using dynamic libraries.

files ...... commands relating to the file system

fourier .... commands relating to Fourier transforms

gui ........ command relating to generating user interfaces

help ....... commands for viewing help.

license .... commands relating to the Prospa license

lists ...... commands for manipulating lists

macros ..... commands for working with macros

math ....... mathematical commands

matrix ..... commands for manipulating matrices

misc ....... miscellaneous commands not falling into another category

sound ...... commands for generating sounds.

strings .... commands for manipulating strings

struct ..... commands for manipulating structures

time ....... commands relating to timing

trig ....... trigonometric commands

variables .. commands for manipulating variables

 

 

Example

 

To list all commands which start with the letters "lo"

 

listcom(lo)

 

listcom(lo) = {

load

loaddlls

loadplotmode

loadpix

log10

loge

log2

} 

 

    To list all file 3d commands which start with "d"

 

 

   listcom("d","3d") = {

       datarange3d

       depthcue

       depthcuerange

       diag

       diff

       draw3d

   }
