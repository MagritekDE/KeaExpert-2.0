image

 

Display a 2D intensity image.

 

Syntax

 

image(matrix,[colorscale])     (ee:m/cm,m)

 

or

 

image(matrix,x,y,colorscale) (eeee:m/cm,dds)

 

matrix ....... matrix to draw as intensity image (MAT)

x ............ minimum and maximum x colour value (MAT)

y ............ minimum and maximum y colour values (MAT)

colorscale ... colour map; see valid values at cmap (STR)

 

Description

 

This command displays an intensity image of a matrix or the real part of a complex matrix ("expression") by mapping the data values to a predefined color map. It does this by determining the minimum and maximum values in the data set and then converting each data value to an RGB (red-green-blue) value appropriate for the chosen colour scale. This information is stored in a bitmap which is displayed on the screen.  In the second version, user-defined x and y scales can be specified by supplying minimum and maximum values as 2-element arrays.

 

The available colour scales are described in cmap.

 

Example

 

m = noise(100,100)

x = [-10,10]

y = [-10,10]

image(m, x, y, hot)
