curcli

 

Get or set the current command line interface

 

Syntax

 

obj = curcli([winNr, objNr])      (ee:dd)

 

winNr ... The window the CLI belongs to (INT)

objNr ... The object number for the CLI (INT)

obj ..... Returned object if no arguments passed (OBJ)

 

Description

 

If winNr and objNr are defined, then the current command line interface is set to the specified object. This means that future print output will go to that CLI. If winNr is not defined, then the CLI object is returned.

 

 

Example

 

Make two CLIs in the normal Prospa interface. Assume these belong to window 2 and window 7. This macro will print the even numbers from 0 to 10 in window 2 and the odd numbers to window 7.

 

for(k = 0 to 10)

 

  if(k%2==0)

      curcli(2,1)

      print(k)

   else

      curcli(7,1)

      print(k)

   endif

 

next(k)
