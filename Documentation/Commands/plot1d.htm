plot1d

 

Generate a 1D plot window.

 

Syntax

 

plt = plot1d(objNr, x, y, w, h)       (eeeee:ddddd)

 

objNr ......... control number ID  (INT)

x,y ........... top left coordinates of control in pixels (INT)

w,h ........... width and height of control in pixels     (INT)  

plt ........... access to plot internal functions and parameters (optional)    (OBJ)

 

Description

 

This command produces a 1D plot object. This can be used for displaying x-y type data. The plot can be divided into multiple regions. Each region can display multiple traces. Each trace is an x-y data set.

 

A plot1d object has a large number of underlying functions which can be used for modifying how it appears and behaves. These functions can be accessed using the returned object plt, also accessible using the getobj command, or by using the current-plot commands provided with versions of Prospa before 3.0. The object functions are listed below along with their current-plot alternatives.

Current-plot access

 

There are a number of commands which modify the current 1D or 2D plot. The current plot can be set by clicking on it with the mouse or by using the following commands

 

curplot(winNr, objNr, x_index, y_index)

 

or

 

curplot(winNr, objNr, region_number)

 

Current-plot commands equivalent to the above are listed below.

 

plot() ......... draw (x,y) on current plot region.

getplotdata() .. extract current trace data

trace() ........ set or get plot current trace parameters

xlabel() ....... set or get current plot region xlabel

ylabel() ....... set or get current plot region ylabel

title() ........ set or get current plot region title

axes() ......... set or get current plot region axes
