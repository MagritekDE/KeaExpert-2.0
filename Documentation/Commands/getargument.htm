getargument

 

Returns the value for the nth argument in the current procedure command.

 

Syntax

 

name = getargument(position) (e:d)

 

position ... the zero-based argument position (INTEGER)

name ....... The argument name (STR)

 

Description

 

Occasionally the position of the argument is more important than its name.

 

Example

 

procedure(test)

 

   a = 23

   b = [1:10]

   c = "Hi"

 

   :testfunc(a, b, c)

 

endproc()

 

procedure(testfunc, a0, a1, a2)

 

  print(getargument(2))

 

endproc()

 

 

Output

 

  getargument(2) = Hi
