scanstr

 

Scan a string for embedded substrings.

 

Syntax

 

(match1, match2, ...)  = scanstr(text, regex)

 

text ..... text to scan (STR)

regex .... regular expression to search for (STR)

matchi ... ith match (STR) (null if no match)

 

Description

 

This command may be used for extracting information from a string. You do this by specifying key words in the regex string, along with special characters matching the information you are looking for.

 

Unknown parts of the string can be matched with the wild card characters * and ?.

 

There may be up to 9 matching strings. These are indicated by appending a percent symbol (%) to an integer (ie. %1, %2 ... %9).

 

Matching characters

 

* .... wild card for 1 or more characters

? .... wild card for 1 character only

\n ... line feed

\r ... carriage return

 

All other characters will be matched exactly.

 

Should a match fail for any reason, scanstr will return a single null variable.

 

Example

 

Suppose you want to find the value of the parameter time_points in a data acquisition file. Assume you know that file file has the following format:

 

...

... time_points = xxx\r\n

...

 

where xxx indicates the number of points present.

 

You could do this with the following macro:

 

txt = load(file, "text")

result = scanstr(txt, "*time_points = %1\r\n")

time_points = eval(result) 

 

result will then have the value found between the equals sign and the \r\n characters. Note that result is still a string and must be converted to a number using the eval command.
